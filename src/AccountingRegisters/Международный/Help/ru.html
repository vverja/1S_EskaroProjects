Регистр бухгалтерии «Международный учет»

Регистр бухгалтерии «Международный учет» используется для ведения бухгалтерского учета, отличного от регламентированного, в частности, в целях дальнейшей подготовки финансовой отчетности по международным стандартам (МСФО).

Регистр использует план счетов «План счетов международного учета» и поддерживает корреспонденции. Это означает, что каждая запись регистра имеет дебетовую и кредитовую часть. Такая организация регистра соответствует традиционному ведению бухгалтерского учета РФ и позволяет получать информацию не только об остатках и оборотах по счетам, но и о корреспонденциях между счетами (что не свойственно традиционному «западному» учету, где не предусмотрена поддержка корреспонденций).

Измерения

Организация

Балансовое измерение регистра «Организация» обеспечивает ведение учета в разрезе юридических лиц.

Пустое значение этого измерения используется для целей подготовки консолидированной финансовой отчетности. Данные таких записей учитываются при составлении консолидированных финансовых отчетов группы, но не учитываются в индивидуальной отчетности каждой из организаций, входящих в группу. В частности, подобные проводки формируются для элиминирования (исключения) результатов внутригрупповых операций при консолидации в пределах одной информационной базы (см. обработку «Консолидация МСФО»)

Валюта

Измерение регистра «Валюта» не является балансовым и обеспечивает учет по валютным счетам в разрезе отдельных наименований иностранных валют.

Ресурсы

Сумма

Балансовый ресурс «Сумма» является основным и обеспечивает учет в валюте международного учета на всех счетах плана счетов. Это относится и к предопределенным счетам, и к тем, которые введены в режиме 1С:Предприятие.

По отдельным видам субконто на счетах суммовой учет может не вестись.

Валютная сумма

Ресурс «Валютная сумма» не является балансовым, а значит, может вестись выборочно, для отдельных счетов. Такие счета имеют признак «Валютный». Ведение валютного учета по каждому виду субконто на таких счетах настраивается отдельно.

Ведение валютного учета по счету в целом и по отдельным видам субконто дает возможность получать остатки и обороты в валюте в разрезе валют.

Количество

Ресурс «Количество» не является балансовым и ведется только для тех счетов, для которых имеет смысл учет в натуральных измерителях. Такие счета имеют признак «Количественный». Ведение количественного учета по каждому виду субконто на таких счетах настраивается отдельно.

Количественный учет по счету и субконто ведется без разделения по единицам измерения. Предполагается, что для каждого вида актива это учет в одной основной единице измерения.

Реквизиты

Содержание

Содержание представляет собой краткое описание сути проводки длиной не более 150 символов.

Первичный документ

Содержит ссылку на первичный документ, который являлся основанием для проводки. Наличие такой ссылки объясняется тем, что основная масса проводок по международному учету, как предполагается, будет получена путем трансляции из регламентированного (национального) учета (то есть с использованием документа «Перенос проводок (международный)»). Таким образом, первичные документы в основном не отражаются в международном учете напрямую. Тем не менее, сохранить связь проводок и первичных документов необходимо для всех случаев.

Номер журнала

Номер журнала является вспомогательным реквизитом для условного разделения проводок по разделам учета (товары, основные средства, операции по банку и кассе и т.п.). Номер журнала представляет собой строку длиной 3 символа и может использоваться для быстрого отбора проводок (по нему установлено индексирование).
