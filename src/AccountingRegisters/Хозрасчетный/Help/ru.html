Регистр бухгалтерии «Бухгалтерский учет»

Регистр бухгалтерии «Бухгалтерский учет» используется для ведения регламентированного бухгалтерского учета.

Регистр использует план счетов «План счетов бухгалтерского учета» и поддерживает корреспонденции. Это означает, что каждая запись регистра имеет дебетовую и кредитовую часть. Такая организация регистра соответствует нормативным актам по бухгалтерскому учету и позволяет получать информацию не только об остатках и оборотах по счетам, но и о корреспонденциях между счетами.

Измерения

Организация

Балансовое измерение регистра «Организация» обеспечивает ведение учета в разрезе юридических лиц.

Валюта

Измерение регистра «Валюта» является небалансовым и обеспечивает учет по валютным счетам в разрезе отдельных наименований иностранных валют.

Ресурсы

Сумма

Балансовый ресурс «Сумма» является основным и обеспечивает учет в национальной валюте на всех счетах плана счетов. Это относится и к предопределенным счетам и к тем, которые введены в режиме 1С:Предприятие.

По отдельным видам субконто на счетах суммовой учет может не вестись.

Валютная сумма

Ресурс «Валютная сумма» является небалансовым, а значит, может вестись выборочно, для отдельных счетов. Такие счета имеют признак «Валютный». Ведение валютного учета по каждому виду субконто на таких счетах настраивается отдельно.

Ведение валютного учета по счету в целом и по отдельным видам субконто дает возможность получать остатки и обороты в валюте в разрезе валют.

Количество

Ресурс «Количество» является небалансовым и ведется только для тех счетов, для которых имеет смысл учет в натуральных измерителях. Такие счета имеют признак «Количественный». Ведение количественного учета по каждому виду субконто на таких счетах настраивается отдельно.

Количественный учет по счету и субконто ведется без разделения по единицам измерения. Предполагается, что для каждого вида актива это учет в одной основной единице измерения.
Примечание. Принципиально регистр бухгалтерского учета может поддерживать учет в различных единицах измерения раздельно. В случае целесообразности такого учета потребуется ввести еще одно небалансовое измерение «Единица измерения».

Реквизиты

Номер журнала

Номер журнала является вспомогательным реквизитом для условного разделения проводок по разделам учета (товары, основные средства, операции по банку и кассе и т.п.). Номер журнала представляет собой строку длиной 2 символа и может использоваться для быстрого отбора проводок (по нему установлено индексирование).

Содержание

Содержание представляет собой краткое описание сути проводки длиной не более 150 символов.
