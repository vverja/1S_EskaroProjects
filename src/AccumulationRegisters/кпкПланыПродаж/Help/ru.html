Регистр накопления «Планы продаж по агентам»

Предназначен для хранения данных по запланированному объему и сумме продаж в разрезе агентов и номенклатурных групп. Записи этого регистра формируются при проведении документа "Планирование для агентов".

Данные из регистра используются при формировании отчета "Выполнение плана агентами".
