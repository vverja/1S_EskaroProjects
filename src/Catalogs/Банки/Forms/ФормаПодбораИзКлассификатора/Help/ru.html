Подбор валюты из классификатора банков

Для добавления данных из классификатора банков необходимо нажать на кнопку "Подбор из классификатора" в списке банков.

При этом откроется таблица классификатора банков.
В этой таблице можно осуществлять поиск стандартными средствами, предоставляемыми программой «1С:Предприятие».

Непосредственно из таблицы можно добавить новый банк в справочник. 
Для этого надо установить курсор на нужной записи и нажать клавишу "ENTER" или два раза нажать левую клавишу мыши.
