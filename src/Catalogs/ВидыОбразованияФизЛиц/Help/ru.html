Справочник «Виды образования физических лиц»

Справочник используется при вводе данных об образовании работников организаций. 

Справочник может быть заполнен по данным фасета «Образование» из Общероссийского классификатора информации о населении ОК  018-95. Для этого предназначена кнопка «Подбор из ОКИН» в форме списка. При этом открывается список фасета «Образование», в котором можно найти, выбрать и занести в справочник нужный вид образования.

Некоторые элементы справочника «Виды образования физических лиц» являются предопределенными и не удаляются пользователями.
