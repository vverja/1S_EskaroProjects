Справочник «Виды отпусков»

В справочнике хранится информация о видах отпусков которые используются в организации.

 

Для каждого вида отпуска требуется ввести следующую информацию:

  • наименование;
  • код;
  • признак ведения учета остатков;

Если вид отпуска является ежегодным то для изменения становятся доступными следующие настройки:

  • учет по календарным годам (если флаг не включен - отпуск учитывается по рабочим периодам начиная с даты приема на работу);
  • отпуск не переносится на следующий год (если флаг не включен - неиспользованный отпуск предыдуших периодов нельзя использовать в текущем);
  • отпуск не компенсируется (если флаг не включен - неиспользованный отпуск не подлежит компенсации по желанию сотрудника или при увольнении);
  • включать в дни отпуска праздничные дни (Если флаг не включен - праздничные и нерабочие дни не будут включены в продолжтельность отпуска).

В справочнике присутствуют следующие предопределенные виды отпусков:основной ежегодный отпуск;

  • дополнительный ежегодный отпуск;
  • прочий дополнительный отпуск;
  • дополнительный отпуск на детей;
  • учебный отпуск;
  • учебный отпуск без оплаты;
  • отпуск за свой счет (обязательный);
  • отпуск за свой счет (по соглашению сторон);
  • чернобыльский отпуск.

Для каждого предопределенного вида отпуска настройки установлены в соответствии с законодательством.
