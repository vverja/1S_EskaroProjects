Справочник "Группы пользователей"

Справочник используется:

  • для определения прав доступа на уровне записей
  • для определения даты запрета изменения данных
  • для адресации задач в процедуре (бизнес-процессе) закрытия месяца
  • для назначения настроек отчетов.

В справочнике необходимо указать пользователей, которые входят в эту группу.

Ограничение доступа

Для настройки прав доступа на уровне записей необходимо выбрать виды объектов доступа по которым будет ограничиваться доступ и выполнить настройку прав доступа (кнопка "Настройка доступа"). Список видов объектов доступа редактируется в настройке параметров ограничения доступа.

В колонке "Чтение" для каждого вида объектов доступа выводится информация о количестве объектов доступных для чтения.

Под списком видов объектов доступа выводится информационная надпись, которая информирует о действующих ограничениях на уровне записей в целом:

  • Доступ не ограничен. Группа пользователей не ограничивает доступ.
  • Доступ ограничен: есть доступные объекты. Группа пользователей ограничивает доступ и есть объекты к которым настроены права доступа. Подробную информацию можно посмотреть в форме настройки прав доступа.
  • Доступ ограничен: нет доступных объектов. Группа пользователей ограничивает доступ и нет объектов к которым настроены права доступа.

Группа "Все пользователи" является служебной и не редактируется. Считается, что все пользователи входят в эту группу.

ВАЖНО!
Выбор множества видов объектов доступа может приводить к падению быстродействия.
