Справочник «Детализация аналитического баланса»

Подчинен элементу справочника «Статьи аналитического баланса» и содержит правила отбора остатков по плану счетов бюджетирования  для каждой подстатьи, детализирующей статью аналитического баланса. Заполняется пользователем в произвольном порядке. В форме элемента справочника пользователь должен выбрать счета (группы счетов) плана счетов бюджетирования, остатки по которым войдут в данную подстатью, детализирующую статью аналитического баланса, а также вид остатка (дебетовый или кредитовый) и знак, с которым остатки по данному счету (группе счетов) войдут в общий итог по подстатье.
