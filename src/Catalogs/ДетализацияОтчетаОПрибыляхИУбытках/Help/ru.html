Справочник «Детализация отчета о прибылях и убытках»

Подчинен элементу справочника «Статьи отчета о прибылях и убытках» и содержит правила отбора оборотов по регистру «Обороты бюджетов»  для каждой подстатьи, детализирующей статью отчета о прибылях и убытках. Заполняется пользователем в произвольном порядке. В форме элемента справочника пользователь должен выбрать статьи (группы статей) из справочника «Статьи оборотов по бюджетам», обороты по которым войдут в данную подстатью, детализирующую статью отчета о прибылях и убытках, а также указать для каждой из выбранных статей (групп статей) оборотов знак, с которым обороты войдут в общий итог по подстатье.
