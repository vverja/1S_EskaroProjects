Справочник «Индивидуальные графики амортизации»

Справочник содержит доли распределения амортизации основного средства по месяцам эксплуатации. Справочник применяется только для расчета амортизации основного средства способом «По индивидуальному графику», установленному при принятии к учету документом «Принятие к учету ОС».

Этот способ используется только для расчета амортизации по управленческому учету.
