Справочник «Источники информации при обращении покупателей»

В данном справочнике пользователь вводит источники информации, благодаря которым покупатель узнал о предприятии или о проведении той или иной рекламной акции, проводимой предприятием.

Элемент данного справочника выбирается в документе «Событие». Источник информации при обращении можно указывать только для входящих контактов с покупателями.

Для источника обращения определяется число дней (реквизит «Период актуальности информации после события»), в течение которых данный источник информации может признаваться актуальным. Число дней считается с момента первой регистрации источника обращения при оформлении события с клиентом. В последующем, когда истечет срок актуальности первого источника, можно для клиента указать следующий источник информации. Таким образом, можно отследить эффективность проведения рекламных акций предприятия.

Для анализа эффективности тех или иных источников информации используется отчет «Анализ событий и источников информации покупателей».
