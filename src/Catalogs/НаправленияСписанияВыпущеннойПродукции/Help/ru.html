Справочник «Направления списания выпущенной продукции»

В документах «Отчет производства за смену» для каждой позиции выпущенной продукции нужно указать направление выпуска. Если выбирается вариант «на затраты (список)», то ввод направления списания затрат по изготовлению указывается в диалоговом окне «Ввод направлений списания продукции (услуг)». Для частого использования одних и тех же комбинаций значений реквизитов описания направления выпуска их можно определить в шаблон. Для этого предназначен справочник «Направления списания выпущенной продукции».

Реквизиты табличной части «Направления»:

  • Подразделение. Подразделение, на которое относятся затраты по управленческому учету.
  • Подразделение организации. Подразделение, на которое относятся затраты по регламентированному учету.
  • Статья затрат. Обязательный разрез учета затрат.
  • Заказ. Заказ покупателя или заказ на производство – аналитика затрат.
  • Счет затрат. Счета затрат для бухгалтерского учета.
  • Аналитика. Аналитика затрат. Вид аналитики зависит от выбранного счета учета затрат.
  • Коэффициент. Коэффициент распределения суммы затрат на каждое направление.
