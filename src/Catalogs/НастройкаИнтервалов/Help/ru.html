Справочник «Настройка интервалов»

Справочник «Настройка интервалов» - вспомогательный справочник, который используется в отчете «Дебиторская задолженность по интервалам».

Каждый элемент справочника представляет собой список интервалов сроков задолженности. В соответствии с этими интервалами будет группироваться дебиторская задолженность в отчете «Дебиторская задолженность по интервалам».

В поле «Наименование» вводится наименование списка интервалов сроков задолженности.  

Границы интервала в днях задаются в поле «Граница интервала». В поле «Подпись» выводится автоматически сформированная подпись, которая будет отображаться для интервала в отчете.

Для корректной работы отчета «Дебиторская задолженность по интервалам» необходимо, чтобы элемент справочника содержал интервал «Остальные (не менее ... дней)» для отображения задолженности, не попавшей в другие интервалы. Такой интервал создается автоматически при вводе нового элемента справочника. По умолчанию для этого интервала значение границы равно 50000. При необходимости его  можно изменить - изменение и удаление этого интервала не контролируется. Подпись для этого интервала меняется с вводом новых интервалов в позицию справочника и зависит от введенных интервалов.

Выбор значений из справочника «Настройка интервалов» производится в поле «Интервал» в отчете «Дебиторская задолженность по интервалам».
