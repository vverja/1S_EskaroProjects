Справочник «Настройки аналитики учета»

Справочник «Настройки аналитики учета» предназначен для просмотра и изменения настройки детализации затрат при использовании расширенной аналитики учета затрат.

При этом можно настраивать только некоторые разрезы аналитики учета затрат. Флажки, соответствующие другим разрезам - недоступны. Для этих разделов настройка выполняется автоматически (например, для аналитик "Раздел учета", "Затрата") или определяется настройками учетной политики и параметров учета (например, для аналитик "Склад", "Проект").

Внимание! Не рекомендуется изменять настройки детализации учета затрат, если в базе существуют проведенные документы позже даты перехода на расширенную аналитику (можно изменять настройки распределения затрат, если распределение еще не выполнялось)
