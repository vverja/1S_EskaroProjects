<P class="" align=justify>Для <STRONG>Настройки</STRONG> указывается:</P>
<UL>
<LI>
<DIV class="" align=justify>наименование</DIV>
<LI>
<DIV class="" align=justify>дата начала ее действия</DIV>
<LI>
<DIV class="" align=justify>признаки отражения в учетах: УУ, БУ</DIV></LI></UL>
<P class="" align=justify>Признаки отражения в учетах устанавливают ограничение, по каким видам учета может быть проведена процедура закрытия месяца, в которой используется эта настройка. 
<HR>
</SPAN><SPAN class=top id=СводныеНастройки_Показывать>На закладке <STRONG><FONT color=#0054e3>Сводные настройки</FONT></STRONG> отображен состав регламентных операций. 
<P></P>
<P>Для каждой регламентной операции&nbsp;можно:</P>
<UL>
<LI>включить/отключить ее использование 
<LI>назначить ответственного за ее&nbsp;выполнение</LI></UL>
<P><STRONG>См.также</STRONG></P>
<P>На закладке <A id=Команда href="1c:ОткрытьЗакладкуСхема">Схема</A> в графическом виде отображается состав и последовательность выполнения регламентных операций.</P>
<P>На закладке&nbsp;<A id=Команда href="1c:ОткрытьЗакладкуОтветственные">Ответственные</A>&nbsp;удобно назначать ответственных за выполнение регламентных операций.</P></SPAN><SPAN class=top id=Схема_Показывать>На&nbsp;закладке <STRONG><FONT color=#0054e3>Схема</FONT></STRONG> отображается состав и последовательность выполнения регламентных операций. 
<P>Здесь&nbsp;можно:</P>
<UL>
<LI>Отключить/включить регламентную операцию&nbsp;(двойной щелчок мыши на операции) 
<LI>Назначить ответственного (контекстное меню&nbsp;операции при нажатии правой клавиши мыши)</LI></UL></SPAN> 
<SPAN class=top id=Ответственные_Показывать>На закладке <STRONG><FONT color=#0054e3>Ответственные</FONT></STRONG> назначаются ответственные (пользователь&nbsp;или группа пользователей) за выполнение каждой регламентной операции. 
<P>Ответственного можно назначить одним из способов:</P>
<UL>
<LI>указать непосредственно в списке 
<LI>выбрать&nbsp;двойным щелчком мыши на элементе списка пользователей (групп) 
<LI>«перетащить» из списка пользователей (групп) 
<LI>использовать специальную кнопку <STRONG>&lt;</STRONG></LI></UL>
<P>С помощью кнопки&nbsp;<STRONG>&lt;</STRONG>&nbsp;можно одновременно назначить ответственного сразу за&nbsp;несколько операций:</P>
<OL>
<LI>в левом табличном поле выбрать несколько регламентных операций (отмечать, удерживая клавишу Ctrl или Shift) 
<LI>в правом табличном поле выбрать пользователя (группу) 
<LI>нажать кнопку <STRONG>&lt;</STRONG></LI></OL></SPAN><SPAN class=top id=КолонкиВидимы_Показывать>
<P><STRONG><FONT color=#ff0000>Важно!</FONT></STRONG> Принадлежность регламентной операции &nbsp;к видам учета (УУ, БУ, НУ)&nbsp;устанавливается автоматически при создании настройки и отображается для справки. Для изменения <STRONG>недоступна</STRONG>.</P></SPAN></BODY></HTML>
