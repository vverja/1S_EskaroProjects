Настройки заполнения форм

Справочник предназначен для хранения данных форм (справочников, документов).

Добавление элементов в справочник выполняется по команде «Сохранить параметры» в форме справочника, документа

Сохраненные данные форм (настройки) используются для заполнения справочников, документов

  • по команде «Восстановить параметры» в форме справочника, документа
  • по команде «Создать по настройке» в панели действий

Список пользователей

При выборе настройки для заполнения пользователю отображаются только доступные настройки.

Для каждой настройки задается перечень пользователей или групп пользователей, которым эта настройка доступна.
