Справочник «Настройки расчета цены номенклатуры»

Справочник предназначен для хранения настроек формирования документов «Установка цен номенклатуры» по результатам расчета себестоимости.

Используется только в режиме расширенной аналитики учета затрат.

Формирование документов по настройке может выполняться:

  • из внешней обработки заполнения документа "Установка цен номенклатуры"
  • автоматически, регламентным заданием

 

Совместное использование со справочником "Настройки расчета себестоимости"

Для того, чтобы при расчете цены номенклатуры каждый раз получать актуальную информацию о себестоимости, необходимо установить флаг "Предварительно рассчитывать себестоимость" и указать настройку расчета себестоимости. В этом случае перед тем, как будет вычисляться цена номенклатуры, автоматически проведется расчет себестоимости в соответствии с указанной настройкой.
