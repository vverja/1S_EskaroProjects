Справочник «Настройки формирования документов выпуска продукции»

Справочник предназначен для хранения настроек формирования документов выпуска продукции по данным документов «Отчет мастера смены».

По настройке могут формироваться документы «Выпуск продукции» или «Отчет производства за смену»

Формирование документов может выполняться

  • в обработке «Формирование документов по данным оперативного учета производства»
  • автоматически, регламентным заданием

Настроек формирования документов может быть несколько:

  • для разных подразделений могут формироваться документы с разными параметрами (на разные склады, по разным организациям)
  • для одного и того же подразделения могут раздельно формироваться документы, отражаемые в разных видах учета, содержащие данные о выпуске или расходе материалов

Параметр Подразделение определяет, по которому подразделению будут

  • отобраны исходные данные (документы «Отчет мастера смены»)
  • сформированы документы «Выпуск продукции» или «Отчет производства за смену»

Если параметр не заполнен, то будут обрабатываться документы всех подразделений.

Если установлен флаг «Не обрабатывать период до завершения смен», то не будут обрабатываться периоды до даты, по которую завершены смены (см. регистр сведений «Завершенные смены»). Эта настройка не будет использоваться, если подразделение не заполнено.

В поле «Начиная с» указана дата, до которой данные документов «Отчет мастера смены» учтены в сформированных документах.
Её значение

  • изменяется автоматически по мере формирования документов
  • может быть изменено вручную (по кнопке выбора даты).

Параметры создаваемых документов

Вид регулярного документа

Выбирается вид документов, которые будут созданы

Детализация

Документы будут сформированы с заданной детализацией. Каждый документ будет включать данные за период

  • не больший, чем период детализации
  • относящийся к одному календарному месяцу

Документы формируются датой последнего дня периода, данные за который содержат.

Например:

Детализация - 1 документ в неделю. Формируются документы по данным за период с 22 января (понедельник) по 04 февраля.
Будет сформировано 3 документа:
  1. включающий данные за период с 22 по 28 января, дата документа - 28 января
  2. включающий данные за период с 29 по 31 января, дата документа - 31 января
  3. включающий данные за период с 01 по 04 февраля, дата документа - 04 февраля

Порядок заполнения документов «Выпуск продукции»

В сформированных документах «Выпуск продукции» будет заполнена закладка  «Продукция». Исходные данные для заполнения – закладка «Выпуск» документов «Отчет мастера смены»
При этом:

  • номенклатурная группа будет определена по реквизиту номенклатуры  «Номенклатурная группа»
  • качество продукции - «Новый»

Порядок заполнения документов «Отчет производства за смену»

Если по настройке создаются документы «Отчет производства за смену», то доступны параметры, определяющие, какие закладки сформированных документов будут заполнены: 
  • заполнить выпуск - в сформированных документах будет заполнена закладка  «Продукция и услуги». Исходные данные для заполнения – закладка «Выпуск» документов «Отчет мастера смены».

    При этом:

    • номенклатурная группа будет определена по реквизиту номенклатуры  «Номенклатурная группа»
    • качество продукции - «Новый»
    • выпуск услуг не отражается в документах «Отчет производства за смену»

  • заполнить материалы - в сформированных документах будут заполнены закладки «Материалы» и «Распределение материалов». Исходные данные для заполнения – закладка «Расход материалов» документов «Отчет мастера смены».

    При этом:

    • на закладку «Материалы» переносятся все материалы, в том числе потраченные на выпуск услуг
    • на закладку «Распределение материалов» переносятся только материалы, которые потрачены выпуск конкретной продукции – на закладке «Расход материалов» документа «Отчет мастера смены» указана продукция
    • статья затрат будет определена по реквизиту номенклатуры «Статья затрат»

Автоматическое формирование документов (по расписанию)

Для автоматического формирования документов следует установить флаг «Использовать расписание» и ввести расписание.

В момент записи редактируемой Настройки будет создано связанное с ней регламентное задание.

При каждом запуске по расписанию формирование документов будет выполняться за период

с даты, следующей за датой, по которую учтены данные в сформированных ранее документах (см. «Начиная с»)

по 

    • момент запуска
    • одну из дат, предшествующих дате запуска
