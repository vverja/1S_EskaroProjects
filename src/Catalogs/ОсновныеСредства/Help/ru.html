Справочник "Основные средства"

Справочник «Основные средства» используется во всех видах учета (управленческом, бухгалтерском, налоговом, международном) и предназначен в первую очередь для хранения наиболее общей информации об объектах основных средств, которая характеризует сами объекты и не зависит от того, какой организации принадлежит объект, в каком он состоянии находится в тот или иной момент времени и т.д.

В частности, к таким параметрам относятся:

  • полное наименование объекта;
  • изготовитель;
  • номер паспорта;
  • заводской номер;
  • дата выпуска;
  • группа основных средств

и т.д.

Специфическая информация об объектах основных средств, которая используется только в каком-то одном виде учета или меняется со временем (например, организация которой принадлежит объект, срок полезного использования, метод начисления амортизации и т.д.) хранится не в справочнике, а в соответствующих регистрах сведений. В справочнике она отражается в качестве информационных полей на соответствующих закладках.

Важно отметить: Если основное средство подлежит амортизации, то необходимо установить флажок "Подлежит амортизации", иначе амортизация начисляться не будет.

Из формы элемента справочника есть возможность распечатать "Инвентарную карточку учета основных средств" (Типовая форма ОЗ-6).

Основные регистры сведений по основным средствам: 

Для управленческого учета

  • «Первоначальные сведения об основных средствах»
  • «Параметры амортизации (управленческий учет)»
  • «Способы отражения расходов по амортизации»

    • и т.д.

    Для бухгалтерского учета

  • «Первоначальные сведения об основных средствах организаций»
  • «Параметры амортизации (бухгалтерский учет)»
  • «Способы отражения расходов по амортизации (бухгалтерский учет)»
  • «Счета бухгалтерского учета основных средств»

    и т.д.

    Для налогового учета

  • «Первоначальные сведения об основных средствах организаций (НУ)»
  • «Виды налоговой деятельности основных средств»

    Для международного учета

    • «Основные средства (международный учет)».
