Справочник «Параметры выработки ОС»

Справочник содержит перечень параметров, по которым в системе учитывается выработка оборудования. Выработка по таким параметрам используется для расчета амортизации, если для основного средства выбран способ расчета пропорционально работам/ пробегу. Параметры выработки могут использоваться при задании нормативов на обслуживание оборудования «Нормативы обслуживания ОС». Учет фактической выработки отражается документом «Выработка ОС».
