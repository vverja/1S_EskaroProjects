Справочник «Подразделения организаций»

В справочнике описывается структура каждой организации, которая может быть иерархической, т.е. отражать подчиненность одних подразделений организации другим.

Справочник подчинен справочнику организаций, т.е. при вводе новых подразделений или при выборе подразделения в документе обязательно надо указать, подразделение какой именно организации выбирается (как правило, эта организация уже указана в документе).

Начать заполнение справочника следует с подразделений «самого» верхнего уровня — т.е. не подчиненных другим подразделениям. Далее для каждого подразделения верхнего уровня вводятся подчиненные ему подразделения (используется клавиша Ins или пункт «Добавить» меню «Действия»).

Помимо ведения кадрового учета работников организации, справочник «Подразделения организаций» используется для ведения аналитического учета в разрезе подразделений каждой организации.

Справочник позволяет вести учет затрат на местах производства работ (цех №1, цех №2) и по местам концентрации хозяйственных функций (дирекция, бухгалтерия).

Подразделения организаций также используются в ряде документов по учету основных средств.

Используется в бухгалтерском, налоговом и международном учете.
