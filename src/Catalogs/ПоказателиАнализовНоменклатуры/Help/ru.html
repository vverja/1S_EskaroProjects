Справочник «Показатели анализов номенклатуры».

В справочнике хранится информация о нормативном значении результатов замера для данного показателя анализа номенклатуры. Возможны следующие варианты результатов анализа:

  • Список значений - Считается что результат анализа будет соответствовать  нормативу, если результатом будет значение из списка допустимых значений.
  • Значение + / - погрешность - Считается что результат анализа соответствует нормативу, если полученный результат измерений соответствует указанному значению с учетом погрешности.
  • Число в интервале - Считается что результат анализа соответствует нормативу, если полученный результат измерений укладывается в указанный интервал.
  • Поддиапазон - Считается что результат анализа соответствует нормативу, если полученный результат измерений (диапазон) лежит внутри указанного интервала значений.

Вариант результата анализа определяется реквизитом «Вид результата анализа» типа план видов характеристик «Виды результатов анализа номенклатуры».
Реквизит «Единица измерения» - является необязательным и служит для указания единиц измерения числовых результатов при формировании печатных форм.
