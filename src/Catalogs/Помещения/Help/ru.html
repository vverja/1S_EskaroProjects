Справочник «Помещения»

Справочник предназначен для хранения списка помещений расположенных на определенной территории предприятия. Территории, на которых  размещены помещения выбираются из справочника-владельца «Территории».

Данные о помещениях используются при планировании занятости и резервировании помещений для проведения различных мероприятий при помощи документа  «Резервирование помещений», а также для указания места проведения встреч при работе с документом «Событие».

Информацию о занятости помещений предприятия можно получить при помощи отчета  «Планируемая занятость помещений».
