Настройка источника данных

Настройка источника данных выполняется для наборов данных типа "Объект". Источники данных предназначены для заполнения данными набора данных.

Источник данных может быть следующих типов:

  • Отчет. Выбирается отчет из списка отчетов конфигурации на базе универсального или на базе типового отчетов.
  • Произвольный отчет. Выбирается отчет из списка произвольных отчетов.
  • Внешняя обработка. Выбирается внешняя обработка.

При выборе источника данных автоматически заполняются параметры. В списке парметров допускается редактирование имени параметра схемы.

При нажатии на кнопку "ОК" выполняется заполнение полей набора данных и параметров схемы компоновки.

Требования к внешней обработке

Обработка должна содержать

  • Функция ПолучитьДанные(ПараметрыИсточникаДанных, НастройкиКомпоновкиДанных). Функция предназначена для получения данных источника данных.
    Где,
    • ПараметрыИсточникаДанных - таблица значений (см "Процедура ЗаполнитьСписокПараметров")
    • НастройкиКомпоновкиДанных - настройки компоновки данных
  • Процедура ЗаполнитьСписокПолей(СписокПолейИсточника). Процедура предназначена для заполнения списка полей источника данных. Вызывается для заполнения полей набора данных схемы компоновки данных.
    Где,
    • СписокПолейИсточника - таблица значений, содержащая колонки: "Поле", "Заголовок", "ТипЗначения", "ВыражениеПоляИтога", "ЭтоРесурс".
  • Процедура ЗаполнитьСписокПараметров(СписокПараметровИсточника). Процедура предназначена для заполнения списка параметров источника данных. Вызывается для заполнения параметров.
    Где
    • СписокПараметровИсточника - таблица значений, содержащая колонки: "Имя", "Заголовок", "ОграничениеИспользования", "Выражение", "ТипЗначения".
