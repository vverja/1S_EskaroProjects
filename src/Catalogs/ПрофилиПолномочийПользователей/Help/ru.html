Профили полномочий пользователей

Для выполнения в рамках системы своих функциональных обязанностей сотруднику требуется определенный набор полномочий - комбинация:

  • прав доступа к объектам (определяется ролями)
  • прав к функциональным возможностям программы (определяется доп. правами)

Профиль полномочий пользователя (далее – просто профиль) – совокупность полномочий (ролей и дополнительных прав), необходимых для выполнения пользователем конкретных функциональных обязанностей в рамках системы.

 

Примеры профилей:

  • Менеджеры по продажам: ввод документов продажи (с правом отражения документов в регл. учете)
  • Кассиры: ввод кассовых документов
  • Старшие кассиры: ввод кассовых документов, ввод заявок на расходование денежных средств

 

Как правило один профиль включает несколько ролей и дополнительные права.

Каждому пользователю назначается только один профиль. Назначение профиля приводит к тому, что пользователь получает полномочия определяемые профилем.

Интерфейс

Роли

Содержит роли текущего профиля. Дополнительные роли подсвечиваются синим цветом.
Роль "Пользователь" автоматически назначается всем пользователям (роль не доступна для выбора).
Если профиль не содержит ролей, то пользователям будет назначена только роль "Пользователь".

Дополнительные права

В нижней части формы расположено табличное поле, содержащее дополнительные права.

Кнопка "Копировать" позволяет копировать выбранные права в другие профили. После нажатия на кнопку открывается форма профилей, в которой необходимо выбрать профили (можно несколько) для которых нужно скопировать права.

Если в состав ролей профиля пользователя входит роль "Полные права", то редактирование доп. прав не допускается. В этом случае все доп. права считаются разрешенными.

Назначение профиля пользователям

Для назначения профиля множеству пользователей рекомендуется использовать сервис "Групповая обработка объектов", который можно открыть по кнопке "Назначить пользователям" из подменю "Администрирование".

Список пользователей, которым назначен выбранный профиль, можно открыть по кнопке "Показать пользователей с текущим профилем" .

Обмен профилями

В подменю "Администрирование" находятся средства для обмена профилями между базами:

  • Выгрузить профили - позволяет выгрузить профили в файл обмена.
  • Загрузить профили - позволяет загрузить профили пользователей из файла обмена, который был сформирован при помощи сервиса "Выгрузить".
