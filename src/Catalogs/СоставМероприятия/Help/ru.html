Справочник «Состав мероприятия»

Справочник «Состав мероприятия»  является подчиненным справочнику «Мероприятия». В нем для каждого  элемента из справочника-владельца  «Мероприятия»  можно добавить новые или изменить перечень составляющих его событий. Для каждого составляющего события задается его продолжительность в виде даты и времени начала и даты и времени окончания. В процессе заполнения даты начала и даты окончания выполняется контроль, чтобы указанные даты не выходили за границы периода проведения мероприятия. Элементы справочника «Состав мероприятия» используются при заполнении документа «Участие в мероприятиях».
