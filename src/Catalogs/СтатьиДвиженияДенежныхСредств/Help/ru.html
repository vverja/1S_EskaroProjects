Справочник «Статьи движения денежных средств»

Данный справочник используется для ведения учета движений денежных средств по их видам. Ведение такого учета необходимо для автоматического заполнения "Отчета о движении денежных средств по МСФО" прямым способом. Для заполнения регламентированного отчета о движении денежных средств по НСБУ 4 справочник не используется, потому что отчет заполняется непрямым способом.

В реквизите «Вид движения» указывается соответствующий определяемому виду движения показатель, используемый для построения отчетности (данные показатели уже предопределены в конфигурации).

Справочник «Статьи движения денежных средств» используется для ведения аналитического учета на счетах денежных средств:

  • 30 «Касса»;
  • 31 «Счета в банках».
