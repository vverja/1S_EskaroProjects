Справочник "Статьи неоперационных расходов"

Справочник "Статьи неоперационных расходов" предназначен для ведения аналитического учета на счетах неоперационных расходов. Такими счетами являются:

  • счет 85 "Другие затраты" для организаций, использующих только 8-й класс счетов расходов;
  • счета 95-99 для организаций, применяющих 9-й класс.

Состав справочника определяется произвольным образом в соответствии с потребностями учета.
