Справочник «Статьи оборотов по бюджетам»

Справочник является ключевым справочником подсистемы бюджетирования.

В соответствии с элементами этого справочника детализируются обороты компании при бюджетировании и анализе фактической информации.

При настройке справочника, необходимо указать следующие данные:

  1. На закладке «Основная»:
    • наименование элемента
    • признаки учета количественных и суммовых оборотов при отражении операций по данной статье
    • признаки учета разделения оборотов по контрагентам и номенклатуре при отражении операций по данной статье
  2. На закладке «Проводки»: шаблон (шаблоны) проводок по плану счетов бюджетирования, которые будут автоматически формироваться при отражении операций по данной статье:
    • счет дебета;
    • счет кредита;
    • коэффициент к исходному значению оборота по статье (суммовому и количественному), который будет использоваться при формировании текущей проводки.
    • Значения аналитических разрезов, выбранных для счетов бюджетирования, подставляются при формировании проводки автоматически.
  3. На закладке «Данные»: источник данных расчетов бюджетирования, ресурсы которого будут использованы при трансляции фактических данных учетных подсистем в оборот по данной статье подсистемы бюджетирования при получении фактических данных:
    • наименование источника;
    • ресурс (показатель) источника для получения суммовых значений;
    • ресурс (показатель) источника для получения количественных значений;
    • способ определения валюты оборота при отражении фактических данных; возможны четыре варианта.


Первые два варианта используются в случае, когда пользователь точно знает, в какой валюте отражаются в учетной подсистеме обороты по выбранному ресурсу источника:

  • валюта регламентированного учета;
  • валюта управленческого учета;

Вариант «Валюта источника данных» выбирается в случае, когда валюта оборотов присутствует в измерениях источника (или ссылках на измерение, например, «Валюта взаиморасчетов» в измерении «Договор взаиморасчетов» регистра «Расчеты с контрагентами»). Для этого, при настройке источника должна быть установлена соответствующая связь.

При указании варианта «Основная валюта статьи» в обороты будет подставляться валюта, используемая по умолчанию для  данной статьи оборота.

Также, на закладке «Основная» можно указать дополнительные данные, используемые для облегчения ввода бюджетной транзакции для данной статьи:

  1. Основные значения аналитики бюджетирования (ЦФО, проект, контрагент, номенклатура), которые будут по умолчанию подставляться в реквизиты документа «Бюджетная операция» при выборе данной статьи оборотов.
  2. Ресурс источника данных, который будет подставляться в реквизит «Цена» документа «Бюджетная операция» при выборе данной статьи оборотов; имеет смысл только в том случае, когда по статье учитывается и количественный, и суммовой оборот
