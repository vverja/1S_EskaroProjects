Справочник "Статьи отсроченных налоговых активов и обязательств"

Справочник "Статьи отсроченных налоговых активов и обязательств" используется для описания статей отсроченных налоговых активов (ОНА) и обязательств (ОНО). По этим статьям производится выявление временных разниц и расчет ОНА и ОНО в соответствии с НСБУ 17 "Налог на прибыль", а также ведение аналитического учета на счетах 17 "Отсроченные налоговые активы" и 54 "Отсроченные налоговые обязательства".

В справочнике предусмотрены предопределенные статьи, расчет ОНА и ОНО по которым выполняется специальным образом и заложен в конфигурацию:

  • "Авансы покупателей" и "Авансы поставщиков". Временные разницы по этим статьям возникают из-за отличий в моменте определения базы доходов и расходов по налоговому и бухгалтерскому учету по операциям с поставщиками и покупателями. Например, доход в бухгалтерском учете возникает в момент отгрузки покупателю, а в налоговом учете по "первому событию". Информация о временных разницах по этим статьям берется из остатков на забалансовых бухгалтерских счетах РВК "Временные разницы по взаиморасчетам с покупателями" и РВП "Временные разницы по взаиморасчетам с поставщиками".
  • "Нематериальные активы" и "Основные средства". Временные разницы по этим статьям определяются путем сравнения балансовой стоимости ОС и НМА по налоговому учету и остаточной стоимости ОС и НМА по бухгалтерскому. При этом остаточная стоимость по бухгалтерскому учету определяется только для тех ОС и НМА, которые принимают участие в расчете налоговой амортизации (производственных).

Помимо предопределенных статей в справочник можно добавить произвольные статьи в соответствии с потребностями учета. Конфигурация будет автоматически рассчитывать ОНА и ОНО по таким статьям, если заполнены реквизиты Счет оценки по бух. учету и Счет оценки по нал. учету. Каждый из этих реквизитов задает счет, остаток по которому будет использоваться в качестве оценки статьи по бухгалтерскому и налоговому учету соответственно. Один из счетов может быть не указан, в этом случае соответствующая оценка будет считаться нулевой. Разница между двумя оценками и есть времменная разница. Необходимым условием расчета ОНА и ОНО является то, что по указанным в реквизитах счетам оценки должен вестись аналитический учет в разрезе Видов налоговой деятельности.

К примеру, для расчета отсроченного налогового актива по обеспечению на гарантийный ремонт необходимо: в плане счетов добавить вид субконто "Виды налоговой деятельности" на счет 473 "Обеспечение гарантийных обязательств", и добавить в справочник новую статью, в которой Счет оценки по бух. учету установить 473, а Счет оценки по нал. учету оставить пустым.

Выявление временных разниц и расчет ОНА и ОНО производится документом "Расчеты по налогу на прибыль".
