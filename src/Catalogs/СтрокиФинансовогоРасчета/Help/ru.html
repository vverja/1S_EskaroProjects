Справочник «Строки финансового расчета»

Справочник «Финансовые расчеты» и подчиненный ему справочник «Строки финансового расчета» предназначены для настройки сводных отчетов финансовых отчетов по данным подсистемы бюджетирования или других учетных подсистем. Основная настройка механизма заключается в настройке структуры справочника «Строки финансового расчета». Можно сказать, что форма списка этого справочника, подчиненного какому – либо элементу справочника «Финансовые расчеты», после сортировки по кодам элементов совпадает с формой отчета, который будет сформирован.

При настройке элемента справочника, указывается:

  • наименование, с которым строка будет выведена в отчет;
  • вид строки: указывается способ заполнения данных по строке бюджета.

В зависимости от значения второго реквизита, различается порядок дальнейшей настройки строки расчета.

  • «Обороты по статье оборотов»: в этом случае, в отчет будет выведен оборот по статье (группе статей) оборотов, которые указываются в реквизите «Статья оборотов»;
    »
  • Остатки или обороты по плану счетов»: в этом случае, в отчет будут выведены данные по плану счетов бюджетирования, определяемые реквизитами «Счет» и «Вид остатка / оборота»;
    »
  • Расчет по другим строкам отчета»: в этом случае, в отчет будет выводиться величина, полученная в результате расчета по формуле, указанной в реквизите «Формула»; заполнение формулы производится по нажатию на кнопку выбора в соответствующем поле ввода;
    »
  • Источники данных для расчетов»: этом случае, в отчет будут выводиться данные по ресурсу источника данных, указанным в реквизитах «Источник» и «Показатель».
