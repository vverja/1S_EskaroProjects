Справочник "Схемы назначения использования"

Справочник предназначен для указания назначений использования переданной в эксплуатацию специальной одежды, специальной оснастки и инвентаря.

Заполнение реквизитов

  • Наименование - произвольный текст. Наименование назначения использования.
  • Вид материала - к какой категории  будет относится материал по назначению использования:
    • Малоценный быстроизнашивающийся предмет;
    • Малоценный необоротный актив;
    • Библиотечный фонд.
  • Счет передачи - счет бухгалтерского учета по назначению использования (счет на котором будет числится материал после передачи в эксплуатацию). Для МБП - не указывается. Для малоценных необоротных активов и библиотечных фондов были заведены специальные субсчета с аналитикой по номенклатуре и количественным учетом ("1122"," 1112"). Эти счета подставляются по умолчанию, при изменении вида материала.
  • Счет амортизации - счет бухгалтерского учета амортизации по назначению использования. На этом счете будет числится амортизация, начисленная после передачи в эксплуатацию малоценных нематериальных активов и библиотечных фондов.
