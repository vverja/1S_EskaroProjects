Справочник «Точки маршрута»

Точка маршрута – это совокупность подразделения предприятия и рабочего центра. Точки маршрута используются для планирования производства. В спецификациях изготовления продукции для каждого выходного изделия можно указать точку маршрута, где это изделие будет произведено, а для каждой комплектующей точка маршрута определяет место, где комплектующая будет использована.
