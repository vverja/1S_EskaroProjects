Справочник «Шаблоны формул расчета»

Справочник предназначен для хранения шаблонов формул расчета нормативного количества комплектующих при вводе спецификаций изготовления продукции. В форме ввода формулы расчета, вызываемой из спецификации, заполнение формулы можно выполнить по данным шаблона. Для этого в форме вода формулы предназначена кнопка «Заполнить» - «Заполнить из шаблона».

В справочнике «Шаблоны формул расчета» задается сама формула расчета, наименование формулы и комментарий.
