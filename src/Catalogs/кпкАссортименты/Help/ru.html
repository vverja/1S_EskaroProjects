Справочник «Ассортименты товаров»

Используется для привязки ассортимента номенклатуры к торговым агентам. Содержит ассортименты товаров, закрепленные за агентом через регистр сведений "Сведения агента". Для торгового агента выгружаются только те товары, которые включены в его ассортимент. Использование справочника необязательно.

Редактирование происходит в диалоговой форме элемента справочника.  Элемент справочника имеет табличную часть, в которой могут храниться как группы, так и элементы из справочника "Номенклатура". Для удобного выбора элементов из справочника "Номенклатура" можно использовать кнопку "Подбор", расположенную на командной панели нижней части формы элемента справочника "Ассортименты".
