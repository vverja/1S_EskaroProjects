Справочник «Виды заданий»

Вспомогательный справочник. Используется при заполнении документов "Задания агента".

Имеет иерархическую структуру, количество уровней иерархии ограничено 2-мя уровнями, таким образом возможно создавать только группы и внутри групп создавать только элементы.

При добавлении нового элемента справочника нужно указать тип значения для добавляемого вида задания.
