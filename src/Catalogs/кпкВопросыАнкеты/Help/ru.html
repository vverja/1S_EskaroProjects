Справочник «Вопросы анкеты»

Данный справочник содержит информацию для проведения анкетирования контрагентов. Элементы этого справочника используются в документе «Посещение».

Редактирование данных справочника производится в диалоговом окне, открывающимся отдельно. 

В поле "Наименование" вводится вопрос для анкеты. На форме есть 2 флажка - "Выгружать в МУ" и "Обязательный вопрос".

Флажок "Выгружать в МУ" регулирует признак выгрузки вопроса в МУ.

Флажок "Обязательный вопрос" означает, что при формировании документа "Посещение" в мобильном устройстве ответ на данный вопрос будет обязательным условием для сохрания документа в базе данных мобильного устройства.
