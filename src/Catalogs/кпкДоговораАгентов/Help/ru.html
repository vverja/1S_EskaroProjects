Справочник «Договоры агентов»

Используется для хранения списка договоров, привязанных к торговым агентам. Использование справочника является обязательным в том случае, если на форме "Настройка параметров работы с мобильным приложением" в поле "Режим выгрузки контрагентов" указано значение "Из договоров агента".

Редактирование происходит в форме списка справочника. Рекомендуется заполнять список договоров для агента в форме записи регистра сведений "Сведения агента".
