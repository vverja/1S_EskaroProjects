Справочник «Карты»

Справочник "Карты" используется для хранения описаний карт, используемых для отображения маршрута торгового агента.

Редактирование значений элементов справочника происходит в отдельном диалоговом окне, открываемом из формы списка справочника.

Данный справочник предназанчен для хранения карт различных форматов в зависимости от приложения, которое используется для просмотра карт.

Если используется приложение компоненты APPlus.dll, тогда на форме элемента справочника необходимо указать любой графический файл карты и выполнить привязку карты, после чего внести точки привязки карты к GPS-координатам - для этого на форме присутствуют кнопки вызова обработки встроенного конвертера координат Перевод градусов в GPS-координаты.

Графический файл карты может быть любым, например, взятым с сайта Google Maps, или отсканированный с обычной бумажной карты, атласа и т.п. Главное условие- правильно указать в справочнике точки привязки карты к GPS-координатам.

Таким образом за каждым агентом может быть закреплена отдельная карта - чтобы была возможность отслеживать перемещения агента в зависимости от территории, за которой закреплен агент. Выбор карты для агента происводится в регистре сведений "Сведения агента". 

В члучае использования приложение Map9 Viewer выбирается файл карты в формате m9. Привязка карты в этом случае производится с помощью другого приложения - Map9 Editor.
