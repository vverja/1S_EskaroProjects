Справочник «Категории документов»

Справочник предназначен для описания категорий документов, выгружаемых в МУ.

К каждой категории документов привязывается цвет, которым документ будет выделяться в журнале документов в МУ.

На форме можно указать список документов, на которые данная категория документов будет действовать.  Таким образом можно выделять категории документов применительно к видам документов. Например, для документа "Приходный кассовый ордер" не имеет смысла использовать категорию "Товар доставлен", а имеет смысл закреплять за документами вида "Заказ покупателя" или "Реализация товаров и услуг".

С помощью этого справочника также устанавливать категории не только непосредственно на сами документы, но и для документов долгов, которые создаются на их основании (например, для документа "Реализация товаров и услуг").

Например, если требуется выделять различные категории как на сам документ, так и на документ догла, то требуется внести 2 элемента этого справочника, для одного из которых будет установлен флажок "Использовать для долгов". 

Категории для документов и для документов долгов задаются в регистре сведений "Сведения о загруженных документах" в колонках "Категория" и "Категория для долга" соответственно.
