Справочник «Команды системы»

Справочник "Команды системы" используется для выполнения различных пользовательских или администраторских команд приложением "1С:Предприятие". Выполнение команд производится поредством встроенного механизма регламентых заданий.

В данной конфигурации может применяеться для любых целей, в частности, для автозапуска расчета истории продаж ночью и подготовки истории продаж к выгрузке для каждого торгового агента. Также для выполнения обмена данными между МУ и информационной базой в автоматическом режиме по заданному расписанию.

При формировании новой команды необходимо указать, в первую очередь, название команды. Для удобства можно формировать команду с помощью шаблона. Для этого нужно выбрать необходимый шаблон в поле "Наменование" из справочника "Шаблоны команд". В поле "Ответственный" указывается пользователь, от имени которого будет запущена текущая команда.

Для составления команды нужно написать код команды на встроенном языке программирования 1С:Предприятия. Задание будет запускаться только в том случае, если будут заданы настройки расписания запуска текущей команды. Команды будет выполняться согласно настройкам периодического обмена данными. Если информационная база работает в файл-серверном режиме, то периодический обмен выполняется только у пользователя, который указан в параметрах учетной политики как "Пользователь для регламентных заданий в файловом режиме". В Клиент-серверном варианте обмен выполняется на сервере 1C:Предприятия.
