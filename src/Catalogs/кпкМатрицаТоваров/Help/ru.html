Справочник «Матрица товаров»

Справочник предназначен для формирования списка номенклатуры (номенклатурных групп) с их рейтингами, которые учитываются при подборе товаров в МУ. Рейтинг – это гибкая характеристика товара, зависящая от категории торговой точки или клиента. Информация о рейтинге представлена в матрице в виде названия  рейтинга и числовой характеристики.

Также для рейтинга указывается цветовая характеристика в справочнике «Рейтинги товаров». В зависимости от указанного цвета для рейтинга в МУ номенклатура с данным рейтингом будет отображаться в виде цветных треугольников в колонке «Товар» правее от названия товара.       

Этот справочник необязательно применять только для отображения рейтингов товаров, но и просто для выделения каких-либо специальных товаров по различным критериям, но в зависимости от категории  клиента или торговой точки.

При формировании документа нужно указать список агентов, для которых будет действовать текущая матрица товаров, а также указать период действия матрицы. Если не заполнено окончание периода, то матрица будет выгружаться в МУ постоянно, начиная с даты, указанной в поле «Начало периода». Если на форме выбрана категория клиента (торговой точки), то  матрица будет действовать только для клиентов (торговых точек) с указанной категорией. Если не выбирать категорию, то матрица будет действовать для всех клиентов (торговых точек), у которых не указана категория (категории для контрагентов и торговых точек удобно задавать с помощью справочника «Категории контрагентов» или с помощью регистра сведений «Сведения о контрагентах» для контрагентов, а для торговых точек в справочнике «Торговые точки».

Для заполнения и редактирования списка номенклатуры используются кнопки, расположенные на командной панели:

 

  • «Подбор»     - открывает подбор по номенклатуре или номенклатурным группам;. 
  • «Заполнить» - производит заполнение текущей матрицы на основании другой матрицы или по ассортименту из справочника «Ассортименты»;

  • «Очистить»   - очищает табличную часть целиком.
