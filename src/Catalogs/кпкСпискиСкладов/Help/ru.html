Справочник «Списки складов»

Справочник содержит списки складов, по которым выгружаются  остатки товаров для торгового агента.

Список складов закрепляется за агентом посредством регистра сведений Сведения агента.

Для каждого агента можно выбрать т.н. «основной» склад. Если у склада будет поставлен этот признак, то при загрузке в ЦБД документы будут оформляться на этот склад.

Редактирование происходит в диалоговой форме элемента справочника.  Для удобного выбора элементов из справочника "Номенклатура" можно использовать кнопку "Подбор", расположенную на командной панели нижней части формы элемента справочника "Списки складов".
