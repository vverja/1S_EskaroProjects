Справочник «Список сертификатов номенклатуры»

Справочник предназаначен для указания списка сертификатов для номенклатуры, которые нужно выгружать в мобильные устройства торговых агентов с дальнейшей возможностью их дальнейшей распечатки. Для одной позиции номенклатуры может быть указано неограниченное количество сертификатов. Список тех сертификатов, которые требуется выгружать в в МУ регулируется флажком "Выгружать" в списке сертификатов.

Данный справочник подчинен основному справочнику "Номенклатура". Сами же сертфикаты при добавлении нового сертификата в список выбираются из отдельного справочника "Сертификаты номенклатуры".

Для удобства информирования пользователей те сертификаты, у которых закончился срок действия выделяются красным цветом в строке справочника.
