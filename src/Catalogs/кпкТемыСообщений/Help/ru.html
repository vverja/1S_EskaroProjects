Справочник «Темы сообщений»

Данный справочник предназначен для хранения тем сообщений, предназначенных для выгрузки агентам.

При создании нового элемента справочника обязательно следует заполнить следующие реквизиты:

1. Тему сообщения;

2. В список агентов указать, для каких агентов предназначена данная тема сообщения; 

3. Выбрать цвет, которым будет подсвечиваться тема сообщения;

4. Дату создания сообщения (по умолчанию указывается текущая дата)

Также следует установить или сбросить флажок "Выгружать", если требуется сооветственно либо выгружать, либо не выгружать данное сообщение.

На вкладке "Сообщения" отражается список сообщений, для которых указана данная тема.
