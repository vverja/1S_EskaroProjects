Справочник «Торговые точки»

Справочник «Торговые точки» предназначен для хранения информации о торговых точках контрагентов. Справочник "Торговые точки" подчинен справочнику «Контрагенты».

Переход к списку торговых точек контрагента производится по кнопке «Перейти»/ «Торговые точки» в форме элемента или списка соответствующего справочника. 

Редактирование информации о торговых точках производится в отдельном диалоговом окне, открываемом из формы списка справочника.

При этом при необходимости указывается информация, характеризующая торговую точку: общая информация, контактные данные, а также дополнительная информация. 

Следует обратить внимание на реквизит "Договор" - он предназначен дял указания договора, который закреплен за торговой точкой. При загрузке документов из МУ в документ будет проставлен указанный договор. Если в константе "Режим выгрузки контрагентов" указан режим "По договорам агента", то при загрузке документов из МУ, сначала будет производиться поиск по договорам, закрепленным за агентом, и, если такой не найден, то уже будет подставлен договор, указанный для торговой точки.

При использовании GPS в системе для отображения флажков торговых точек на карте следует указать также следующие параметры:

• Широта, Долгота - GPS-координаты местоположения торговой точки;

• Цвет, которым будет отображаться флажок расположения торговой точки на карте.
