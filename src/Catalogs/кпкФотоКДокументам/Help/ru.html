Справочник «Фотографии документов»

Справочник предназначен для хранения фотографий документов, полученных от торговых агентов.

Записи в справочнике формируются автоматически при обмене данными. При этом поля «Наименование» и «Имя файл фото в МУ» заполняются автоматически, по умолчанию загружаются с одинаковыми названиями,  наименование фото может быть изменено, поле "Имя файла фото в МУ" недоступно для редактирования, поскольку фотографии в МУ имеют свою системную нумерацию, и при обмене данными привязываются к документам автоматичски.  

Редактирование элементов справочника производится в диалогом окне справочника. В реквизите «Комментарий» проставляются комментарии, сделанные в МУ при создании фотографий. В реквизиты «Дата и время» проставляются дата и время создания фотографий в МУ.

В форме списка справочника имеется возможность осуществить отбор по необходимому документу. Для этого нужно выбрать в поле «Фильтр по документу» тип необходимого документа, после этого в открывшемся списке нужно выбрать необходимый документ.

Кнопка «Просмотр», расположенная на форме списка справочника предназначена для просмотра фотографий просмотрщиком операционной системы.

За каждым документом может быть закреплено неограниченное количество фотографий.
