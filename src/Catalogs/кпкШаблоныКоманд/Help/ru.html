Справочник «Шаблоны команд»

Вспомогательный справочник, используется для заполнения справочника "Команды".

Формируется шаблон текста команды, который пишется на встроенном языке программирования 1С: Предприятия.

Существует несколько предопределенных команд, которые можно использовать в качестве примера:

1. Автообмен с "Агент Плюс СОД" – команда запуска обработки "Обмен данными с мобильными устройствами" с признаком автообмена с "Агент Плюс СОД.

2. Отправка данных через "Агент Плюс СОД" – команда автоматической выгрузки данных всем агентам через "Агент Плюс СОД".

3. Прием данных через "Агент Плюс СОД" – команда автоматического приема данных от всех агентов через "Агент Плюс СОД".

4. Отправка данных по ФТП-серверу – команда автоматической выгрузки данных всем агентам через ФТП-Сервер.

5. Прием данных по ФТП-серверу – команда автоматического приема данных от всех агентов через ФТП-Сервер.

6. Отправка данных по почте серверу – команда автоматической выгрузки данных всем агентам через почту.

7. Прием данных по почте – команда автоматического приема данных от всех агентов через почту.

8. Расчет истории продаж – автоматическая выгрузка данных по истории продаж для всех агентов.

9. Очистка треков за период – очитска регистра сведений "Сведения по GPS-трекам" за указанный период.
