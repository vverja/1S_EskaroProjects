План видов расчета «Взносы в фонды»

Содержит список взносов в фонды социального страхования как сотрудников организаций так и взносы на фонд оплаты труда.

Основная задача элемента плана видов расчета - определить структуру базы. То есть указать какие виды начисления входят в базу для взноса. Если состав базы налога меняется необходимо создать новый вид расчета плана видов расчета "Взносы в фонды" и правильно определить изменившуюся структуру. Затем прописать данный вид расчета в регистре сведений ВидыРасчетовВзносовВФонды.

Элементы даного плана видов расчета подразделяются на несколько смысловых групп: взносы работников и взносы на ФОТ и т.д. Принадлежность той или иной группе определяется по реквизиту Способ расчета.

Реквизит "Способ расчета по шкале" определяет методику заполнения и расчета взносов (как для ФОТ так и для взносов сотрудника). Если установлено значение "Прогрессивный" - в документе "Начисление зарплаты работникам организаций" будет заполнено столько строк, сколько существует актуальных порогов по шкале взноса. Далее сумма будет расчитана по прогрессивной шкале

Например: Шкала от 0 до 100 - 1%; от 100,01 до 200 - 2%; от 200,01 до 999999999999,99 - 3%. Сумма облагаемого дохода 500грн.

Для способа "По шкале" при заполнении будет получена одна запись. При расчете получим результат = 500*3% = 15грн

Для способа "Прогрессивный" при заполнеии будет получено 3 записи (Порог1, Порог2, Порог3).

При расчете получим следующие результаты:
Результат Порог 1 = 100*1% = 1 грн;
Результат Порог 2 = (200-100)*2% = 100*2% = 2 грн;
Результат Порог 3 = (500-200)*3% = 300*3% = 9 грн;
Общий результат по сумме 3-ех строк = 12грн


 

В конфигурации реализованы два подхода к расчету взносов:
- Первый подход - учитывает ставку по периоду регистрации;
- Второй подход - анализирует для расчета ставку актуальную для месяца, к которому относится начисление

Первый подход можно представить в виде следующих формул:
Сумма Взноса = (База всего начислено с учетом ограничений - База начисленного взноса за прошлые периоды с учетом ограничений) * Ставка
База = База всего начислено- База начисленного налога за прошлые период;
База Взноса = База всего начислено с учетом ограничений - База начисленного взноса за прошлые периоды с учетом ограничений

Второй подход можно представить в виде следующих формул:
Сумма Взноса = (Сумма всего начислено * Ставку)- Сумма начисленного взноса за прошлые периоды
База = База всего начислено- База начисленного налога за прошлые период;
База Взноса = База всего начислено с учетом ограничений - База начисленного взноса за прошлые периоды с учетом ограничений

Выбор Первого или Второго подхода регулируется реквизитом "Учитывать ставку по периоду регистрации"
