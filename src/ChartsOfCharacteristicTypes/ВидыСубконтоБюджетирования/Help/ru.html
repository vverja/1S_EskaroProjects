Виды субконто бюджетирования

План видов характеристик «Виды субконто бюджетирования» предназначен для установки признаков аналитического учета счетам плана счетов «Бюджетирование». Имеет пять предопределенных видов субконто:

  • ЦФО
  • Проект
  • Контрагент
  • Номенклатура
  • Статья оборотов

Состав предопределенных видов субконто полностью соответствует аналитическим измерениям бюджетной транзакции, что позволяет автоматически формировать проводки при записи транзакции без запроса значений аналитики для дебетуемого и кредитуемого счета.

Таким образом, изменять состав видов субконто бюджетирования не рекомендуется (добавленные виды субконто невозможно будет обслуживать без серьезного изменения программы).
