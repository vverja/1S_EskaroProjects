Форма «Параметры авторезервирования»

В данной форме выполняется настройка параметров для выполнения автоматического резервирования

  • Резервирование - при установке флага выполняется автоматическое резервирование товаров на складах
  • Размещение - при установке флага выполняется автоматическое размещение товаров в заказах поставщикам
  • Стратегия авторезервирования по заказам - определяет, в какой очередности необходимо выполнять резервирование. Имеет смысл, если выполняется Резервирование и Размещение.
  • Очищать размещение перед заполнением - при установке данного флага перед авторезервированием будет очищена колонка «Размещение»

Примечание:
При использовании сервиса авторезервирования в документе «Корректировка заказа покупателя» при установке флага «Очищать размещение перед заполнением» колонка «Размещение» будет очищена только для строк с положительным количеством. Для строк с отрицательным количеством данная колонка при авторезервировании не заполняется, поэтому не должна очищаться
