Инструмент для управления аттестациями

Инструмент менеджера предназначен для основных работ над аттестациями:

  • назначение аттестации работнику;
  • контроль факта проведения аттестации;
  • контроль оценок полученных работниками;
  • анализ распределения оценок для контроля адекватности оценочных мероприятий.
