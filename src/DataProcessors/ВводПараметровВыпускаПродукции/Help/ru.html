Обработка «Ввод параметров выпуска продукции»

Форма предназначена для ввода фактических значений параметров выпуска продукции. В шапке формы для информации показывается для какой продукции вводятся параметры. Продукция выводится с точностью до характеристики и серии. Также показывается для какого объема фактического выпуска продукции вводятся параметры.

 

В табличной части «Параметры выпуска продукции» указываются параметры выпуска и их фактические значения. Табличную часть можно заполнить автоматически нормативными данными параметров, указанными в спецификации изготовления продукции. Для этого нужно нажать кнопку «Заполнить» - «Заполнить по спецификации». Далее значения параметров и сам перечень параметров можно откорректировать вручную.
