Обработка «Ввод формулы»

Форма предназначена для ввода формул расчета нормативного количества комплектующих в спецификациях. При вводе формулы все математические и логические операции, а также числа и аргументы формулы вводятся не через клавиатуру, а нажатием соответствующих кнопок на форме ввода.

В качестве аргументов формулы можно использовать значения параметров выпуска продукции, либо свойства характеристик номенклатуры, либо значение реквизита «Количество» из табличной части «Исходные комплектующие» спецификации. Важно, что тип значений параметров и свойств характеристик номенклатуры должен быть либо числовым, либо логическим.

Подстановка значений аргументов в формулу осуществляется двойным щелчком мыши по выбранной позиции аргумента. Причем возможна подстановка либо кода, либо имени выбранного аргумента. Это регулируется кнопкой «Коды/имена» на панели инструментов.

Для того чтобы в формулу подставить в качестве аргумента значение реквизита «Количество» из табличной части спецификации следует нажать кнопку «Параметры» - «Количество».

Формулу можно также ввести используя шаблоны формул. Шаблон формулы выбирается из справочника «Шаблоны формул расчета» при нажатии кнопки «Заполнить» - «Заполнить из шаблона».
