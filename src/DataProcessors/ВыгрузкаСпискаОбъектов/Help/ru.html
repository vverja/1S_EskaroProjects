Выгрузка списка объектов

Обработка предназначена для выборочной выгрузки объектов.

Выгрузка выполняется в файл путь, к которому указывается в поле "Файл данных".

Объекты, предназначенные для выгрузки, указываются в списке "Выгружаемые объекты". При выборе группы объектов будут выгружены все входящие в нее элементы.
