Обработка «Загрузка почтовых сообщений»

Обработка предназначена для загрузки электронных писем из основного почтового клиента операционной системы во встроенного почтового клиента конфигурации. При загрузке писем создаются документы «Электронное письмо». Загрузка электронных сообщений производится основным почтовым клиентом операционной системы по технологии Simple MAPI.

Необходимо указать учетную запись из встроенного почтового клиента конфигурации, для которой будут создаваться электронные письма. По нажатию кнопки «Заполнить» производится получение данных о электронных письмах из основного почтового клиента операционной системы и заполнение табличного поля. Создание электронных писем встроенного почтового клиента производится по нажатию на кнопку «Зарегистрировать» нижней командной панели формы.
