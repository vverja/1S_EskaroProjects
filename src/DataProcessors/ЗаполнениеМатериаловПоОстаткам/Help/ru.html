Обработка "Заполнение материалов по остаткам"

Обработка предназначена для заполнения табличных частей  "Материалы" документов. Она вызывается из формы документа по нажатию кнопки "Заполнить по остаткам". В зависимости от установленных флагов "Отражать в ..." в вызывающем документе, обработка считывает остатки из регистров накопления "Партии материалов в эксплуатации (управленческий учет)", "Партии материалов в эксплуатации (бухгалтерский учет)", Партии материалов в эксплуатации (налоговый учет)" и заполняет максимальным значением из них  табличную часть .  Для считывания остатков возможно использовать отбор.

Отбор

Отбор данных отчета. В каждой строке задается один элемент отбора, содержащий:

  • Флажок использования элемента отбора. Позволяет легко отключать и включать отдельные элементы отбора без удаления и добавления элементов.
  • Поле, на значения которого накладываются условия отбора.
  • Вид сравнения – условие отбора.
  • Значение или значения для сравнения. Для условий отбора на вхождение в интервал задается двумя значениями – границами интервала. Если накладывается условие на вхождение в список, значения сравнения задаются списком значений.
