Настройка программы «Использовать регистр «Свободные остатки»»

Контроль остатков товаров на складах может выполняться по регистру «Свободные остатки». Этот регистр содержит сводные данные по регистрам «Товары на складах», «Товары в рознице», «Товары в резерве на складах», «Товары к передаче со складов» (далее регистры-основания).

При выборе «Использовать регистр «Свободные остатки»» требуется ввести дату, начиная с которой регистр будет заполнен данными регистров-оснований. Например, при вводе даты 01.01.2007 регистр «Свободные остатки» будет содержать остатки на 01.01.2007 и движения, начиная с этой даты. При вводе даты следует учитывать, что последующие изменения данных до этой даты могут привести к рассогласованию регистра. Поэтому необходимо обеспечить запрет изменения данных регистров-оснований до введенной даты.

При подтверждении использования регистра «Свободные остатки» будут выполнены следующие действия:

  • Если в информационной базе существуют движения по регистру «Свободные остатки», эти движения будут удалены

  • Будут записаны остатки документом «Корректировка записей регистров» на выбранную дату.

    Если режим устанавливается не первый раз, то рекомендуется предварительно удалить созданные ранее документы по вводу остатков.

  • Если в системе есть документы, по которым были сформированы движения по регистрам-основаниям более поздней датой, чем дата, начиная с которой заполняется регистр «Свободные остатки», то по этим документам будут записаны движения в регистр «Свободные остатки».

После того, как установлено использование регистра «Свободные остатки», можно проверить согласованность данных с помощью отчета Анализ регистра «Свободные остатки».

При отключении использования регистра «Свободные остатки» изменения данных регистра «Свободные остатки» не происходит.
