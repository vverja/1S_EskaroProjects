Настройка обмена данными

В форме выполняется настройка обмена данными. Форма открывается из меню "Сервис" - "Настройка обмена данными".

Обмен данными
  • Префикс номеров документов и справочников текущей информационной базы - префикс нужен для поддержания уникальности номеров во всех информационных базах, для которых организован обмен данными. Рекомендуется устанавливать префикс в том случае, если документы или элементы справочников могут быть созданы в нескольких информационных базах. В том случае, если объекты информационной базы создаются только в одном из узлов, участвующем в обмене, установка префикса необязательна.
  • Префикс внутреннего штрихкода - указывается, если предполагается обмен данными между информационными базами, чтобы избежать дублирования внутренних штрихкодов.
  • Использовать механизм автообмена данными при возникновении определенных событий - при установке флага обмен данными будет запускаться в автоматическом режиме при возникновении определенных событий на клиентском месте (например, при входе в программу, завершении работы или появлении каталога).
  • Сохранять информацию о месте создания объектов информационных баз - если флаг установлен, при выполнении обменов данными между информационными базами будет регистрироваться информация о том, в какой именно базе был создан каждый объект.
