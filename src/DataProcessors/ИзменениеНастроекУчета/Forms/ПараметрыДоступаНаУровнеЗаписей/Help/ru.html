Параметры доступа на уровне записей

Форма открывается из интерфейса "Администрирование пользователей" - меню "Доступ на уровне записей"-"Параметры".

В форме выполняется настройка параметров доступа к данным на уровне записей:

  • Ограничить доступ на уровне записей по видам объектов - позволяет ограничить доступ к данным на уровне записей по видам объектов для групп пользователей.
