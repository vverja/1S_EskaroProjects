Инструмент для утверждения отпусков

Инструмент позволяет провести процедуру утверждения для каждого отпуска в отдельности и документа целиком. Вызывается инструмент из меню «Планирование».

В этом инструменте реализованы следующие возможности:

  • собирает и предоставляет данные для формализации принятия решения об утверждении отпуска,
  • проводит проверки корректности ввода отпусков с построением отчета об ошибках и предупреждениях,
  • графическая иллюстрация текущей ситуации с планируемыми отпусками,
  • подготовка отчета о принятых решениях.
