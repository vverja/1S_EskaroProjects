Обработка «Конструктор спецификаций»

Обработка предназначена для создания и редактирования спецификаций номенклатуры на всех переделах и визуализации дерева спецификаций.

 

Закладка «Настройка»

 

На закладке производится установка реквизитов, влияющих на работу конструктора спецификаций.

  • Выводить спецификации на дату - дата, на которую определяются основные спецификации номенклатуры.

 Группа реквизитов «Настройка дерева спецификаций»:

  • Флажок «Редактировать в диалоге» определяет способ редактирования строки спецификации. При установке данного флажка редактирование строки спецификации производится в отдельном окне.
  • Флажок «В списках чередовать цвета строк» - при установке данного флажка каждая вторая строки всех списков конструктора спецификаций будет выделяться  цветом.
  • Флажок «Показывать дерево номенклатуры» определяет видимость дерева групп номенклатуры на закладке «Конструктор спецификаций».
  •  Флажок «Разворачивать дерево спецификаций» - при установке данного флажка на закладке «Конструктор спецификации» при формировании дерева спецификаций будет выполняться раскрытие всех уровней дерева.
  • Флажок «Выводить выходные изделия» определяет вывод строк выходных изделий в дереве спецификаций.
  • Флажок «Выводить возвратные отходы» определяет вывод строк возвратных отходов в дереве спецификаций.
  • Выводить в доп. информацию для номенклатуры - признак вывода в дереве спецификаций для номенклатуры артикула или кода.
  • Количество уровней разворота дерева - дерево спецификаций будет развернуто до уровня, указанного в данном поле. Если в поле указано значение 0, то дерево спецификаций будет развернуто полностью по всем уровням.

 

Группа переключателя «При вводе новой строки» определяет действие при интерактивном вводе новой строки исходных комплектующих дерева спецификации:

  • Добавлять новую строку спецификации - всегда добавляется новая строка в дерево спецификации.
  • Выводить меню - выводится меню:
    • Добавить новую строку - добавляется новая строка в дерево спецификаций.
    • Добавить новую спецификацию - создается новая спецификация номенклатуры. После сохранения, спецификация устанавливается как основная для указанной номенклатуры.

 

Группа «Запрашивать подтверждения»:

  • Флажок «Запрашивать подтверждение при смене спецификации» - при установке данного флажка будет выдаваться дополнительный вопрос при установке спецификации в качестве основной.

 

Закладка «Конструктор спецификаций»

 

На закладке производится редактирование дерева спецификаций номенклатуры.

 

Дерево спецификаций

 

В табличном поле выводится дерево спецификаций номенклатуры по всем переделам. В первой строке дерева спецификаций выводится конечная продукция.

  • Наименование - наименование номенклатуры или наименование табличной части спецификации (исходные комплектующие, выходные изделия, возвратные отходы).
  • Характеристика - характеристика номенклатуры.
  • Спецификация - установленная основная спецификация номенклатуры.
  • Количество - количество исходных комплектующих, выходных изделий или возвратных отходов.
  • Единица - единица измерения количества.
  • Статья затрат - статья для учета затрат по исходным комплектующим или возвратным отходам.

В дереве спецификаций поддерживается ввод и редактирование строк исходных комплектующих, выходных изделий и возвратных отходов.

 

По кнопке «Печать» производится формирование печатных форм:

  • Дерево спецификаций - формируется печатная форма полного дерева спецификаций.
  • Покупные изделия - формируется печатная форма полного списка покупных изделий всего дерева спецификаций. В печатную форму выводится номенклатуры с выбранным видом воспроизводства «Покупка».
  • Полуфабрикаты - формируется печатная форма полного списка полуфабрикатов всего дерева спецификаций. В печатную форму выводится номенклатуры с выбранным видом воспроизводства «Производство».
  • Отчет по дереву спецификаций – формируется отчет, в который выводится вся информация по спецификации номенклатуры. Выбор полей отчета, группировка и отбор информации выполняется в форме отчета по кнопке «Настройка…«. Этот отчет может быть вызван не только из формы конструктора спецификаций, но и из перечня отчетов в меню программы.

 

Номенклатура

 

Для заполнения дерева спецификаций доступны следующие списки:

  • Дерево групп номенклатуры - видимость данного списка зависит от установки флажка «Показывать дерево номенклатуры».
  • Список номенклатуры.
  • Список характеристик номенклатуры - видимость данного списка зависит от настроек использования характеристик номенклатуры в конфигурации.
  • Список спецификаций номенклатуры.

 

Список номенклатуры

 

При выборе строки в списке номенклатуры производится формирование дерева спецификаций для данной номенклатуры.

 

Поддерживается перетаскивание из указанных списков в дерево спецификаций. При перетаскивании с нажатой основной (по умолчанию левой) кнопкой мыши производится добавление новой строки исходных комплектующих, выходных изделий или возвратных отходов. При перетаскивании с нажатой дополнительной (по умолчанию правой) кнопки мыши выводится меню:

  • Изменить номенклатуры - производится изменение номенклатуры в текущей строке.
  • Добавить новую строку - добавляется новая строка исходных комплектующих, выходных изделий или возвратных отходов.

 

Список спецификаций номенклатуры

 

В списке полужирным шрифтом выделяется спецификация, которая установлена как основная для текущей номенклатуры в списке номенклатуры. Основная спецификация определяется на дату, заданную на закладке «Настройка».
