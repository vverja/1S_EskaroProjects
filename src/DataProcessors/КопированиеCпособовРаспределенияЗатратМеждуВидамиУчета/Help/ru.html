Данная обработка предназначена для копирования  способов распределения статей затрат из регистра сведений "Способы распределения статей затрат" в регистр сведений "Способы распределения статей затрат организаций". Либо наоборот. 

      Перед началом выполнения обработки необходимо заполнить следующие реквизиты на закладке "Основное":

  • Направление копирования (из бухгалтерского в управленческий или из управленческого в бухгалтерский)
  • Организацию. При этом, если копирование выполняется из бухгалтерского учета в управленческий, этот реквизит показывает данные какой   организации будут получены для копирования в управленческий учет. Если копирование выполняется из управленческого учета в бухгалтерский, то именно для указанной организации будут сформированы новые записи
  • Период выбираемой информации. Если конечная дата будет не указана, то данные будут выбраны по текущую дату 
  • Период новых записей. При этом период может быть либо равным периоду переносимых записей, либо какая-то определённая дата.

       На закладке "Отбор" можно указать дополнительный фильтр копируемой информации по подразделению и статье затрат.

     Для выполнения обработки необходимо нажать на кнопку "Выполнить". На основании информации о соответствии подразделений бухгалтерского и управленческого учета (регистр сведений "Соответствие подразделений и подразделений организаций" ) будут сформированы записи в регистре сведений "Способы распределения статей затрат"(если записи копируются из бухгалтерского учета в управленческий)  или в регистре сведений "Способы распределения статей затрат организаций" (если записи копируются из управленческого учета в бухгалтерский).

     Информация в регистре сведений будет изменена безвозвратно.
