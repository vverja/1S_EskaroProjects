Обработка «Копирование данных бюджетирования»

Обработка предназначена для создания новых бюджетных операций и документов расчета по моделям бюджетирования на основании уже существующих в подсистеме. Возможны несколько вариантов использования обработки:

  • Копирование из интервала в интервал. В этом случае, интервалы отбора исходных данных и формирования новых равны и кратны наибольшей из периодичностей сценариев, выбранных в качестве исходного и принимающего. При этом варианте копирования, указывается возможно указать абсолютное изменение исходных оборотов по количеству и сумме и (или) коэффициент, на который будут умножены исходные обороты при копировании (таким образом, возможна реализация формулы n=a+kb, где n - новые обороты, b - исходные обороты, а - абсолютное изменение, k - коэффициент), а также определить, будут ли исходные данные равномерно распределяться по периодам новых в случае, когда периодичность исходного сценария больше периодичности принимающего (например, при копировании из плана с периодичностью в квартал в план с периодичностью в месяц).
  • Распределение нормированных значений. При таком варианте копирования, интервал отбора исходных данных изменяется кратно наибольшей из периодичностей сценариев, выбранных в качестве исходного и принимающего, а интервал принимающих данных кратен периодичности принимающего сценария. Исходными данными для заполнения принимающего интервала будут суммарные обороты за исходный интервал, деленные на количество периодов принимающего сценария, укладывающиеся в исходный интервал копирования. Далее, необходимо указать, как будут заполняться периоды в принимающем интервале. Возможны несколько вариантов заполнения:
    • равномерное заполнение с указанием значения базового коэффициента, на который будет умножено исходное значение для заполнения;
    • с базовым изменением коэффициента для каждого периода (для первого периода коэффициент равен базовому, для каждого следующего периода коэффициент получается суммированием значения коэффициента предыдущего периода и заданного приращения, т.е. ki = ki-1 + p = k1 + (i-1) * p, где k1 – базовый коэффициент, p - приращение);
    • с цепным изменением (для первого периода коэффициент равен базовому, для каждого следующего периода коэффициент получается суммированием значения коэффициента предыдущего периода и приращения, вычисляемого умножением коэффициента предыдущего периода на заданную величину, т.е. ki = ki-1 + ki-1 * p = ki-1 * (1 + p));

Важно: В обоих вариантах копирования возможно указать правила отбора исходных данных для копирования по измерениям бюджетирования и (или) правила установки новых значений аналитики бюджетирования для формируемых транзакций.

Важно: при формировании новых транзакций происходит их анализ на предмет существования зависимых статей оборотов; причем, структура зависимостей и правила формирования зависимых транзакций берутся из установленных для принимающего интервала копирования.

Также, при копировании из интервала в интервал возможен перенос всех документов расчета по модели бюджетирования (необходимость определяется соответствующим флагом). Как и в случае формирования бюджетных операций, скопированные документы расчетов будут пересчитаны и перепроведены в соответствии с новой датой расчета по документу.

После выполнения копирования необходимо:

  • перейти на закладку «Новые бюджетные операции», поставить флажки у  всех сформированных операций или только необходимых и провести их нажатием соответствующей кнопки;
  • перейти на закладку «Новые документы расчета по модели» поставить флажки у  всех сформированных документов или только необходимых, рассчитать и провести их нажатием соответствующих кнопок.

Важно: при повторном копировании, созданные ранее в этом же сеансе работы с обработкой документы не удаляются. При необходимости, их можно принудительно удалить нажатием соответствующих кнопок на закладках с таблицами сформированных документов.
