Набор персонала

Инструмент предназначен для работы с заявками кандидатов.

 

Основная область инструмента предназначена для работы с заявками кандидатов. Вспомогательная область позволяет разбирать электронную почту, пришедшую по электронной почте на адреса с назначением "Переписка с кандидатами". Если работа с электронной почтой не ведется, область с неразобранными сообщениями можно скрыть.

 

Основная область состоит из нескольких частей:

- В левой части отображается дерево заявок кандидатов, отсортированных и сгруппированных по реквизиту заявки. Реквизит, по которому группируется и сортируется список, выбирается пользователем. Такими реквизитами могут быть: дата события, состояние, наименование, вакансия, ответственный, подразделение, должность и группа заявок.

- В правой часте отображаются: описание заявки, список писем по заявке и текст выбранного электронного письма. В отличие от списка неразобранных писем, в списке писем по заявке отображаются письма по всем электронным адресам, а не только по адресам с назначением "Переписка с кандидатами".

Менять реквизиты заявки можно непосредственно из описания заявки, а также из меню Заявки и с помощью контекстного меню.

Кроме изменения реквизитов заявки доступны такие функции как:

  • Переписка по заявке
  • Планирование и просмотр запланированных встреч
  • Создание физического лица и сотрудника по заявке
  • Закрытие заявки

 

Область работы с неразобранными письмами состоит из двух частей:

- В левой части отображается дерево списка писем, отсортированных и сгруппированных по по дате или по отправителю, в зависимости от выбора пользователя.

- В правой части отображается текст выбранного электронного письма.

На основании письма можно сразу оформить заявку кандидата. Тогда это письмо исчезнет из списка неразобранных и будет прикреплено к созданной заявке.

Выбранное письмо можно отметить как нежелательное. Такое письмо будет перенесено в папку "Нежелательные".

 

Из формы инструмента можно открыть справочники "Вакансии" и "Источники информации", а также инструмент "Кадровое планирование".
