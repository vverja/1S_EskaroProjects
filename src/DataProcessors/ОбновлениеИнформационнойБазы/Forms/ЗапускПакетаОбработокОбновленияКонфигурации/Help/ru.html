Обработка «Запуск пакета обработок обновления конфигурации»

Используется в контексте обновления на новые версии конфигурации.

Обработка предназначена для пакетного запуска набора обработок, которые должны запускаться после обновления конфигурации до начала работы. Целесообразно использовать данную обработку в случае, когда последовательно запускать отдельные обработки вручную является неудобным либо неоптимальным.

Список обработок отображается в виде дерева. При пакетном запуске будут выполняться только обработки, отмеченные в списке флажками. Для того чтобы определить, какие обработки необходимо выполнить, нужно найти в дереве узел с номером версии, с которой выполняется обновление, и установить в нем пометку. При этом будут помечены все нижестоящие узлы.
