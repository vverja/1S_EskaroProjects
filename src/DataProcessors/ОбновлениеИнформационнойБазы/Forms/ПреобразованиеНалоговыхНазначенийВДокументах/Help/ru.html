Обработка «Обновление содержимого регистра сведений «Объекты доступа документов»

Обработка выполняет заполнение добавленного измерения «Вид объекта доступа» для всех существующих записей регистра сведений «Объекты доступа документов».
Обновление необходимо для правильного функционирования механизма ограничения прав доступа на уровне записей к журналам документов.
