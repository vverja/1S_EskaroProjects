Попередній перегляд

Форма "Попередній перегляд" призначена для перегляду й редагування сформованих для друку аркушів регламентованих звітів. Ця форма відкривається при виборі пункту "Показати бланк" з підменю "Друк", що відкривається при натисканні відповідної кнопки нижньої командної панелі діалогу регламентованого звіта.

Форма складається із двох основних частин

У лівій частині форми виводиться список сформованих аркушів для друку й розташовані керуючі елементи для управління відображенням сформованих аркушів і виведенням їх на друк

У правій частині форми відображаються сформовані друковані аркуші регламентованого звіту. Одночасно показується тільки один аркуш - той, на якому встановлений курсор у списку аркушів в лівій частині форми

У списку аркушів, що друкуються, є можливість вибору виведених на печатку аркушів і зміни послідовності печатки аркушів (кнопки "Перемістити нагору" і "Перемістити вниз").

Для зміни позначок одночасно у всіх аркушів використовуйте кнопки "Зняти позначки" і "Встановити позначки", розташовані над списком аркушів, що друкуються.

Для навігації по сформованих аркушах зручно користуватися кнопками "Наступний" і "Попередній".

Для редагування сформованих аркушів необхідно встановити курсор на потрібному аркуші в списку аркушів, що друкуються - обраний аркуш відобразиться в табличному полі.
Для зручності редагування використовуйте прапорці "Відображати сітку" і "Відображати заголовки", розташовані в лівому нижньому куті форми

Основні кнопки розташовані на нижній командній панелі форми

Кнопка "Друк" призначена для печатки зазначених аркушів, із числа сформованих аркушів відповідно до зазначених параметрів друку

в параметрах друку є можливість задати кількість екземплярів друку й порядок друку аркушів - для цього служить прапорець "Розібрати по копіях".

Якщо прапорець встановлений, спочатку буду друкуватися всі аркуші однієї копії звіту, потім - всі аркуші наступної копії, і так далі.

При знятому прапорці спочатку буде надрукований перший обраний аркуш у зазначеній кількості екземплярів, потім - другий, теж у зазначеній кількості екземплярів, і так далі.

По кнопці "Зберегти" можна зберегти обрані аркуші в зазначеному каталозі; аркуші збережуться у файлах формату табличних документів 1С:Підприємства 8.0 (розширення MXL).

Кнопка "Перейти в батьківський звіт" активізує форму, з якої була відкрита форма попереднього перегляду

Для зручності можна використовувати гарячі клавіші, що відповідають кнопкам нижньої командної панелі:
F5 - друк;
F6 - зберегти;
F12 - перейти в батьківський звіт

У лівому куті нижньої командної панелі відображаються кількість обраних аркушів і загальна кількість аркушів паперу, необхідна для друку з урахуванням установленої кількості копій
