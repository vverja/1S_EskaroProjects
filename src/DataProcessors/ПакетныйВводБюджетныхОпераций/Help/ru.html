Обработка «Пакетный ввод бюджетных операций»

Обработка предназначена для ввода группы бюджетных операций, имеющих один или несколько одинаковых параметров бюджетирования (период планирования, валюта операции, ЦФО, проект).

Реквизиты, которые будут общими для всех бюджетных операций, вводятся на закладке «Общие реквизиты бюджетных операций», другие реквизиты, значение которых будет индивидуально для каждой бюджетной операции, вводятся в табличную часть на закладке «Бюджетные операции».

Важно! Установка значения сценария является обязательной, этот разрез бюджетирования будет единым для всех бюджетных операций и устанавливается в шапке документа.

При установке признака использования общего реквизита соответствующая колонка в табличной части «Бюджетные операции» скрывается, однако значение ее, которое было введено до этого, сохраняется до конца сеанса работы обработки. То есть если сначала для какого-либо реквизита установить признак его общего использования во всех операциях, например, проект, то колонка «Проект» в табличной части «Бюджетные операции» скроется, а если потом снять признак использования общего реквизита, то колонка «Проект» появится, причем значение, которое было введено в колонку до скрытия – восстановится. В остальном порядок заполнения табличной части «Бюджетные операции» аналогичен настройке документа «Бюджетная операция». В нижней части закладки «Бюджетные операции» отображается список оборотов, зависимых от бюджетной транзакции, которая настраивается в текущий момент.

После настройки параметров формирования бюджетных операций необходимо:

  • нажать кнопку «Выполнить»;
  • перейти на закладку «Сформированные документы», поставить флажки у  всех сформированных операций или только необходимых и провести их нажатием соответствующей кнопки.

Важно! При записи бюджетных операций реквизиты, для которых установлен признак использования общего реквизита, заполняются значениями, указанными на закладке «Общие реквизиты бюджетных операций», даже если при вводе для этих реквизитов были указаны другие значения

Важно! При повторном нажатии кнопки «Выполнить», созданные ранее в этом же сеансе работы с обработкой документы не удаляются. При необходимости, их можно принудительно удалить нажатием соответствующей кнопки на закладке с таблицей сформированных документов.
