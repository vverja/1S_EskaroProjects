Обработка "Планирование"

Обработка предназначена для предварительного планирования потребности в производственных мощностях. Главное назначение:

  • формирование и корректировка производственной программы;
  • определение дефицита производственных мощностей (в различные периоды времени).

Также она может использоваться для создания и редактирования планов производства, планов продаж и планов закупок.

 

Внимание!

Перед началом расчета производственной программы следует:

  • Для всей номенклатуры, по которой требуется рассчитать производственную программу, установить активные спецификации
  • Для всех выходных изделий, запланированных к выпуску, в спецификации указать точки маршрута
  • Установить графики для рабочих центров, в которых планируется выпуск, иначе будет считаться, что они работают по регламентированному производственному календарю
  • Указать производственные мощности рабочих центров по выпускаемой номенклатуре, иначе при расчете производственной программы она будет запланирована к выпуску в один день. Важно! В колонке "Исполнитель" следует указывать рабочий центр.

Командная панель формы обработки

  • Сценарий - устанавливается отбор по определенному сценарию планирования для планов, которые отображаются на закладке "Планы" (табличное поле "Планы"). Если реквизит не заполнен, отбор по сценарию по умолчанию отключен.
  • Плановый период - дата начала и дата окончания периода, на который требуется рассчитать производственную программу.

Закладка "Планы"

Здесь можно создавать и редактировать планы различных видов, устанавливать и просматривать взаимосвязи между ними.

Табличное поле "Планы"

В виде дерева отображаются планы, сгруппированные по видам (продажи, производство, закупки), и связи между ними:

  • План - представление периода плана, который определяется периодичностью сценария и датой планирования
  • Подразделение - информация о подразделении, к которому относятся данные плана

В табличном поле отображаются только те планы, данные которых относятся к плановому периоду, выбранному в командной панели формы обработки. Некоторые позиции состава плана могут являться источниками обеспечения других планов. Для таких позиций состава плана в документе указан план-источник. Позиции состава плана, для которых указаны источники, являются зависимыми, остальные позиции - независимыми. Зависимые и независимые части одного плана располагаются в дереве планов на различных уровнях. Независимые части планов выводятся в дереве планов только один раз на втором уровне дерева. Зависимые части планов могут отображаться в списке несколько раз на различных уровнях. При этом зависимые части планов выводятся на следующем уровне иерархии тех планов-источников, для которых они являются источником обеспечения.

По кнопке "Настройка" в командной панели табличного поля "Планы" можно:

  • управлять составом отображаемых видов планов
  • установить режим, при котором к представлению периода плана будут добавлены номер и дата документа планирования

Табличное поле "Состав плана"

При перемещении по строкам списка в табличном поле "Планы" здесь выводится информация о составе текущего плана. Для независимых частей планов будет выводиться  информация только о независимых позициях состава плана, а для зависимых — только зависимые позиции. При необходимости можно выбрать режим, при котором для каждой части плана (и зависимой и независимой) будет показываться полный состав плана (кнопка "Настройка" командной панели табличного поля "Состав плана", пункт меню "Полный состав плана").

Дополнительно для позиции состава плана можно показать ее распределение по подпериодам планирования. Выбрать нужную периодичность детализации можно в реквизите "Периодичность" (командная панель табличного поля "Состав плана").

Расчет источников обеспечения и потребностей

Для части состава плана, которая выводится в табличном поле "Состав плана", можно рассчитать источники ее обеспечения (кнопка "Действия" командной панели табличного поля "Состав плана", пункт меню пункт "Рассчитать источники обеспечения"). В результате для каждой позиции состава плана будут отображены источники, за счет которых может быть обеспечена эта позиция. Можно включить режим автоматического расчета источников обеспечения (кнопка "Настройка" командной панели табличного поля "Состав плана", пункт меню "Автоматический расчет источников обеспечения").

Обеспеченные позиции состава плана отображаются в списке зеленым цветом, необеспеченные  — розовым.

Табличное поле "Потребности"

Для позиции состава плана производства  можно рассчитать потребности в исходных комплектующих (кнопка "Действия" командной панели табличного поля "Состав плана", пункт меню "Рассчитать потребности"). Рассчитанная потребность для текущей позиции состава плана будет выведена в табличном поле "Потребности". Расчет потребности производится разузлованием до первого уровня спецификации позиции состава плана. Можно включить режим автоматического расчета потребностей (кнопка "Настройка" командной панели табличного поля "Состав плана", пункт меню "Автоматический расчет источников обеспечения").

Форма "Редактирование плана"

По кнопке "Редактировать" командной панели табличного поля "Состав плана" можно открыть специальную форму, в которой будет отображен полный состав текущего плана.

Закладка "Состав плана"

Здесь выводится состав плана. В форме нельзя редактировать состав плана и количество, но можно изменить детализацию состава плана по подпериодам планирования. Если детализация была скорректирована, то при нажатии на кнопку "Записать" информация будет записана в документ планирования.

Закладка "Производственная программа"

Чтобы составить производственную программу и оценить ее выполнимость с учетом доступных производственных мощностей, сначала следует указать, какие позиции составов планов производства должны быть включены в производственную программу. В табличной части

  • указываются позиции, которые должны быть включены в производственную программу
  • определяется период, на который требуется распланировать выпуск этой позиции
  • определяются другие параметры, которые будут использоваться при расчете производственной программы (спецификация, точка маршрута, вид воспроизводства

Табличная часть может быть заполнена вручную или по составу плана производства. В табличную часть можно включить позиции, которых нет в плане производства, или исключить позиции, имеющиеся в плане. По сути, на этой закладке можно вручную составить производственную программу и не выполнять ее расчет. Позиция состава плана считается распланированной, если для нее указан рабочий центр. Распланированные позиции отображаются зеленым цветом, нераспланированные — розовым.
При нажатии на кнопку "Записать" введенные в табличную часть данные будут сохранены в исходном документе "План производства"  в дополнительной табличной части, которая не видна при интерактивной работе с документом, документ будет проведен. При проведении документа распланированные позиции состава плана будут записаны в регистр накопления "Производственная программа".

Закладка "Производственная программа"

Закладка "Плановые задания"

Здесь производится автоматический расчет производственной программы. При переходе на эту закладку  в табличной части будут показаны  позиции составов планов производства,  которые были включены в производственную программу с учетом периода планирования, выбранного в командной панели формы обработки. В табличной части будут содержаться как распланированные позиции (отображаются зеленым цветом), так и нераспланированные (отображаются розовым цветом).
Расчет производственной программы выполняется при нажатии на кнопку "Выполнить планирование" в соответствии с выбранным режимом:

  • Планировать все — выполняется пересчет всех позиций, включенных в производственную программу;
  • Планировать нераспланированные — выполняется пересчет только не распланированных ранее позиций;
  • Планировать выбранные — выполняется пересчет всех выделенных в табличном поле позиций.


При расчете производственной программы учитываются доступные производственные мощности рабочих центров, которые заданы в регистре сведений "Производственные мощности". Позиции, которые удалось распланировать, будут отображены зеленым цветом.

Закладка "Диаграмма Ганта"

Здесь полученная производственная программа представлена в виде диаграммы.

Бежевым цветом выделены выходные дни согласно регламентированному производственному календарю. Для каждого рабочего центра отображается две шкалы:

  • верхняя шкала (темно-зеленая) - загруженность рабочего центра;
  • нижняя шкала (светло-зеленая) - график рабочего центра (отображение включается по кнопке "Настройка").

О загруженности рабочих центров можно судить по отображению верхней шкалы:

  • занятость в рабочие дни (согласно графикам) - темно-зеленый цвет;
  • занятость в выходные дни - штриховка;
  • отсутствие занятости в рабочие дни - не окрашено.

На нижней шкале отсутствие занятости в рабочие дни показано штриховкой.

 

Детализацию диаграммы по периодам можно изменить. Для этого в колонке "Единица времени" следует указать  нужное значение периода (пункт меню "Настройка" при нажатии правой клавиши мыши в поле на закладке "Диаграмма Ганта").

По окончании расчета производственной программы все полученные данные будут записаны в исходные документы планирования и исходные документы будут проведены. При проведении документа данные, являющиеся распланированными, будут записаны в регистр накопления "Производственная программа".
