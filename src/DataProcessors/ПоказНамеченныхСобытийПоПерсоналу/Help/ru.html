Обработка "Показ намеченных событий по персоналу"

Обработка предназначена для подтверждения намеченных ранее событий.

 

Рекомендуется запускать эту обработку с определенной периодичностью, например, раз в месяц, когда уже поступили сведения о выходах работников на работу, возвращении в подразделения, откуда они временно переводились и т.д.

В этой обработке нужно указать период просмотра намеченных событий и кнопкой «Заполнить» обновить список событий. Далее, нужно отметить те события, которые уже произошли и их требуется зарегистрировать.

Обработка показывает события как управленческого, так и регламентированного учета. В списке событий, у событий управленческого учета поле организация не заполнено.

Кнопка «Создать» сформирует необходимые документы, которые будут заполнены в соответствии с характером намеченных событий:

  • Для подтверждения увольнений будет создан документ «Увольнение». Если увольнение происходит по управленческому учету, для увольняемых после испытательного срока будет указана именно эта причина увольнения, а для всех остальных — «по собственному желанию».
  • Для возвратившихся из отпуска или командировки будет создан документ «Возврат на работу».

  • Для временно переведенных в другое подразделение (на другую должность) — очередное кадровое перемещение.
