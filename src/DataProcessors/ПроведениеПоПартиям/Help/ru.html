Обработка «Проведение по партиям»

Предназначена для регламентированного отражения списания партий запасов по указанную дату в управленческом, бухгалтерском и налоговом учете в случае, если списание партий не производилось в момент проведения соответствующих документов (то есть в настройке параметров учета не был установлен флажок «Списывать партии при проведении документов»). Кроме того, обработка применяется для восстановления последовательностей партионного учета (независимо от того, установлен ли вышеописанный флажок), необходимость в котором может возникнуть после проведения документов задним числом.

Внимание! Если в настройках параметров учета установлен режим использования расширенной аналитики учета затрат, данная обработка не используется.

Перед выполнением обработки необходимо удостовериться, что все документы, касающиеся движения запасов, зарегистрированы в информационной базе, а также установлена учетная политика для всех необходимых видов учета в части оценки МПЗ (см. регистры сведений «Учетная политика (управленческий учет)», «Учетная политика (бухгалтерский и налоговый учет)»).

В случае, если предполагается выполнение обработки по управленческому учету, то списание по партиям будет зависеть от выбранной организации (организаций) и способа ведения управленческого партионного учета по ней. Особенности:

  • если в качестве организации выбрать <По компании>, то проведение по партиям будет выполнено по всем организациям, для которых установлен способ ведения управленческого партионного учета «По компании в целом»
  • если выбрать конкретную организацию, то проведение по партиям будет выполнено
    • для этой организации, если для нее установлен способ ведения управленческого партионного учета «По организации»
    • для организаций, у которых установлен способ ведения управленческого партионного учета «По головной организации» и в качестве головной организации в справочнике «Организации» указана выбранная

Если предполагается выполнение обработки по бухгалтерскому и налоговому учету, то проведение по партиям будет осуществлено по каждой организации, по которой в информационной базе ведется регламентированный учет. Проведение по партиям по бухгалтерскому и налоговому учету можно проводить для какой-то конкретной организации (организаций).
