Проверка наличия, получение и установка комплекта поставки обновления конфигурации

Мастер предназначен для проверки наличия, получения и установки комплекта поставки обновления.

Комплект поставки  обновления конфигурации

При выпуске новой версии конфигурации формируется дистрибутив конфигурации, включающий все необходимые файлы для первичной установки конфигурации, а также комплект поставки обновления, содержащий файлы, необходимые для выполнения обновления уже установленной конфигурации.

Комплекты поставок обновлений помещаются на диски информационно-технологического сопровождения (ИТС), выпускаемых ежемесячно. Технология поддержки конфигураций предполагает выполнение последовательных обновлений текущих конфигураций.

После выхода очередного ИТС поставщиком конфигураций могут быть выпущены новые версии. Поэтому для оперативного выполнения обновления на вэб-сервере размещается комплект поставки обновления, включающий все изменения с момента выхода последнего ИТС. Мастер выполняет проверку наличия комплекта поставки, его получение и установку.

Каждый комплект поставки описывается номером версии, датой опубликования и списком номеров версий конфигураций, для которых он предназначен.

Если текущая конфигурация своевременно не обновлялась (пропущены несколько ИТС с обновлениями) и в списке номеров номер текущей версии отсутствует, то необходимо воспользоваться обновлениями на вышедших дисках ИТС.

Описание работы

Мастер выполняет все действия поэтапно.

Переход к следующему шагу производится нажатием кнопки «Далее >», возврат - кнопкой «Назад <».

При нажатии кнопки «Далее >» выполняется очередное действие. 

На первом шаге (при открытии) на экран выводится стартовая страница.

Проверка наличия обновлений конфигурации

На втором шаге производится проверка наличия комплекта поставки обновления данной конфигурации.

Если комплект поставки обновления есть, то в форму выводится описание обновления.

Если номер текущей конфигурации не входит в список номеров версий, для которых предназначен комплект поставки обновления, то Мастер выводит сообщение о невозможности использования данного комплекта. В этом случае получение и установка комплекта поставки не выполняется.

Если номер версии комплекта поставки совпадает с номером  версии текущей конфигурации, то обновление конфигурации не требуется. В этом случае получение и установка комплекта поставки не выполняется.

Просмотр описания изменений

Для просмотра описания изменений, произведенных в новой версии, нажмите ссылку «Показать описание изменений». Описание выводится в отдельном окне.

Получение и установка комплекта поставки обновления

После проверки наличия на вэб-сервере комплекта поставки обновления для его получения и установки нажмите кнопку «Далее >».

Производится получение комплекта поставки обновления и, в случае успешного получения, запускается программа установки и осуществляется стандартная установка обновления (см. книгу «1С:Предприятие 8.0. Руководство по установке и запуску»).

Выполните действия по установке комплекта поставки обновления.

Завершите программу установки.

Для закрытия Мастера нажмите кнопку «Готово». Если установлен флажок «При закрытии запускать конфигуратор», то после закрытия формы производится запуск конфигуратора. В списке информационных баз нужно выбрать базу, которую требуется обновить и нажать кнопку «Конфигуратор».

Настройка автоматической проверки наличия комплекта поставки

Если требуется, чтобы программа автоматически выполняла проверку наличия комплекта поставки обновления (при каждом запуске), установите флажок «Выполнять проверку наличия на вэб-сервере комплекта поставки при запуске программы». Если он установлен, то при каждом запуске выполняется проверка наличия и в случае обнаружения актуального комплекта запускается данный Мастер (со второго шага).

Выполнение обновления конфигурации

Выполнение обновления конфигурации производится в режиме «Конфигуратор» (см. книгу «1С:Предприятие 8.0. Конфигурирование и администрирование»).

Краткую инструкцию для выполнения данной процедуры можно прочитать при установке обновления, если установлен соответствующий флажок.
