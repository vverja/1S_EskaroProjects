Форма инструмента «Проверка штрихкодов»

Данный инструмент позволяет осуществлять проверку заполнения накладной при помощи сканера штрихкодов или терминала сбора данных (ТСД). После выполнения проверки возможна печать документа, содержащего информацию о позициях, по которым существует несоответствие.

Порядок работы

  1. Задать необходимость удаления строк из таблицы (строки, по которым достигнуто соответствие) при помощи флажка «Не удалять совпадающие».
  2. Отсканировать этикетки со штрихкодом товаров (в случае, если для товара предусмотрен серийный номер, сканировать нужно именно его) или получить список штрихкодов из терминала сбора данных (кнопка «Загрузить из терминала»).
  3. В случае несоответствия можно активизировать пункт «Проверяемые товары» меню «Печать».
  4. Закрыть форму проверки.
