Обработка «Расчет лимитов отпуска материалов»

В этой форме настраивается стратегия расчета лимитов отпуска материалов со склада в подразделение, результаты расчета помещаются в документ «Изменение лимитов отпуска материалов».

Форма имеет три закладки:

  • Стратегия расчета. Настраиваются источники получения информации о лимитах отпуска.
  • Отбор. Настраиваются правила отбора информации.
  • Результат. Результат расчета согласно стратегиям и отборам.

 

Стратегия расчета

 

Настраиваются правила выборки данных из базы данных. Такими источниками могут быть данные о производстве, заказов, фактического расхода.

Если выбрана стратегия расчета «Планы производства в размере (%)», то за указанный период будет выбрана продукция из планов производства и согласно спецификациям изготовления выполнено разузлование до последнего передела, до уровня материалов. Тем самым лимиты отпуска материалов будут равны объему материалов, необходимому по нормативам для производства запланированного объема продукции.

Для стратегий расчета «Планы продаж в размере (%)», «Заказы покупателей в размере (%)», «Внутренние заказы в размере (%)» оценка объемов материалов выполняется аналогичным образом.

Лимиты отпуска можно установить исходя из фактического расхода материалов за некоторый период времени. Для этого используются стратегии расчета «Расход со склада в размере (%)», «Расход на выпуск в размере (%)».

Одновременно могут быть применены несколько стратегий расчета, результаты расчета в этом случае объединяются. Итоговый результат расчета можно изменить на заданный процент и округлить до выбранной точности.

 

Отбор

 

Настраиваются правила отбора информации, полученной согласно выбранным стратегиям расчета. Отбор информации всегда производится по данным склада и подразделения. По умолчанию значения склада и подразделения заполняются данными из шапки документа «Изменение лимитов отпуска материалов».

В табличной части «Отбор» можно указать условия отбора для полей. Перечень полей зависит от того, какой источник был выбран в качестве стратегии расчета. При выборе поля в списке доступных полей будет показано сначала название стратегии, а потом название поля.

 

Результат

 

После настройки стратегий расчета и правил отбора следует на закладке Результат нажать кнопку «Рассчитать». Табличная часть будет заполнена результатами расчета лимитов согласно правилам. Результаты автоматического расчета можно изменить вручную.

Для массовой установки значения периодичности лимита или признака контроля лимита всем строкам табличной части предназначена кнопка «Изменить…«.

Для того чтобы результат расчета бал помещен в табличную часть документа «Изменение лимитов отпуска материалов» нужно нажать кнопку «ОК».
