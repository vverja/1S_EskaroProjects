Обработка «Расчет плановой себестоимости»

Обработка предназначена для расчета плановой себестоимости номенклатуры. Для расчета необходимо последовательно пройти все закладки по кнопке «Далее». При этом будут выполнены все действия по расчету плановой себестоимости.

 

Общие принципы расчета плановой себестоимости

 

Плановая себестоимость рассчитывается по каждой номенклатуре с детализацией по статьям затрат. Сумма по каждой статье затрат рассчитывается по заданному правилу расчета в зависимости от характера затрат и вида расходов. Сумма плановой себестоимости рассчитывается по данным управленческого учета в валюте управленческого учета.

 

Рассчитываются статьи затрат со следующим характером затрат:

  • Производственные расходы;
  • Общепроизводственные расходы;
  • Административные расходы;
  • Брак в производстве.

Для производственных затрат правила расчета задаются для каждого вида затрат:

  • Материальные;
  • Оплата труда и отчисления на соц. нужды;
  • Амортизация;
  • Прочие.

 

Закладка «Настройка расчета»

 

Правило расчета плановой себестоимости состоит из набора параметров:

  • Метод расчета - выбранный метод расчета суммы плановой себестоимости по статье затрат. Возможен выбор следующих методов:
    • По фактической себестоимости - сумма по статье затрат будет рассчитана исходя из фактической себестоимости номенклатуры за указанный период. В качестве исходных данных используются данные регистра «Затраты на выпуск продукции».
    • По плановой себестоимости - сумма по статье затрат будет взята из плановой себестоимости номенклатуры на указанную дату.
    • Не рассчитывать - сумма по данным статья затрат рассчитана не будет.
  • За период с - период исходных данных для расчета плановой себестоимости по методу «По фактической стоимости».
  • На дату - дата определения суммы плановой себестоимости по методу «По плановой стоимости».
  • Флажок «Изменить на процент» - признак изменения суммы плановой себестоимости статьи затрат на указанный процент. «Процент» - процент изменения суммы плановой себестоимости статьи затрат.
  • Флажок «Изменить на сумму» - признак изменения суммы плановой себестоимости статьи затрат на указанную сумму. «Сумма» - сумма изменения плановой себестоимости статьи затрат.
  • Флажок «Округлить до» - признак округления суммы плановой себестоимости статьи затрат. «Округлить до» - порядок округления суммы плановой себестоимости статьи затрат.

 

Закладка «Основная»

 

На закладке указываются данные, которые будут использоваться при заполнении списка номенклатуры для расчета плановой себестоимости:

  • Дата основной спецификации - на указанную дату будет определяться основная спецификация номенклатуры.
  • Флажок «Использовать свойства и категории» - при установке данного флажка в качестве фильтров можно будет выбирать свойства и категории номенклатуры и характеристики номенклатуры.
  • Список «Отбор». В списке задаются фильтры, которые будет использоваться при заполнении списка номенклатуры:
    • Поле - указывается поле, по которому будет осуществлен отбор.
    • Тип сравнения - указывается тип сравнения при установке отбора по полю.
    • Значение - указывается значение для отбора по выбранному полю.

 

При нажатии на кнопку «Далее» производится заполнение списка номенклатуры.

 

Закладка «Номенклатура»

 

На закладке указывается список номенклатуры, характеристик номенклатуры и спецификаций, для которых будет рассчитывается плановая себестоимость.

 

По кнопке «Заполнить» производится заполнение списка номенклатуры с отбором по значениям, установленным на закладке «Основная».

 

Закладка «Производственные расходы»

 

На закладке указываются установки правил расчета плановой себестоимости для статей затрат производственных расходов. Правила задаются отдельно для каждого вида расходов: материальные, оплата труда и отчисления на соц. нужды, амортизация, прочие.

 

Закладка «Косвенные расходы»

 

На закладке указываются установки правил расчета плановой себестоимости для статей затрат общепроизводственных, административных расходов и расходов по браку в производстве.

 

Закладка «Результат»

 

На закладке показывается результат расчета плановой себестоимости. Расчет плановой себестоимости выводится в виде дерева с двумя уровнями:

  • На первом уровне показывается номенклатура, характеристика номенклатуры и общая сумма плановой себестоимости.
  • На втором уровне показывается детализация плановой себестоимости по статьям затрат. Для статьи затрат выводится характер затрат и вид затрат. Сумма плановой себестоимости по статье затрат может быть изменена вручную.

По кнопке «Рассчитать» производится расчета плановой себестоимости и вывод результата расчета в список.

По кнопке «Изменить...» производится групповое изменение статей затрат в дереве результата расчета плановой себестоимости. Возможны следующие действия:

  • Добавить статью затрат - для каждой номенклатуры добавляется указанная статья затрат.
  • Изменить статью затрат - со всех строках результата расчета изменить указанную статья затрат на новую статью затрат.
  • Удалить статью затрат - во всех строках результата расчета удалить указанную строку затрат.

 

Закладка «Установить»

 

На закладке выбираются действия, которые должны быть выполнены на окончательном этапе расчета плановой себестоимости:

  • Флажок «Установить плановую себестоимость» - выполняется запись результата расчета в регистра сведений «Плановая себестоимость номенклатуры».
  • Флажок «Установить цены номенклатуры» - производится заполнение документа «Установка цен номенклатуры» результатом расчета плановой себестоимости.
  • Флажок «Сформировать отчет» - производится формирование отчета по расчету плановой себестоимости.
  • Флажок «Закрыть форму расчета плановой себестоимости» - форма расчета плановой себестоимости будет закрыта.

 

По кнопке «Далее» будут выполнены указанные действия. Если не установлен флажок закрытия формы, то будет открыта закладка «Основная» для продолжения расчета.
