Розшифровка регламентованої звітності

Для форм регламентованої звітності, які можуть заповнюватися автоматично за даними інформаційної бази, існує можливість показати алгоритм формування значень показників  звіту. Такий режим називається "розшифровкою". Режим доступний не для всіх форм звітності: формальною ознакою можливості розшифровки для конкретного регламентованого звіту є присутність кнопки "Розшифровка" у верхній командній панелі форми звіту. Розшифровка доступна після того, як звіт автоматично заповнений за даними інформаційної бази. Інформація для розшифровки накопичується в процесі автозаповнення звіту.

Форма розшифровки містить таблицю з алгоритмом обчислення значення показника, для якого викликана розшифровка, або декілька таблиць - для всіх показників форми звітності. Кожний рядок таблиці містить опис одного з доданків і значення цього доданку. Підсумкове значення показника видається в нижній частині таблиці. 

Форма розшифровки може мати кілька закладок для різних показників, якщо розшифровка викликалася декілька раз

З розшифровки можна викликати стандартні звіти для більш детального розкриття сум звіту. Якщо при наведені курсору миші на суми у вікні розшифровки курсор приймає вид лупи, тоді подвійним кліком миші буде викликаний один зі стандартних звітів - оборотно-сальдовая відомість по рахунку або аналіз рахунку, що містить дані рахунки/коррахунки, що брали участь у формуванні поточного значення доданку показника

У нижній командній панелі форми розшифровки розташований ряд кнопок для керування розшифровкою

 "Активізувати батьківський звіт" - кнопка активізує звіт, для якого була викликана ця розшифровка.

"Друк" - по натисканню цієї кнопки видається запит для вибору, розшифровку яких показників варто друкувати. Після установки прапорців для необхідних показників і натискання кпноки "ОК" будуть сформовані друковані форми для обраних показників.

"Зберегти" - кнопка дозволяє зберегти розшифровку у вигляді файлу формату табличного документа "1С:Підприємства 8.0".

"Закрити" - закриває вікно розшифровки.
