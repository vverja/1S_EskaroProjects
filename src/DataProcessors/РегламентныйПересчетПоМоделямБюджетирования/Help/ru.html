Обработка «Регламентный пересчет по моделям бюджетирования»

Обработка предназначена для выполнения пересчета и перепроведения группы документов вида «Расчет по модели бюджетирования». Для отбора документов по сценарию и дате расчета в форме обработки указываются необходимый сценарий и временной интервал.

Важно. Поскольку пересчет и перепроведение большого количества документов «Расчет по модели бюджетирования» может занять продолжительное время и вызвать серьезные изменения в данных, настоятельно рекомендуется внимательно относиться к настройке правил отбора документов для обработки.
