Форма редактирования данных распределения

Форма позволяет просматривать и редактировать данные распределения в виде кросс-таблицы (шахматки).

Значения ячеек желтого цвета можно изменять, при этом значения зависимых ячеек будут пересчитаны.

После окончания редактирования по кнопке «ОК» измененные данные будут перенесены в вызвавший форму объект.

Состав отображаемых полей, строк и колонок можно изменить в форме настройки структуры отчета (кнопка «Настройка – Настройка структуры»).

Можно создавать и сохранять разные варианты настроек отчета для пользователя. Для варианта настроек отчета можно указать описание и перечень пользователей (групп пользователей), которым будет доступен вариант настройки (кнопка «Настройка – Описание и пользователи»).

Для сохранения выполненных настроек следует нажать кнопку «Настройка – Сохранить» и выбрать одно из действий:

  • Сохранить настройку – текущий вариант отчета будет сохранен с новыми настройками
  • Сохранить настройку как – будет сохранен новый вариант отчета. В открывающейся форме указывается наименование настройки, после этого следует нажать кнопку «Сохранить»

При следующем открытии формы, она будет открыта с последней использованной данным пользователем сохраненной настройкой. Если данным пользователем форма открывается в первый раз, то будет использована первая в списке из доступных пользователю сохраненных настроек.
Можно выбрать другую доступную пользователю настройку (кнопка «Выбрать настройку»).
Для удаления настройки следует нажать кнопку «Выбрать настройку», выделить курсором нужную строку и выбрать пункт «Установить пометку удаления» контекстного меню.

Об использовании обработки редактирования данных распределения см. также Обработка «Редактирование данных распределения».
