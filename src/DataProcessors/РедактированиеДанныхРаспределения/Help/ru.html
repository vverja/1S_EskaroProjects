Обработка «Редактирование данных распределения»

Обработка предоставляет средства просмотра и редактирования данных произвольной таблицы значений в виде кросс-таблицы (шахматки). В таком виде целесообразно редактировать данные распределения, например:

  • материалов по продукции
  • планов по периодам
  • затрат по номенклатуре
  • расходов по видам деятельности.

Интерфейс редактирования данных может быть гибко настроен с использованием средств системы компоновки данных.

Обработка является сервисной и не может быть открыта из пункта «Операции» главного меню.
