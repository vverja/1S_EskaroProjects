Согласование заявок на расходование денежных средств

Для согласования заявок на расходование денежных средств необходимо выполнить следующие действия:

  • в списке выбрать нужные заявки (можно использовать множественный выбор).
  • в подменю "Изменить состояние" выбрать необходимое действие.
  • в открывшейся форме подтвердить решение, нажав на кнопу "Согласовать заявки", "Отклонить заявки", "Отложить согласование" (заголовок кнопки зависит от выбранного действия).
    При необходимости можно снять флажки с заявок, состояние которых не нужно изменять.

Настройка списка

В настройке списка предусмотрены дополнительные отборы, влияющие на список отображаемых заявок:

  • Ожидают согласования. Будут отображены заявки, которые прошли предыдущие этапы согласования и не обработаны текущим пользователем.
  • Отложены. Будут отображены заявки, по которым текущий пользователь принял решение "Отложить".
  • Ожидают согласования на предыдущих этапах. Будут отображены заявки, которые еще не согласованы на предыдущих этапах. При необходимости по таким заявкам пользователь может принять решение, минуя предыдущие этапы согласования.
  • Обработаны. Будут отображены заявки, по которым текущий пользователь уже принял решение, но которые еще не прошли следующие этапы согласования. По таким заявкам можно изменить принятое решение.

Настройки отбора и настройку отчета можно сохранять (кнопка "Сохранить параметры") и восстанавливать (кнопка "Восстановить параметры") - см "Сохранение параметров формы".

Настройка отчета

При изменении состояния заявок можно использовать произвольный отчет, который указывается в форме "Настройка отчета". Отчет формируется в форме изменения состояния заявок.
При формировании отчет, может учитывать заявки, состояние которых будет изменено. Это позволяет оценить ситуацию при изменении статуса заявок, например, хватает ли денежных средств.

Выделение цветом

Заявки в списке выделяются цветом фона:

  • Отложенные - серым фоном
  • Утвержденные - зеленым фоном
  • Отклоненные - розовым фоном
