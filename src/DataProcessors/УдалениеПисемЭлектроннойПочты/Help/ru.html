Обработка «Удаление писем электронной почты»

Обработка доступна только при использовании встроенного почтового клиента и предназначена для удаления электронных писем без установки монопольного режима.

В верхнем табличном поле отображаются электронные письма, которые помечены на удаление, в нижнем - для каждого электронного письма ссылки на него. При нажатии на кнопку «Удалить» нижней командной панели формы, программа производит контроль ссылок на помеченные на удаление электронные письма и по возможности удаляет их.
