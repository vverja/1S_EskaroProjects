Обработка «Формирование документов по данным оперативного учета»

Обработка используется для формирования по данным оперативного учета производства документов других подсистем.

Формирование документов выполняется по предварительно подготовленной настройке.

Порядок использования обработки:

  • выбрать настройку формирования документов
  • установить период обработки данных
  • нажать кнопку «Сформировать документы»

После формирования документов на закладках «Сформированные документы» и «Учтенные документы» отображаются списки документов.

Над документами из списков можно выполнять групповые операции:

  • проведение и отмену проведение,
  • изменение пометки удаления,
  • печать формы по-умолчанию.
