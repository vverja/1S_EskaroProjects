Отчет "Карта маршрута агента"

Отчет предназначен для представления маршрута передвижения агента с использованием GPS-координат, зафиксированных при проведении документов в МУ. Для получения координат из МУ требуется поддержка GPS на МУ.

Для корректного формирования отчета на форме нужно выбрать день, за который необходимо просмотреть маршрут агента, а также агента, для которого строится маршрут. При выборе агента файл карты проставляется автоматически, если он указан в регистре сведений «Сведения агентов» (в качестве карты может выступать графический файл любого формата).

1. Графическое представление маршрута

Данный вид отображения предназначен для графического представления маршрута агента на карте, указанной для агента (в качестве карты может выступать графический файл любого формата).

Маршрут отображается в виде квадратиков с цифрами (пронумерованными от 1 в порядке возрастания времени формирования документов в МУ), соединенными между собой пунктирной линией.       

При установке флажка «Отображать перемещения агента» на карте может отображаться трек перемещения агента, выполненный в виде промежуточных GPS - координат, зафиксированных в МУ (при условии включения функции трекинга в МУ). Трек агента на карте отображается в виде маленьких кружочков, соединенных между собой пунктирной линией.

Дополнительно на форме можно выбрать цвет для оформления маршрута агента (по умолчания устанавливается цвет, указанный в настройках в регистре сведений «Сведения агентов»). Также на карте отображаются флажки, которые показывают расположение фирм (GPS-координаты фирмы задаются в регистре сведений «Сведения организаций»).

Также имеется возможность отображнения флагов организаций и местоположения офисов контрагентов (торговых точек). GPS-координаты офисов и цвет, которым будут подсвечиваться флажки соответственно задаются либо в регистре «Сведения организаций», либо в справочнике «Торговые точки»).

2. Отображение маршрута в табличном документе

Также дополнительно к графическому оформлению маршрута можно сформировать детальную расшифровку маршрута и трека передвижения агента в виде списка посещений по точкам, которая выводится в табличный документ.

Для отображения отчета нужно нажать кнопку "Расшифровка". Если на форме выбран флажок «Отображать перемещения агента», то в табличный документ дополнительно выводится таблица с детализацией треков перемещения агента. 

В расшифровку выводится следующая информация: 

  • Место посещения (клиент или торговая точка, в зависимости от установленного режима);

  • Документ, созданный в точке посещения; 

  • GPS-координаты; 

  • Расстояние и время, пройденное агентом от точки до точки.
