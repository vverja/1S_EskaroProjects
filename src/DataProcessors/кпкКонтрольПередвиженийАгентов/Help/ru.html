Отчет "Контроль передвижений агентов"

Отчет предназначен для просмотра на карте GPS-данных о перемещениях агентов, событиях инициированных агентами в различных местах (например, такие события, как "создание документа", "отправка данных" и т.п.) с помощью приложения Map9 Viewer. Приложение Map9 Viewer использует карты, специально подготовленные в другом приложении – Map9 Editor.

Для корректного формирования отчета, на форме нужно выбрать период, за который необходимо просмотреть данные по агентам, а также карту, на которой будут отображены данные. В качестве карты по умолчанию проставляется карта, которая указана в форме настройки констант для Агент+, но также можно указать любую другую карту из справочника «Карты» (Внимание! Данная обработка поддерживает формирование карт только в формате "m9").

Также на форме следует указать список агентов, данные для которых будут выводиться на карту (по умолчанию список заполнятеся всеми агентами, определенными в системе). Для каждого агента задается цвет, которым будет отображаться его трек перемещения и которым будут окрашены события, инициированные агентом в определенной точке (создание документа, отправка данных в ЦБД и другие). По умолчанию цвет для каждого агента проставляется значением цвета, определнным в регистре сведений «Сведения агентов», но при желании пользователь может выбрать любой нужный цвет из списка.

Кнопка "Заполнить", расположенная на панели управления, предназанчена для заполнения таблицы агентами по умолчанию.

На вкладке "Данные" можно указать, какие данные следует выводить на карту, это могут быть:

  • Треки перемещений - точки, в которых были зафиксированы GPS-координаты в определнный момент времени. По совокупности треков строится маршрут передвижения агентов на карте.

  • Документы, сформированные агентом за указанный период, при условии, что в МУ при его формировании были определены GPS-координаты

  • Флаги организаций  - обозначение расположения офиса организации на карте. Для каждой организации в регистре «Сведения организаций» задается цвет отображения флага на карте, а также GPS-координаты.

  • Флаги контрагентов (торговых точек) - флаги с месторасположением офисов контрагентов (или торг. точек), на которых были оформлены документы за указанный перид. Координаты и цвет флажка задаются в регистре сведений «Сведения контрагентов» (для торговых точек параметры указываются в справочнике «Торговые точки»). При указании этого флажка можно выбрать режим отображения торговых точек: либо выводить полный список флагов контрагентов/торговых точек, закрепленными за агентом, либо фильтровать этот список только по контрагентам/торговым точкам из документов, попавших в указанный период.

    • Дополнительно на этой форме имеется возможность формировать файлы настроек для приложения Map9 Viewer в виде отдельных файлов с возможностью запуска приложения  Map9 Viewer с выбранными настройками вручную, например, если у руководителя не установлена система "1С: Предприятие", но установлено приложение Map9 Viewer.

    Для этого на панели управления, расположенной внизу формы есть подменю "Дополнительно", в нем следует выбрать пункт меню "Сохранить файлы параметров". При выборе этого пункта меню следует выбрать каталог для сохранения файла настроек.

    Для запуска Map9 Viewer без участия 1С: Предприятия потребуется загрузить в приложение Map9 Viewer сохраненные файлы, а также загрузить файл карты, указанный на форме текущей обработки.
