Обработка "Контроль передвижений агентов"

Данная обработка предназначена для осуществеления контроля над передвижениями мобильных сотрудников.

На форме присутствует панель настройки, поля ввода исходных данных , а также область отображения карты.

Панель настройки

Панель настройки доступна при нажатии кнопки «Настройка», которая находится на командной панели.
Панель включает в себя следующие элементы:

  • Поле «Источник карты»
    В поле «Источник карты» производится выбор поставщика карты. Доступны карты от трех поставщиков: Google («Google maps»), Yandex(«Яндекс карты») и OpenStreetMap.
  • Поле «Интервал деления треков»
    Данное поле предназначенно для формирования треков на основе данных с регистра "Сведения по GPS-трекам". Если величина указанная в поле меньше интервала времени между смежными отчетами с КПК, то производится разбиение трека. При изменении интервала, обновить список треков можно с помощью кнопки «Обновить список треков».
  • Кнопка «Настройка флагов»
    Позволяет настроить отбор отображенния объектов на карте. Отбор производится по контрагентам, организациям, торговым точкам, а также видам документов.
    Сведения о контрагентах содержатся в регистре «Сведения о контрагентах». В рамке группы «Настройки для GPS» необходимо ввести координаты месторасположения в соответствующие поля, а также выбрать иконку отображения контрагента. При этом, если при введенных координатах GPS не выбрана иконка контрагента, то производится его отображение на карте с использованием иконки отображения контрагентов по умолчанию, которая доступна для изменения с кнопки «Настройка иконок». В противном случае, если координаты GPS не заданы, производится соответствующее оповещение.
    Сведения об организациях содержатся в регистре «Сведения организаций».В рамке группы «Настройки для GPS» необходимо ввести координаты месторасположения в соответствующие поля, а также выбрать иконку отображения организации. При этом, если при введенных координатах GPS не выбрана иконка организации, то производится ее отображение на карте с использованием иконки отображения организаций по умолчанию, которая доступна для изменения с кнопки «Настройка иконок». В противном случае, если координаты GPS не заданы, производится соответствующее оповещение.
    Сведения о торговых точках содержатся в справочнике «Торговые точки». На закладке «Дополнительно», в рамке группы «Координаты» необходимо ввести координаты месторасположения в соответствующие поля, а также выбрать иконку отображения торговой точки. При этом, если при введенных координатах GPS не выбрана иконка торговой точки, то производится ее отображение на карте с использованием иконки отображения торговых точек по умолчанию, которая доступна для изменения с кнопки «Настройка иконок». В противном случае, если координаты GPS не заданы, производится соответствующее оповещение.
    Сведения о документах содержатся в регистре "Сведения о загруженных документах" . Отображение документа определяется иконкой документов, которая доступна для изменения с кнопки «Настройка иконок». Координаты отображения определяются соответствующими полями «Широта» и «Долгота». При этом, «Агент-автор» у документа должен совпадать с агентом, выбранным по поле «Агент».
    Галочка «Отображать флаги согласно плану посещений» позволяет отображать план посещений агента на текущую дату трека. План посещений содержится в документе «Планирование для агентов».
  • Кнопка «Настройка иконок»
    Позволяет произвести выбор иконок отображения объектов, в формате (*.png).
  • Рамка «Добавление объекта на карту»
    Позволяет добавить геграфические сведения об объекте учета путем щелчка по карте и нажатием кнопки "+".
  • Рамка «Обновление данных»
    Позволяет включить отображение местоположения агента в режиме реального времени. Данные обновления определяюся регистром "Сведения по GPS-трекам".
  • Поле «Объединить документы ближе чем:»
    Поле позволяет объединить значки документов на карте, если расстояние между ними меньше расстояния указанного в поле. Также происходит слияние иконок точек маршрута вблизи указанного расстояния от документа.
  • Команда «Открыть папку иконок»
    Позволяет открыть папку хранения иконок расположенной по адресу: «C:\Icons_AgentPlus».

Поля ввода исходных данных

Поля ввода исходных данных доступны для редактирования при нажатии на кнопку «Настройка треков». 

  • Поле «Агент»
    Позволяет произвести выбор агента, данные о котором хранятся в регистре сведений «Сведения агента».
  • Поля «Показать треки за период» 
    Позволяют произвести выбор отображения треков агента за период указанный в полях «с» и «по». Также, определить период можно с помощью более удобного инструмента нажатием кнопки «...».
  • Поле «Список треков агента»
    Отображает треки выбранного агента.

Область отображения карты

Вид области, а также набор инструментов работы с картой определяется ее источником: Google, Yandex или OpenStreet.

  • Карта Google
    Карта содержит следующие инструменты работы: кнопки навигации, кнопки выбора режима отображения карты, строку поиска объектов на карте, область отображения дальности просмотра карты.
    Также, при щелчке по карте в любом месте отображаются координаты точки в том формате, который используется для занесения сведений об объекте учета.
    Кнопки навигации расположенны в левом верхнем углу и предназначены для перемещения по карте.
    Кнопки выбора режима отображения карты находятся в правом верхнем углу и позволяют отображать карту в вариантах: «Карта», «Спутник» и «Рельеф». Режим «Карта» позволяет отображать карту с названиями объектов. Режим «Спутник» отображает карту в виде снимков с космоса. Режим «Рельеф» позволяет отобразить карту в виде снимков с космоса с названиями объектов.
    Строка поиска объектов на карте позволяет найти объект по введенному в поле адресу, например: «Москва, ул. Гагарина 5». Естественно, если данные об объекте отсутствуют у сервиса поставщика карты, то объект найден не будет.
    Область отображения дальности просмотра находится в левом нижнем углу и указывает на текущее расстояние до поверхности Земли.
  • Карта Yandex
    Карта содержит следующие инструменты работы: кнопки навигации, кнопку измерения расстояния на карте, кнопки выбора режима отображения карты, строку поиска объектов на карте, область отображения дальности просмотра карты и миниатюрное окно отображения карты.
    Также, при щелчке по карте в любом месте отображаются координаты точки в том формате, который используется для занесения сведений об объекте учета.
    Кнопки навигации расположенны в левом верхнем углу и работают аналогично картам «Goolge».
    Кнопка измерения расстояния на карте расположенна справа от кнопок навигации и определить расстояние маршрута, трека и т.д.
    Кнопки выбора режима отображения карты, а также строка поиска объектов на карте работают аналогично картам «Goolge».
  • Карта OpenStreet
    Карта содержит следующие инструменты работы: кнопки навигации, кнопку настройки и область отображения текущих координат, область отображения дальности просмотра карты.
    Кнопки навигации работают аналогично двум предыдущим картам и располагаются в левом верхнем углу.
    Кнопка настройки располагается в правом верхнем углу и позволяет осуществлять контроль над отображением меток и треков карты.
    Область отображения текущих координат расположена в правом нижнем углу и показывает долготу и широту курсора соответсвенно.
