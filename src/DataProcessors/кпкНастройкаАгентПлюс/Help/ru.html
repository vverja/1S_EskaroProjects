Обработка "Настройка мобильного приложения на устройстве"

Данная обработка позволяет настроить приложение «Агент Плюс» на МУ, подключенном к настольному компьютеру через прямое соединение. Обработка позволяет установить способ обмена и его настройки, настроить отображение и редактирование справочников, отображение вкладок в журнале документов, установить настройки документов, настроить работу с маршрутами а также установить / удалить базу данных в МУ.

На форме данной обработки отображаются три вкладки: "Настройки", "Журнал" и "БД".

Вкладка "Настройки"

На этой вкладке происходит общая настойка ПО, установка способа обмена, настройка для справочников, документов, настройки для GPS и другие дополнительные настройки.

Вкладка "Журнал"

На этой вкладке можно регулировать состав видов документов, которые будут отображаться в МУ, а также устанавливать дополнительные настройки для журналов документов.

Вкладка "БД"

Здесь можно настраивать список баз данных, которые будут использоваться в МУ. Можно установить или удалить БД в МУ.

 

На командной панели формы присутствуют следующие кнопки:

1. "Установить в МУ". В результате нажатия на эту кнопку будут переданы выбранные настройки в МУ при условии, что он подключен напрямую.

2. "Получить с МУ". Будут получены текущие настройки с МУ при условии, что он подключен напрямую.

3. Подменю "Прочее" содержит следующие пункты:

3.1. "Сохранить общие настройки" - сохраняет текущие настройки во временный файл на жестком диске с возможность указания пути сохранения файла. Удобно использовать эту функцию в качестве шаблона настроек для агентов.

3.2. "Загрузить общие настройки" - загружает настройки из указанного временного файла, сохраненного с помощью пункта меню, описанного выше.

3.3. "Сохранить файл реестра" - сохраняет файл с настройками реестра для послдующего его импортирования в МУ. Имя файла обязательно должно иметь имя "config.reg". Для дистанционной загрузки сохраненного файла в МУ агента, его следует поместить в архив с файлом выгрузки для МУ From1C.zip.

4. Подменю "Приложение "Агент Плюс" содержит следующие пункты:

4.1. "Установить "Агент Плюс" - если устройство подключено напрямую, то с помощью этой команды можно произвести установку ПО Агент+ на устройство. Дистрибутив ПО Агент+ должен находиться в папке "C:\Program Files\1cv81\bin\PocketPC\Distrib\"

4.2. "Регистрация "Агент ПЛюс" - если устройство подключено напрямую, то с помощью этой команды можно произвести регистрацию установленной программы "Агент Плюс". При этом регистрационный файл для Агент+ должен находиться в папке "C:\Program Files\1cv81\bin\PocketPC\Distrib".
