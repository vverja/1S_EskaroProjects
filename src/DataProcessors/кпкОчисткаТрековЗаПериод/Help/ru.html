Обработка "Очистка треков за период"

Сервисная обработка - предназначена для очистки треков перемещения агентов (агента) за выбранный период.

Производит очистку данных из регистра сведений "Сведения по GPS-трекам".

Для корректного формирования обработки на форме нужно выбрать период, за который необходимо очистить треки перемещения агента (агентов), а также указать список агентов, для которых следует очистить треки (по умолчанию список заполняется всеми агентами, определенными в системе).
