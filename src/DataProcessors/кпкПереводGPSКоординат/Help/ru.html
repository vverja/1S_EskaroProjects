Обработка "Перевод градусов в GPS-координаты

Обработка предназначена для перевода географических координат в формат, принятый в GPS.

Интерактивный запуск из пунктов меню этой обработки не предусмотрен. Данная обработка запускается из формы элемента справочника "Карты" при формировании точек привязки.
