Журнал «Отложенное проведение документов»

Журнал предназначен для ввода, просмотра и редактирования документов, для которых применяется отложенное проведение.

Отображаются документы:

  • отраженные в бухгалтерском учете
  • только тех организаций, для которых используется отложенное проведение
  • начиная с самой ранней из дат начала отложенного проведения

В журнал попадают все документы, в том числе непроведенные и помеченные на удаление.
Проведенные документы показываются разными пиктограммами в зависимости от того, проведены они полностью (стандартная пиктограмма) либо требуют допроведения (пиктограмма желтого цвета).

Есть возможость:

  • провести документ полностью по всем регистрам (меню «Действия» пунт «Провести полностью», либо с помощью контекстного меню)
  • сформировать диаграмму отложенного проведения (кнопка «Отчеты»).

Журнал доступен пользователям с ролью «Выполнение регламентных операций (регл. учет)».
