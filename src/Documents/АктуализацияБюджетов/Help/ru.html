Документ «Актуализация бюджетов»

Документ предназначен для выравнивания запланированных оборотов в части бюджета, по которой уже исполнены хозяйственные операции, по фактическим.

При проведении документа делаются корректирующие проводки по плану счетов бюджетирования и регистру «Обороты бюджетов», выравнивающие плановые и фактические обороты по актуализируемому сценарию и интервалу.

Существует возможность при актуализации бюджетов переносить (полностью или частично) разницу между планом и фактом на последующий период (периоды) бюджетирования. Сумма может быть либо полностью перенесена на какой – либо период, либо распределена по профилю изменения планов по периодам. Диалоговое окно настройки правил переноса оборотов открывается по кнопке «Перенос транзакций» главного меню формы документа.

Таким образом, при проведении документа возможно выполнение одного или обоих действий (определяется установкой соответствующих флагов):

  • актуализация бюджетов (выравнивание запланированных оборотов);
  • перенос транзакций.

Если запланированные данные по актуализируемому интервалу не были зафиксированы, при выполнении актуализации граница фиксации автоматически сдвигается на конец интервала актуализации.

Для проведения документа, необходимо заполнить следующие реквизиты:

  • сценарий, по которому актуализируются данные;
  • интервал актуализации, кратный периодичности планирования по сценарию;

Важно. Перед выполнением актуализации нужно убедиться, что необходимые фактические данные присутствуют в подсистеме бюджетирования (получены путем трансляции данных из других подсистем или введены вручную).

Важно. Актуализация бюджетов делает невозможным анализ отклонений план/факт по актуализированному периоду.
