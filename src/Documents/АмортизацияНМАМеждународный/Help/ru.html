Документ «Амортизация НМА (международный)»

Документ «Амортизация НМА (международный)» предназначен для ежемесячного автоматического начисления амортизации нематериальных активов в международном учете. Амортизация начисляется за месяц, определяемый по дате документа.

Данные для начисления амортизации для каждого объекта нематериальных активов (начислять ли амортизацию вообще, метод начисления амортизации, счета и аналитика для начисления амортизации и т.д.) к моменту проведения документа должны содержаться в регистре сведений «Нематериальные активы (международный учет)». Тогда по кнопке «Заполнить» в табличную часть документа будет занесен список нематериальных активов, на которые, по имеющимся данным, должна начисляться амортизация.

Изменение данных регистра сведений влияет на начисление амортизации начиная с месяца регистрации этого изменения.

Поддерживается один из трех методов начисления амортизации:

  • линейный
  • уменьшаемого остатка
  • пропорционально объему продукции

В табличной части документа для справки приводится метод начисления амортизации, предусмотренный для соответсвующего нематериального актива. В случае, если по какому-либо нематериальному активу предусмотрено использование метода «пропорционально объему продукции», в табличную часть необходимо ввести количество продукции, которая была произведена за текущий месяц с использованием данного нематериального актива (иначе амортизация не будет начислена, так как ее сумма будет считаться равной нулю). 

При проведении документа формируются бухгалтерские записи по отражению начисленной амортизации в международном учете. Также для каждого объекта нематериальных активов, по которому произведено начисление амортизации, производится запись в регистр сведений «Нематериальные активы (международный учет)» - отражаются сведения о сумме начисленной амортизации нарастающим итогом. Кроме того, если амортизация относится на счета прямых затрат на производство, то формируется также запись в регистре накопления «Незавершенное производство (международный учет)», а если амортизация относится к косвенным расходам на производство (общепроизводственным расходам) - то формируется запись в регистре накопления «Затраты (международный учет)».
