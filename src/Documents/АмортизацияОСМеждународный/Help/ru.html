Документ «Амортизация основных средств (международный)»

Документ «Амортизация основных средств (международный)» предназначен для ежемесячного автоматического начисления амортизации основных средств в международном учете. Амортизация начисляется за месяц, определяемый по дате документа.

Данные для начисления амортизации для каждого объекта основных средств (начислять ли амортизацию вообще, метод начисления амортизации, счета и аналитика для начисления амортизации и т.д.) к моменту проведения документа должны содержаться в регистре сведений «Основные средства (международный учет)». Если такие данные имеются, то по кнопке «Заполнить» в табличной части документа появится список всех объектов, подлежащих амортизации в текущем месяце.

Изменение данных регистра влияет на начисление амортизации начиная с месяца регистрации этого изменения.

Поддерживается один из трех методов начисления амортизации:

  • линейный
  • уменьшаемого остатка
  • пропорционально объему продукции.

Справочно метод начисления амортизации для каждого объекта основных средств показывается в табличной части документа. Если выбран метод «пропорционально объему продукции», то необходимо ввести в табличную часть объем выпущенной продукции за текущий месяц - иначе амортизация такого объекта не будет начислена (будет считаться равной нулю).

При проведении документа формируются бухгалтерские записи по отражению начисленной амортизации в международном учете. Также для каждого объекта основных средств, по которому произведено начисление амортизации, производится запись в регистр сведений «Основные средства (международный учет)» - отражаются сведения о сумме начисленной амортизации нарастающим итогом. Кроме того, если амортизация относится на счета прямых затрат на производство, то формируется также запись в регистре накопления «Незавершенное производство (международный учет)», а если амортизация относится к косвенным расходам на производство (общепроизводственным расходам) - то формируется запись в регистре накопления «Затраты (международный учет)».
