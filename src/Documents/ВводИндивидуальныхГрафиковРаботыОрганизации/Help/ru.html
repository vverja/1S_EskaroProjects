Документ «Ввод индивидуальных графиков работы организации»

Документ предназначен для учета нормы рабочего времени для сотрудников организации, график работы которых определяется очень поздно, либо необходимо уточниить график работы.

Документ позволяет ввести график работы для сотрудников работающих со скользящими выходными
