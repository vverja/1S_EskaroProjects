Документ "Ввод остатков свертка базы SQL"

Документ предназначен для автоматического ввода остатков обработкой "СверткаБазыSQL", по регистрам сведений, накопления, бухгалтерии.

Для использования необходимо сделать его регистратором всех регистров сведений, накопления, бухгалтерии.

Для каждого регистра создается отдельный документ, для каждого счета регистра бух.итогов также создаются отдельные документы.

При заполнении обработкой документы разбиваются на части по 10000 строк.

Может использоваться для ручной корректировки движений по регистрам.
