Документ "Корегування записів регістрів накопичення"

Документ призначений для ручного корегування записів регістрів накопичення і для введення залишків

По кнопці «Настроювання» верхньої командної панелі форми відкривається форма настроювання складу регістрів. Прапорцями відзначаються ті регістри, записи по яких планується ввести

На формі документа з'являються закладки, що відповідають обраним регістрам

У табличні поля на закладках вводяться необхідні дані
