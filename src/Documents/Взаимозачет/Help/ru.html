Документ "Взаимозачет"

ВНИМАНИЕ! В редакции 1.2 "Управления производственным предприятием для Украины" данный документ не используется. Для осуществления операций взаимозачета или переноса задолженности необходимо воспользоваться документом "Корректировка долга".

 

Документ "Взаимозачет" предназначен для погашения долга или переноса долга.

В документе указывается контрагент и договор, по которому необходимо произвести изменения.

В графу "Увеличение долга" вводится сумма, на которую необходимо увеличить долг контрагента, а в графу "Уменьшение долга" та сумма, на которую долг контрагента надо уменьшить.

В одном документе может быть введено несколько строк по различным договорам, оформленным с контрагентом. Суммы вводятся в той валюте, которая определена, как валюта взаиморасчетов по договору с контрагентом.

 

На закладке "Корреспондирующий счет по НДС" указывается счет доходов (расходов) в корреспонденции с которым будут формироваться проводки по НДС при погашении (переносе) долгов по отгрузке.

 

Для операции "Погашение долга" корректными считаются следующие наборы параметров:

  • Долг - Поставщик/Оплата, Погашение - Поставщик/Отгрузка;
  • Долг - Поставщик/Отгрузка, Погашение - Поставщик/Оплата;
  • Долг - Покупатель/Оплата, Погашение - Покупатель/Отгрузка;
  • Долг - Покупатель/Отгрузка, Погашение - Покупатель/Оплата;

    • Для операции "Перенос долга" корректными считаются следующие наборы параметров:

  • Увеличение - Поставщик/Оплата, Уменьшение - Поставщик/Оплата;
  • Увеличение - Поставщик/Отгрузка, Уменьшение - Поставщик/Отгрузка;
  • Увеличение - Покупатель/Оплата, Уменьшение - Покупатель/Оплата;
  • Увеличение - Покупатель/Отгрузка, Уменьшение - Покупатель/Отгрузка;

    •  

    ВНИМАНИЕ!!! Перед выполнением операций взаимозачета необходимо проверить наличие непогашенных долгов по контрагентам/договорам указанным в данном документе.
