Документ «Внутреннее перемещение основных средств (международный)»

Документ «Внутреннее перемещение основных средств (международный)» предназначен для завершения отражения в международном учете операций внутреннего перемещения основных средств, то есть для внесения изменений в регистр сведений об объектах основных средств в связи с операцией внутреннего перемещения.

Табличная часть документа заполняется вручную. Для каждого перемещаемого основного средства автоматически определяются его местонахождение, МОЛ, счет счет затрат по начисленной амортизации и значения его субконто на момент перемещения.

Далее пользователю предоставляется возможность редактирования данных о каждом основном средстве, по которому была произведена операция внутреннего перемещения:

  • Изменение его местонахождения (подразделения организации)
  • Изменение МОЛ
  • Изменение счета и аналитики учета затрат по начисленной амортизации


При проведении документа все изменения данных об объектах основных средств записываются в регистр сведений «Основные средства (международный учет)» и в дальнейшем оказывают влияние на начисление амортизации (начиная с месяца,  внесения изменений).
