Документ «Внутреннее перемещение средств при бюджетировании»

Документ предназначен для планирования перемещения остатков между счетами и аналитическими измерениями плана счетов бюджетирования. Такое перемещение средств отражается только формированием соответствующих проводок по плану счетов бюджетирования, на оборотах по бюджетам не отражается. Корреспондирующие счета определяются шаблоном проводок указанной в документе статьи оборотов.
