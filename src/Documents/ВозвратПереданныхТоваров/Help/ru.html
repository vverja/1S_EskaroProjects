Документ «Возврат переданных товаров»

Документ предназначен для отражения в управленческом, бухгалтерском и налоговом учете следующих операций:

  • Материалы из переработки. Возврат собственных материалов от стороннего переработчика, не использованных для выпуска продукции по договору на переработку сырья.
  • Продукции от заказчика. Возврат продукции, выпущенной из давальческого сырья,  от заказчика, например заказчика не устроило качество выпущенной продукции. Данный возврат можно оформлять только если не отражена реализация услуг по переработке.

Общие реквизиты документа

В шапке документа можно выбрать следующие реквизиты:

  • Склад, на который приходуются возвращаемые материалы или продукция. Значение реквизита выбирается из справочника «Склады». Реквизит является обязательным для заполнения.

  • Заказ. В зависимости от выбранной в документе операции:

    • Материалы из переработки -  Заказ поставщику, по которому материалы были переданы стороннему переработчику. Выбирать можно только заказы с видом операции «Переработка». Заказ не указывается, если передача материалов в переработку осуществлялась без указания заказа.

    • Продукция от заказчика - Заказ покупателя, по которому производился выпуск продукции из давальческого сырья. Выбирать можно только заказы с видом операции «Переработка». Реквизит является обязательным для заполнения.

Закладка «Материалы»

Данная закладка доступна для ввода если в документе выбрана операция «материалы из переработки».

На закладке указываются материалы, которые возвращаются от стороннего переработчика.

  • Номенклатура. Материал, возвращаемый из переработки. Значение реквизита выбирается из справочника «Номенклатура» и является обязательным для заполнения.

  • Характеристика номенклатуры. Характеристика материала, если для данного материала ведется учет по характеристикам. 

  • Серия номенклатуры. Серия материала, если для данного материала ведется учет по сериям.

  • Единица. Единица измерения материала.

  • Количество. Количество материала в указанной единице измерения.

  • Цена. Залоговая цена материала, по которой материал был передан в переработку.

  • Сумма. Залоговая стоимость материала.
  • Документ партии. Документ передачи материалов в переработку. Значение реквизита выбирается из списка документов «Передача товаров». Выбирать можно только документы с операцией «передача материалов переработчику».
  • Качество. Качество возвращаемого материала.

Для целей бухгалтерского учета материалов, отданных в переработку, указываются:

  • Счет учета. Счет бухгалтерского учета материалов на складе предприятия (например, 201).
  • Счет передачи. Счет бухгалтерского учета отданных в переработку материалов (например, 206). 
  • Статья прироста-убыли материала на складе предприятия.
  • Налоговое назначение. Указывается то же налоговое назначение, которое было указано при передаче материала в переработку.

Закладка «Продукция»

Данная закладка доступна для ввода если в документе выбрана операция «продукция от заказчика».

На закладке указывается продукция, выпущенная из давальческого сырья, которая передается заказчику.

  • Номенклатура. Продукция, возвращаемая от заказчика. Значение реквизита выбирается из справочника «Номенклатура» и является обязательным для заполнения.

  • Характеристика номенклатуры. Характеристика продукции, если для данной продукции ведется учет по характеристикам.

  • Серия номенклатуры. Серия продукции, если для данной продукции ведется учет по сериям. 

  • Единица. Единица измерения возвращаемой продукции.

  • Количество. Количество продукции в указанной единице измерение.

  • Документ партии. Документ передачи продукции заказчику. Значение реквизита выбирается из списка документов «Передача товаров». Выбирать можно только документы с операцией «передача продукции заказчику».

Закладка «Тара»

На закладке указывается возвратная тара, в которой возвращаются материалы или продукция.

  • Номенклатура. Возвращаемая тара. Значение реквизита выбирается из справочника «Номенклатура» и является обязательным для заполнения.

  • Количество. Количество возвращаемой тары в единице хранения остатков.

  • Цена. Залоговая цена тары, по которой тара была передана.

  • Сумма. Залоговая стоимость тары.

Для целей бухгалтерского учета материалов, отданных в переработку, указываются:

  • Счет учета. Счет бухгалтерского учета тары на складе. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции на основании данных регистра сведений «Номенклатура организаций».

Особенности проведения документа

Операция «Материалы из переработки»

В бухгалтерском учете формируется проводка: Дт счета учета (например, 201) - Кт счета передачи (например, 206).

Операция «Продукция от заказчика»

Данная операция не отражается в бухгалтерском учете.
