Документ "Вивантаження регламентованих звітів"

Документ "Вивантаження регламентованих звітів"  призначений для візуалізації текстів вивантаження даихі звітності у формат ДПАУ, затверджений наказом ДПАУ від 03.05.2006 № 242 і формат  Арма, розробленого ФСС на випадок тимчасової втрати працездатності, затвердженого Постановою Правління ФСС від 03.03.2006 № 7 (файли з розширенням *.xml).

Документ носить службовий характер і не призначений для самостійного використання. Документ автоматично створюється щораз при вивантаженні даних звітності, що здійснюється по кнопках "Вивантажити у формат ДПАУ (у файл *.xml)" ("Вивантажити у формат Арма (у файл *.xml)"), розташованим в діалоговій частині  форм відповідних звітів.
