Документ «Денежный чек»

Документ «Денежный чек» предназначен для отражения намерения получения наличных денежных средств с расчетного счета организации посредством денежного чека чековой книжки. При отражении документа в оперативном учете делаются четыре записи на сумму документа в следующие разделы учета:

  • «Денежные средства»: безналичные денежные средства списанные с расчетного счета, указанного в документе;
  • «Движение денежных средств»: безналичные денежные средства списанные с расчетного счета, указанного в документе;
  • «Денежные средства к получению»: наличные денежные средства в кассу, указанную в документе;
    • Денежные средства к списанию»: безналичные денежные средства с расчетного счета, указанного в документе;

      Дополнительно, в документе возможно указать перечень ведомостей на выплату заработной платы сотрудникам организации, для выдачи по которым снимаются денежные средства.

      Все реквизиты документа, за исключением табличной части «Выплата заработной платы» являются обязательными для заполнения.

      Регистрация в системе фактического списания денежных средств с расчетного счета производится при оплате документа (установлен флаг «Оплачено» и указана дата оплаты). Оплата производится либо непосредственно из формы документа, либо с помощью обработки «Выписка банка».

      На основании документа возможно ввести документ «Приходный кассовый ордер», фиксирующий поступление денежных средств в кассу.

      Если в документе «Денежный чек» заполнена табличная часть «Выплата заработной платы», при проведении приходного кассового ордера суммы по указанным ведомостям будут автоматически зарезервированы в кассе к выплате.
