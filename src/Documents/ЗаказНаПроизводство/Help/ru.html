Документ «Заказ на производство»

Заказ на производство — это заказ производственному подразделению на изготовление продукции, полуфабрикатов, оказание услуг (внутренних или контрагенту). Для оформления заказа на производство предназначен документ «Заказ на производство». Использование заказов на производство доступно в конфигурации, если в настройках параметров учета установлен флажок «Использовать заказы на производство».

Механизм заказов на производство предназначен для отражения следующих операций:

  • планирование выпуска готовой продукции, полуфабрикатов, оказания услуг и контроль исполнения планов;
  • планирование потребностей  в материалах и полуфабрикатах необходимых для выполнения заказа на производство;
  • резервирование материалов и полуфабрикатов на складах под заказ на производство;
  • учет затрат и расчет себестоимости в разрезе заказов на производство.

Под выполнением заказа на производство понимается, что  выпущена вся продукция, полуфабрикаты, оказаны услуги, указанные в заказе.

Ввод информации в заказ на производство

Заказ на производство является документом управленческого учета. Несмотря на это, в документе указывается организация. Это организация, по которой предполагается в дальнейшем оформлять выпуск продукции, полуфабрикатов, услуг, которые планируется выпустить по этому заказу.

Подразделение, на которое оформляется заказ, выбирается из перечня подразделений предприятия. В этом подразделении предполагается выпуск продукции по данному заказу на производство.

В реквизите «Запуск» указывается предполагаемая дата запуска заказа - это дата, когда заказ начнет исполняться.

В реквизите «Исполнение» устанавливается срок исполнения заказа — это дата, при наступлении которой должна быть выпущена вся продукция, полуфабрикаты, оказаны услуги, указанные в заказе. Дата исполнения заказа используется в подсистеме планирования, когда для целей планирования учитываются заказы на производство.

Реквизиты «Запуск» и «Исполнение» позволяют определить длительность выполнения заказа. Они используются в отчете «График заказов».

Если необходимо построить цепочку заказов на производство, когда один заказ на производство является обеспечением для другого заказа на производство, то в реквизите «Основной заказ на производство» следует выбрать заказ на производство, для выполнения которого он оформляется. Например, оформлен заказ на производство, в котором указана готовая продукция, в состав которой входят полуфабрикаты. На выпуск полуфабрикатов также оформляется отдельный заказ на производство. В заказах на полуфабрикаты указывается  основной заказ на производство вышестоящего уровня. В этом случае в структуре подчиненности документов можно увидеть всю цепочку заказов. При вводе одного заказа на производство на основании другого этот реквизит заполняется автоматически.

Ввод данных о продукции (услугах)

Информация о продукции, полуфабрикатах, услугах, которые должны быть выпущены по заказу на производство, заполняется на закладке «Продукция и услуги». Продукция и полуфабрикаты указываются с точностью до характеристики. Дополнительно можно указать спецификацию изготовления.

Считается, что выпуск продукции по заказу на производство осуществляется собственными силами, т.е. на продукцию, которую планируется изготовить силами стороннего переработчика, заказ на производство не оформляется.

Для каждой позиции готовой продукции (полуфабриката, услуги) можно определить параметры ее выпуска. Определение параметров выпуска выполняется в отдельном окне, которое открывается при нажатии на кнопку «Параметры». Перечень параметров можно заполнить по данным спецификации, указанной для продукции (полуфабриката, услуги).

Для выпускаемой продукции (полуфабриката) можно указать заказ в реквизите «Заказ» — заказ покупателя, внутренний заказ или заказ на производство, под который будет производиться выпуск. В этом случае при проведении документа будет выполнено размещение номенклатуры из указанного заказа в оформляемом заказе на производство.

Табличную часть «Продукция и услуги» можно заполнить автоматически. Для этого предназначена кнопка «Заполнить». Предусмотрены следующие варианты автоматического заполнения: 

  • Заполнить по заказу покупателя. Табличная часть будет заполнена по остаткам выбранного заказа покупателя (неисполненной части). Если в табличной части уже были заполненные строки, они будут удалены.
  • Добавить по заказу покупателя.  Аналогично «Заполнить по заказу покупателя», но без удаления уже заполненных строк в табличной части.
  • Заполнить по внутреннему заказу. Табличная часть будет заполнена по остаткам внутреннего заказа (неисполненной части). Если в табличной части уже были заполненные строки, они будут удалены.
  • Добавить по внутреннему заказу.  Аналогично «Заполнить по внутреннему заказу», но без удаления уже заполненных строк в табличной части.
  • Заполнить по заказу на производство. Заполнение по заказу на производство возможно в том случае, если в конфигурации используется учет потребностей заказов на производство (в настройках параметров учета установлен флаг «Вести учет потребностей заказов на производство в материалах и полуфабрикатах»). В этом случае табличная часть «Продукция и услуги» будет заполнена остатками потребности выбранного заказа на производство (номенклатуры, указанной в выбранном заказе на производство на закладке «Материалы» с видом воспроизводства «Производство»). Если в выбранном заказе на производство на закладке «Материалы» для собственных полуфабрикатов указаны различные подразделения, то перед заполнением будет предложен выбор подразделения. Если выбрано конкретное подразделение, то при заполнении будет использоваться отбор по подразделению. Если в табличной части документа уже были заполненные строки, то при заполнении ее по заказу на производство они будут удалены.
  • Добавить по заказу на производство. Аналогично «Заполнить по заказу на производство», но без удаления уже заполненных строк в табличной части.

Ввод данных о потребностях заказа на производство

На закладке «Материалы» вводится перечень полуфабрикатов и материалов, необходимых для изготовления полуфабрикатов, продукции, оказания услуг, указанных на закладке «Продукция и услуги».  Данная информация не является обязательной для заполнения.

Для того, чтобы вести учет потребностей заказа на производство в материалах и полуфабрикатах, необходимо включить данную возможность в настройках параметров учета - установить флаг «Вести учет потребностей заказов на производство в материалах и полуфабрикатах». Если учет потребностей заказов на производство в конфигурации не включен и заполнена закладка «Материалы», то данная информация будет считаться справочной и не будет участвовать в анализе удовлетворения потребности заказа.

Табличную часть на закладке «Материалы» можно заполнить автоматически по данным спецификаций продукции, указанной на закладке «Продукция и услуги». Для этого следует нажать на кнопку «Заполнить» и выбрать пункт «Заполнить по спецификации». Будет выполнена оценка потребности в полуфабрикатах и материалах согласно спецификациям изготовления продукции, полуфабрикатов, услуг. В табличную часть будут подобраны полуфабрикаты и материалы, которые встретятся на первом уровне переделов продукции (полуфабрикатов), услуг.

В табличной части «Материалы» указывается номенклатура, характеристика, количество, единица измерения. Можно указать вид воспроизводства номенклатуры:

  • Производство — для полуфабрикатов собственного производства;
  • Переработка — для полуфабрикатов, выпускаемых сторонним переработчиком;
  • Закупка — для собственных материалов;
  • Принятые в переработку — для материалов, принятых в переработку (давальческое сырье).

Информация о виде воспроизводства носит справочный характер. Также эта информация может использоваться при автоматическом заполнении табличной части «Продукция и услуги» заказа на производство, введенного на основании другого заказа на производство. Колонки «Продукция», «Характеристика продукции», «Спецификация» служат для связи потребности с выпускаемой продукцией, полуфабрикатами, услугами, указанными на закладке «Продукция и услуги». Например, если на закладке «Продукция и услуги» содержится несколько строк с разной номенклатурой или с номенклатурой, которая отличается характеристикой и/или спецификацией, то для каждого материала или полуфабриката на закладке «Материалы» можно указать, для какой продукции, характеристики, спецификации он является потребностью.

Для полуфабрикатов собственного производства можно указать производственное подразделение, в котором будет выпущен полуфабрикат. Информация о выпускающем подразделении используется в механизмах автоматического заполнения табличной части «Продукция и услуги» по заказу на производство, а также при вводе одного заказа на производство на основании другого заказа на производство.

Полуфабрикаты и материалы для выполнения заказа на производство можно зарезервировать из свободного остатка на складе или разместить в заказе поставщику, внутреннем заказе или другом заказе на производство. Для этого в табличной части «Материалы» предназначен реквизит «Размещение».

При оперативном проведении документа можно использовать механизм автоматического резервирования и размещения. Для этого необходимо нажать на кнопку «Заполнить и провести» в командной панели документа. Будет открыта форма настройки параметров авторезервирования. В форме следует установить соответствующие флажки «резервирование» и/или «размещение», выбрать стратегию авторезервирования по заказам и нажать на кнопку «Заполнить и провести». При этом значения в колонку «Размещение» будут введены автоматически (если в форме настройки параметров авторезервирования установлен флажок «Очищать размещение перед заполнением», то предварительно реквизит «Размещение» будет очищен), документ будет проведен. Подробней описание этого механизма см. ниже.

Резервирование полуфабрикатов и материалов под заказ на производство, размещение потребностей заказа на производство в заказе поставщику, внутреннем заказе, другом заказе на производство может быть выполнено также при помощи других документов, например, «Резервирование товаров».

В некоторых случаях может быть удобно начать заполнение табличных частей документа с закладки «Материалы». Если заполнять закладку «Материалы» c помощью подбора по спецификации, то табличная часть «Продукция и услуги» будет заполнена автоматически.

Описание работы кнопки «Заполнить и провести»

При нажатии на кнопку «Заполнить и провести» происходят следующие действия:

1. Выбор параметров авторезервирования и авторазмещения.

2. Исходя из выбранных параметров, происходит заполнение колонки «Размещение» в табличной части «Материалы». При этом учитывается значение параметра «Стратегия авторезервирования по заказам».

    2.1. Авторезервирование - заполнение колонки «Размещение» складом для целей резервирования (происходит в случае, если в параметрах был установлен соответствующий признак).

            2.1.1. Определяется список доступных для резервирования складов: группа доступности складов

  • Если для пользователя в настройках по умолчанию указана группа доступности складов - используется группа из настроек по умолчанию
  • Если группа доступности складов не указана в настройках пользователя - резервирование происходит на всех складах

           2.1.2. Определяются остатки на складах, доступные для резервирования. Расчет происходит на основании остатков в регистре «Товары на складах» за минусом

  • уже выполненных резервов (регистр «Товары в резерве на складах») 
  • остатков, предназначенных для передачи по ордерной схеме (регистр «Товары к передаче со складов»)

     2.2. Авторазмещение - заполнение колонки «Размещение» заказом поставщику для целей размещения (происходит в случае, если в параметрах был установлен соответствующий признак).

            2.2.1. Определяются остатки в заказах поставщикам, доступные для размещения. Расчет происходит на основании остатков в регистре «Заказы поставщикам», при этом список заказов ограничивается следующими условиями:

  • Склад в заказе ограничивается списком доступных складов, которые определяются на основании сведений п.2.1.1
  • Дата поступления в заказе поставщику должна быть меньше или равна дате исполнения заказа на производство. Если дата исполнения не заполнена, используется дата документа
  • Организация в заказе поставщику должна соответствовать организации в заказе на производство

Из остатка по регистру «Заказы поставщикам» исключаются уже выполненные в данных заказах поставщикам размещения

    2.3. Из полученных свободных остатков для резервирования и размещения вычитается количество номенклатуры, для которой размещение в документе указано вручную (учитываются строки, в которых указано размещение).

     2.4. В строках документа, для которых не заполнено размещение, происходит заполнение колонки «Размещение». При необходимости одна строка может быть разбита на несколько строк с разным значением размещения. Количество номенклатуры, которое не удалось разместить, выделяется в отдельную строку с пустым значением размещения.
