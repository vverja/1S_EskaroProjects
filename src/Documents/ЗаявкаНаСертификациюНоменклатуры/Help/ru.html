Документ «Заявка на сертификацию номенклатуры»

Документ «Заявка на сертификацию номенклатуры» служит для отражения факта заявления на сертификацию материалов, товаров, продукции во внешние сертификационные органы или для внутренней сертификации.

Вид операции, которая будет произведена в результате проведения документа, определяется непосредственно в самой форме документа с помощью кнопки «Операция».

В документе заполняется информация о номенклатуре, серии номенклатуры, нормативном документе. Поле «нормативный документ» заполняется системой автоматически, на основании данных регистра сведений «Нормативные документы сертификации», но может быть выбрано вручную, из справочника «Нормативные документы сертификации».

Заявка на внутреннюю сертификацию оформляется документом, в котором установлен вид операции «внутренняя».

Для заявки на внешнюю сертификацию доступно поле «Орган по сертификации». Для заявки на внешнюю сертификацию доступна печатная форма «Заявка на сертификацию». Заявка на внутреннюю сертификацию оформляется документом, в котором установлен вид операции «внутренняя».

Для заявки на внутреннюю сертификацию могут быть заполнены поля «Склад», «Сертифицирующее подразделение».  По умолчанию значения полей заполняется значением соответствующей настройки для пользователя. Для заявки на внутреннюю сертификацию доступны печатные формы «Извещение на продукцию», «Извещение на материалы».

Документ «Заявка на сертификацию» может быть введен на основании документов «Поступление товаров и услуг» и «Отчет производства за смену». В обоих случаях ввод на основании производится посредством выбора из автоматически сформированного списка пары «серия-номенклатура». В список попадают только те серии, для которых необходимо провести сертификацию, но сертификация не оформлена, а также не оформлена «Заявка на сертификацию». Необходимость проведения сертификации определяется по реквизитам «Требуется внешняя сертификация», «Требуется внутренняя сертификация» справочника «Номенклатура». Если по серии необходимо провести и внутреннюю, и внешнюю сертификацию, серия попадает в список два раза, с пометкой «внутр.» и «внеш.». Для такой серии необходимо создать две заявки на сертификацию (внешнюю и внутреннюю).
