Документ «Изменение лимитов отпуска материалов»

В документе устанавливаются лимиты отпуска материалов со склада в подразделение на определенный период. Лимит – это максимальная величина расхода материала в определенный период, например за неделю или за месяц. Документ можно проводить по управленческому и по бухгалтерскому учету.

Реквизиты шапки документа:

  • Организация. Организация, по которой планируется в дальнейшем отражать расход материалов со склада в производство в регламентированном учете.
  • Подразделение. Подразделение, для которого по управленческому учету назначается лимит отпуска материалов, указанных в документе.
  • Подразделение организации. Подразделение, для которого по бухгалтерскому учету назначается лимит отпуска материалов, указанных в документе. На это подразделение в дальнейшем будут относиться затраты по материалам, списанным со склада.
  • Склад. Склад, для которого устанавливаются лимиты отпуска материалов.
  • Ответственный. Сотрудник, ответственный за ввод информации в документ.

 

Лимиты

 

  • Номенклатура. Характеристика номенклатуры. Материалы, для которых устанавливаются лимиты, указываются с точностью до характеристики.
  • Периодичность. Лимит – это максимальная величина расхода материала в определенный период, например за неделю или за месяц. Такими периодами могут быть день, неделя, декада, месяц, квартал, полугодие, год. В одном документе для материала одновременно можно установить лимиты расхода для разных периодов. Например, для использования в производстве мебельного лака могут быть установлены лимиты: не более 10 литров каждый день, но в целом за месяц не более 280 литров. Контроль лимитов осуществляется отдельно по каждому из периодов.
  • Лимит отпуска. Максимальное количество материалов в выбранных единицах измерения, которое может быть передано в подразделение со склада за указанный период.
  • Контролировать лимит. Флаг для явного использования (или не использования) информации об установленных лимитах. Если этот признак установлен, то контроль лимита будет выполняться в момент списания материала со склада на затраты, если отключен, то контролироваться расход материала не будет.

 

Перечень материалов и значения лимитов их отпуска можно рассчитать автоматически. Для этого в документе следует нажать кнопку «Заполнить». Откроется форма выбора стратегии расчета лимитов.
