Документ "Изменение налогового назначения ОС"

Документ "Изменение налогового назначения ОС" используется для указания нового налогового назначения, в  котором будет задействовано основное средство.

 

Обрабатываемые объекты выбираются из справочника "Основные средства".

Для каждого объекта основных средств указывается текущая балансовая стоимость в налоговом  учете и сумма модернизации текущего периода (текущего года). Для операций перевода в непроизводственные фонды в документ добавлены реквизиты НДС:

  • Учитывать НДС.  Если установлен данный флаг, то становятся доступными и видимыми остальные реквизиты НДС.
  • Авторасчет НДС. Данный флаг делает недоступным для редактирования поле Сумма НДС. При записи документа пересчитывается НДС по всему документу с учетом погрешностей по строкам.
  • Сумма включает НДС. Признак того, как нужно рассчитывать сумму НДС.

При установленном флажке Учитывать НДС  в табличной части  необходимо заполнить реквизиты:

  • Сумма. Сумма для обложения НДС
  • Ставка НДС.
  • Сумма НДС.
  • Счет учета НДС. Счет налоговых обязательств (по умолчанию 6435 "Условная продажа"). Этот реквизит видим, только если установлен флажок "Отражать в бухгалтерском учете".

По своей специфике документ по умолчанию отражаться в налоговом учете. Если установлен флажок "Отражать в бухгалтерском учете", то при начислении НДС будут сделаны корректировочные проводки по НДС в бухгалтерском учете. Если нет НДС, то "Отражать в бухгалтерском учете" ни на что не влияет.

Для более быстрого заполнения табличной части документа  необходимо воспользоваться кнопкой "Подбор".

После формирования списка основных средств, подготавливаемых к передаче необходимо воспользоваться кнопкой "Заполнить" для заполнения табличной части  информацией об объектах основных средств по данным учета (Журнала проводок "Налоговый учет"), сумма стоимости в налоговом учете ОС, учитываемых не пообъектно, рассчитывается пропорционально первоначальной стоимости ОС группы учета.

При проведении документа по каждой строке будут сформированы проводки по налоговому учету.
