Документ "Зміна податкового призначення ОЗ"

Документ "Зміна податкового призначення ОЗ" використовується для встановлення нового податкового призначення, в  якому будуть  використовуватись основні засоби

Потрібніі об'єкти вибираються з довідника "Основні засоби".

Для кожного об'єкта основних засобів вказується поточна балансова вартість в податковому  обліку і сума модернізації поточного періоду (поточного року). Для операцій переведення засобів в невиробничі фонди в документ додані реквізити ПДВ:

  • Враховувати ПДВ.  Якщо встановлено даний прапорець, то стають доступні і видимими інші реквізити ПДВ.
  • Авторозрахунок ПДВ. Даний прапорець робить недоступним для редагування поле Сума ПДВ. При записі документа перераховується ПДВ по всьому  документу з урахуванням погрішностей по рядках
  • Сума включає ПДВ. Ознака того, як потрібно розраховувати суму ПДВ.

При встановленому прапорці Враховувати ПДВ  у табличній частині  необхідно заповнити реквізити:

  • Сума. Сума для обкладання ПДВ
  • Ставка ПДВ.
  • Сума ПДВ.
  • Рахунок ПДВ. Рахунок податкових зобов'язань (за замовчуванням 6435 "Умовний продаж"). Цей реквізит бачимо, тільки якщо встановлено прапорець "Відображати в бухгалтерському обліку".

По своїй специфіці документ за замовчуванням відображається в податковому обліку. Якщо встановлено прапорець "Відображати в бухгалтерському обліку", то при нарахуванні ПДВ будуть зроблені коректувальні проводки по ПДВ в бухгалтерському обліку. Якщо немає ПДВ, то прапорець "Відображати в бухгалтерському обліку" ні на що не впливає

Для більш швидкого заповнення табличної частини документа  необхідно скористатися кнопкою "Підбір".

Після формування списку основних засобів, що підготовлені до передачі необхідно скористатися кнопкою "Заповнити" для заповнення табличної частини  інформацією про об'єкти основних засобів за даними обліку (Журналу проводок "Податковий облік").Сума вартості в податковому обліку ОЗ, що враховуються не пооб'єктно, розраховується пропорційно первісної вартості ОЗ групи обліку

При проведенні документа по кожному рядку будуть сформовані проводки з податкового обліку
