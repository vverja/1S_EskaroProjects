Документ «Изменение параметров эксплуатации нематериальных активов(международный)»

Документ «Изменение параметров эксплуатации НМА (международный)» предназначен для отражения изменений в регистре сведений об объектах нематериальных активов для целей международного учета.

В документ вносятся новые сведения об объектах нематериальных активов (например, новый срок полезного использования в случае его изменения, новый метод начисления амортизации или счет затрат, на который будет относиться амортизация и т.д.).

При проведении документа эти сведения (наряду со «старыми» значениями, которые сохранены в документе) записываются в регистр сведений «Нематериальные активы (международный учет)».

При этом не предполагается, что такое изменение каким-либо образом связано с хозяйственной операцией с объектами, то есть не приводит к необходимости делать какие либо записи (проводки) непосредственно в международном учете.

Для отражения хозяйственных операций с нематериальными активами предназначены другие документы (например, «Выбытие нематериальных активов (международный)», и т.д.).
