Документ «Изменение параметров эксплуатации основных средств (международный)»

Документ «Изменение параметров эксплуатации основных средств (международный)» предназначен для отражения изменений в регистре сведений об объектах основных средств для целей международного учета в тех случаях, когда не требуется отражать какие-либо движения основных средств на счетах.

В документ вносятся новые сведения об объектах основных средств (например, новый срок полезного использования в случае его изменения, новый метод начисления амортизации или счет затрат, на который будет относиться амортизация и т.д.).

При проведении документа эти сведения (наряду со «старыми» значениями, которые сохранены в документе) записываются в регистр сведений «Основные средства (международный учет)».

При этом не предполагается, что такое изменение каким-либо образом связано с хозяйственной операцией с объектами, то есть не приводит к необходимости делать какие либо записи (проводки) непосредственно в международном учете.

Для отражения хозяйственных операций с основными средствами предназначены другие документы (например, «Выбытие основных средств (международный)», «Внутреннее перемещение основных средств (международный)» и т.д.).
