Документ "Изменение способа отражения расходов по амортизации НМА"

Документ позволяет изменить способ отражения расходов по начисленной амортизации по объектам НМА .

Данный документ не формирует никаких проводок, он лишь изменяет значение соответствующих ресурсов регистра сведений, хранящего информацию о нематериальных активах  ("Способы отражения расходов по амортизации НМА"  и "Способы отражения расходов по амортизации НМА (бухгалтерский учет)" ).

Влияет на распределение начисленной амортизации по бухгалтерскому учету.
