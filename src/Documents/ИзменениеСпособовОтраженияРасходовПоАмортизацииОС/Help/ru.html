Документ "Изменение способов отражения расходов по амортизации"

Документ позволяет изменить способ отражения расходов по амортизации по объектам основных средств.

Данный документ не формирует никаких проводок, он лишь изменяет значение соответствующих ресурсов регистров сведений, хранящих информацию об основных средствах ("Способы отражения расходов по амортизации" - для управленческого учета, "Способы отражения расходов по амортизации (бухгалтерский учет)").

Влияет на распределение начисленной амортизации по управленческому и бухгалтерскому учету.
