Документ «Инвентаризация брака в производстве»

Документ предназначен для ввода фактических остатков расходов по браку в производстве на конец периода в суммовом выражении. При проведении расчета себестоимости выпуска расходы по браку в производстве списываются на производственные расходы за исключением указанных в документе остатков.

Остатки вводятся по всем разрезам учета брака в производстве, а именно: подразделению, статье затрат, номенклатурной группе, заказу, продукции, характеристике продукции и серии.

 

Реквизиты шапки документа:

  • Подразделение - подразделение в управленческом учете, по которому вводятся результаты инвентаризации брака в производстве.
  • Подразделение организации - подразделение организации (юридического лица), по которому вводятся результаты инвентаризации брака в производстве.

 

Затраты по браку в производстве

  • Статья затрат. Статья затрат по учету расходов, связанных с исправлением брака. Характер статьи затрат должен быть «Брак в производстве».
  • Номенклатурная группа. Номенклатурная группа продукции, на которую отнесены расходы по браку.
  • Заказ. Заказ на производство или заказ покупателя, к которому привязаны расходы по браку.
  • Продукция, характеристика, серия  - данные о продукции, при изготовлении которой был выявлен брак и на которую относятся затраты по исправлению брака.
  • Сумма остатка. Сумма затрат по браку по управленческому учету.
  • Сумма (Регл). Сумма затрат по регламентированному учету.
  • Счет затрат (БУ). счет, по которому были учтены затраты в бухгалтерском учете.

Табличную часть можно заполнить автоматически с помощью кнопки «Заполнить» - «Заполнить по остаткам». Табличная часть заполняется затратами на брак, которые числятся в НЗП на дату документа.

 

Особенности документа

 

Документ предназначен только для фиксации результатов инвентаризации. Поэтому документ не формирует проводки и не выполняет движений по регистрам.
