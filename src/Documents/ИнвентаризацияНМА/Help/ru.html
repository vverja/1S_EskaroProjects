Документ "Инвентаризация НМА"

Документ "Инвентаризация НМА" предназначен для печати ведомости НМА,  находящихся на ответственном хранении материально-ответственного лица подразделения . Документ может быть проведен по:

  • управленческому учету;
  • бухгалтерскому учету.

При проведении документ не формирует движений, но проверят правильность заполнения. Проведенный документ символически означает завершение инвентаризации.

Реквизиты группы "Данные для управленческого учета"

  • Подразделение. Подразделение выбирается из справочника "Подразделения". Это подразделение, в котором  числится нематериальный актив. Если нематериальный актив может использоваться в разных подразделениях и амортизация по нему может начисляться на затраты этих подразделений. 
  • МОЛ. Материально-ответственное лицо выбирается из списка работников предприятия. При этом открывается форма для выбора работников, которые на момент оформления документа приняты на работу. Если подразделение заполнено, то в форме выбора работника показывается список работников только выбранного подразделения.

Реквизиты группы "Данные для бухгалтерского учета"

  • Подразделение. Подразделение выбирается из справочника "Подразделения организаций". Это подразделение, в котором  числится нематериальный актив по бухгалтерскому учету. При начале выбора подразделения открывается список подразделений только той организации, которая указана в документе.
  • МОЛ. Материально-ответственное лицо выбирается из списка работников выбранной организации. При этом открывается форма для выбора работников, которые на момент оформления документа приняты на работу в организацию. Если подразделение заполнено, то в форме выбора работника показывается список работников только выбранного подразделения организации.

Сервис по заполнению табличной части.

В документе предусмотрен сервис по заполнению табличной части остатками НМА на дату и время документа.

Документ имеет две печатные формы:

  • "Управленческий учет" в которую выбираются НМА, если хотя бы один реквизит из "Количество (УУ), Количество факт. (УУ), Примечание (УУ)", является не пустым.
  • "Бухгалтерский учет" в которую выбираются НМА, если хотя бы один реквизит из "Количество (БУ), Количество факт. (БУ), Примечание (БУ)", является не пустым.
