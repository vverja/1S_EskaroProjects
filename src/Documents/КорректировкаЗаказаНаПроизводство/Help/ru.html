Документ «Корректировка заказа на производство»

Вид операции - «Корректировка заказа на производство»

С помощью этого документа можно провести корректировку заказа на производство, связанную с изменением состава и/или объема заказанной продукции, потребностей заказа на производство в материалах и полуфабрикатах.

 

Заказ на производство, который требуется скорректировать, указывается в шапке документа в реквизите «Заказ».

 

Заполнение табличных частей документа

Документ имеет табличные части «Продукция и услуги» и «Материалы». Состав и назначение табличных частей документа аналогичны табличным частям документа «Заказ на производство».

С помощью документа можно корректировать:

  •  данные о продукции, полуфабрикатах и услугах, которые требуется выпустить по заказу на производство (на закладке «Продукция и услуги»);
  •  данные о потребностях заказа на производство в материалах и полуфабрикатах (на закладке «Материалы»).  

Можно корректировать данные о продукции и данные о потребностях заказа в одном документе. Также можно корректировать только данные о продукции или данные о потребностях - в этом случае заполняется только одна табличная часть документа.

Если в конфигурации не используется учет потребностей заказов на производство (в настройках параметров учета не установлен флажок «Вести учет потребностей заказов на производство в материалах и полуфабрикатах») и заполнена закладка «Материалы», то данная информация будет считаться справочной и не будет участвовать в анализе удовлетворения потребности заказа. 

 

Для позиций номенклатуры, которые необходимо добавить в заказ на производство, количество указывается со знаком «+», для позиций, которые необходимо исключить из заказа на производство - со знаком «-».

 

Табличную часть «Продукция и услуги» можно заполнить автоматически по заказу на производство. Табличную часть «Материалы» также можно заполнить автоматически: по спецификации или по остаткам потребности заказа на производство.

 

Вид операции - «Закрытие потребностей заказа на производство»

С помощью документа можно провести закрытие потребностей заказа на производство. Операция доступна, если в настройках параметров учета установлен способ закрытия потребностей «Явно». Заказ на производство, по которому требуется провести закрытие потребностей, указывается в шапке документа в реквизите «Заказ». Табличную часть «Материалы» можно заполнить автоматически: по остаткам потребности заказа на производство. При проведении документа будут сформированы движения по регистру «Потребности заказов на производство».

 

При оперативном проведении документа для заполнения колонки «Размещение» на закладке «Материалы» можно использовать механизм автоматического резервирования и размещения. Для этого необходимо нажать на кнопку «Заполнить и провести» в командной панели документа. Для строк с отрицательным количеством колонка «Размещение» заполняется уже выполненным размещением, для строк с положительным количеством колонка «Размещение» заполняется возможным размещением на складах и в других заказах.

Описание работы кнопки «Заполнить и провести»

При нажатии на кнопку «Заполнить и провести» происходят следующие действия:

1. Выбор параметров авторезервирования и авторазмещения.

2. Исходя из выбранных параметров, происходит заполнение колонки «Размещение» в табличной части «Материалы» для строк с отрицательным количеством. При этом учитывается значение параметра «Стратегия авторезервирования по заказам».

    2.1. Авторезервирование - заполнение колонки «Размещение» складом  (происходит в случае, если в параметрах был установлен соответствующий признак). Значение склада определяется исходя из существующих резервов по данному заказу на производство.

    2.2. Авторазмещение - заполнение колонки «Размещение» заказом поставщику, заказом на производство или внутренним заказом (происходит в случае, если в параметрах был установлен соответствующий признак). Значение заказа определяется исходя из существующих размещений, выполненных для данного заказа на производство

3. Исходя из выбранных параметров, происходит заполнение колонки «Размещение» в табличной части «Материалы» для строк с положительным количеством. При этом учитывается значение параметра «Стратегия авторезервирования по заказам».

    3.1. Авторезервирование - заполнение колонки «Размещение» складом для целей резервирования (происходит в случае, если в параметрах был установлен соответствующий признак).

            3.1.1. Определяется список доступных для резервирования складов: группа доступности складов

  • Если для пользователя в настройках по умолчанию указана группа доступности складов - используется группа из настроек по умолчанию
  • Если группа доступности складов не указана в настройках пользователя - резервирование происходит на всех складах

           3.1.2. Определяются остатки на складах, доступные для резервирования. Расчет происходит на основании остатков в регистре «Товары на складах» за минусом

  • уже выполненных резервов (регистр «Товары в резерве на складах») 
  • остатков, предназначенных для передачи по ордерной схеме (регистр «Товары к передаче со складов»)

     3.2. Авторазмещение - заполнение колонки «Размещение» заказом поставщику для целей размещения (происходит в случае, если в параметрах был установлен соответствующий признак).

            3.2.1. Определяются остатки в заказах поставщикам, доступные для размещения. Расчет происходит на основании остатков в регистре «Заказы поставщикам», при этом список заказов ограничивается следующими условиями:

  • Склад в заказе ограничивается списком доступных складов, которые определяются на основании сведений п.3.1.1
  • Дата поступления в заказе поставщику должна быть меньше или равна дате исполнения заказа на производство. Если дата исполнения не заполнена, используется дата документа
  • Организация в заказе поставщику должна соответствовать организации в заказе на производство

Из остатка по регистру «Заказы поставщикам» исключаются уже выполненные в данных заказах поставщикам размещения

    3.3. Из полученных свободных остатков для резервирования и размещения вычитается количество номенклатуры, для которой размещение в документе указано вручную (учитываются строки, в которых указано размещение).

     3.4. В строках документа, для которых не заполнено размещение, происходит заполнение колонки «Размещение». При необходимости одна строка может быть разбита на несколько строк с разным значением размещения. Количество номенклатуры, которое не удалось разместить, выделяется в отдельную строку с пустым значением размещения.
