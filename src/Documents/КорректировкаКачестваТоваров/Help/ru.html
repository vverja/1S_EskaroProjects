Документ «Корректировка качества товаров»

Документ предназначен для изменения статуса качества позиций номенклатуры. При поступлении товаров для них из справочника «Качество номенклатуры»  заполняется по умолчанию предопределенный вид качества - «Новый». При необходимости это название может быть изменено пользователем.

В том случае, если при поступлении товаров или при инвентаризации товаров на складе был обнаружен товар другого качества, например, бракованный товар, то вводится документ «Корректировка качества товаров».

В шапке документа указывается склад, где хранятся позиции номенклатуры, для которых необходимо изменить статус качества.

В табличной части документа указываются старые и новые значения качества товаров, а также количество позиций. При проведении документа будет списано указанное в документе количество позиций со старыми значениями качества и оприходовано указанное количество позиций номенклатуры с новым качеством товаров. 

Документ может быть оформлен только на оптовый или розничный склад, оформление данного документа для неавтоматизированной торговой точки не предусмотрено.

Выполнить корректировку качества можно только для товаров, которые находятся в свободном остатке.

Корректировать качество товара, поступившего по приходному ордеру без права продажи, нельзя.

Чтобы скорректировать качество товаров, зарезервированных под заказ, следует выполнить последовательность действий:

  • снять товар с резерва под заказ документом «Резервирование товаров»;
  • выполнить корректировку качества товара документом «Корректировка качества товаров».

Важно! Корректировка качества оборудования в конфигурации не поддерживается, поэтому в документе не следует указывать позиции номенклатуры, которые были оприходованы как оборудование.

Важно! В табличной части документа не должно содержаться строк, в которых значения реквизитов «Качество» и «Новое качество» совпадают. Проверка этого условия выполняется при проведении документа. При наличии таких строк документ не будет проведен.
