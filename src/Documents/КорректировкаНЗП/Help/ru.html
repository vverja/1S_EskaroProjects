Документ «Корректировка незавершенного производства»

Документом отражается операция корректировки аналитики материальных производственных затрат: изменение подразделения, номенклатурной группы, заказа, статьи , счета затрат, налогового назначенния.

 

В шапке документа можно выбрать следующие реквизиты:

  • Подразделение. Производственное подразделение, по которому производится корректировка незавершенного производства. Значение реквизита выбирается из справочника «Подразделения». Реквизит является обязательным для заполнения.
  • Подразделение организации. Производственное подразделение организации,  по которому производится корректировка незавершенного производства. Значение реквизита выбирается из справочника «Подразделения организаций». Реквизит является обязательным для заполнения.
  • Тип стоимости. Тип указания стоимости корректировки незавершенного производства. Можно выбрать следующие значения:
    • Рассчитывается. Стоимость корректировки незавершенного производства рассчитывается в конце месяца при расчете себестоимости выпуска.
    • Фиксированная. Стоимость корректировки незавершенного производства задается в документе и не корректируется при расчете себестоимости выпуска.

 

Материалы

 

В табличной части указывается аналитика производственных затрат на момент корректировки:

  • Номенклатура. Материал, корректировка аналитики которого в НЗП производится.
  • Характеристика номенклатуры. Характеристика материала, если для  данного материала ведется учет по характеристикам.
  • Серия номенклатуры. Серия материала, если для  данного материала ведется учет по сериям. 
  • Единица. Единица измерения материала.
  • Количество. Количество материала в указанной единице измерения.
  • Заказ, Номенклатурная группа, Статья затрат, Счет затрат. Аналитика производственных затрат на момент корректировки.
  • Цена. Средняя цена единицы материала в НЗП в валюте управленческого учета. Указывается, если выбран тип стоимости «Фиксированная».
  • Сумма. Стоимость материала в НЗП в валюте управленческого учета. Реквизит указывается, если выбран тип стоимости «Фиксированная».
  • Сумма (регл). Стоимость материала в НЗП в валюте регламентированного учета. Реквизит указывается, если выбран тип стоимости «Фиксированная».
  • Сумма НДС. Сумма налогового кредита, начисленного при приобретении материала, списанного в НЗП.
  • Налоговое назначение, указанное при списании материала в НЗП
  • Ст. прироста-убыли запасов. Исходная статья прироста-убыли запасов налогового учета незавершенного производства.

 

Предусмотрено заполнение данной табличной части по кнопке «Заполнить». Возможны следующие варианты заполнения:

  •  Заполнить по остаткам. Заполнение производится по остаткам незавершенного производства.
  • Добавить из требования-накладной. В табличную часть добавляются данные из документов «Требование-накладная», которые выбирает пользователь в открывающемся диалоговом окне.

 

Получатели

 

На закладке указываются новые значения аналитики незавершенного производства. Каждая строка закладки «Получатели» является расширением соответствующей строки закладки «Материалы». Ввод новых строк, удаление, сортировка на данной закладке не доступны. Новые значения аналитики незавершенного производства можно предварительно заполнить исходными значениями, нажав кнопку «Заполнить»- «Заполнить по данным о материалах». Тогда новые значения подразделения, номенклатурной группы, статьи затрат, заказа, счета затрат будут заполнены соответствующими данными с закладки Материалы, либо данными из шапки документа. Далее значения аналитики незавершенного производства можно откорректировать вручную.
