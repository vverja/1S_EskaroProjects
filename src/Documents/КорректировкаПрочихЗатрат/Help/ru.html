Документ «Корректировка прочих затрат»

Документом отражается операция корректировки аналитки нематериальных затрат любого характера: подразделения, номенклатурной группы, заказа, статьи затрат, счета учета, налогового назначения.

Документ "Корректировка прочих затрат", в частности, рекомендуется использовать в тех случаях, когда порядок распределения косвенных затрат не может быть воспроизведен документами конфигурации или воспроизведен неверно. В таких случаях проводится ручной расчет и результаты отражается в учете с помощью этого документа.

 

В шапке документа можно выбрать следующие реквизиты:

  • Подразделение. Подразделение, по которому производится корректировка нематериальных затрат. Значение реквизита выбирается из справочника «Подразделения». Реквизит является обязательным для заполнения.
  • Подразделение организации. Подразделение организации, по которому производится корректировка нематериальных затрат. Значение реквизита выбирается из справочника «Подразделения организаций». Реквизит является обязательным для заполнения.

 

Затраты

 

На закладке указываются сумма и аналитика нематериальных затрат на момент корректировки:

  • Заказ, Статья затрат, Счет затрат. Аналитика затрат на момент корректировки.
  • Аналитика. Дополнительная аналитика учета затрат на момент корректировки, состав которой зависит от характера затрат: номенклатурная группа, объект строительства и пр. Вид аналитики справочно показывается в реквизите «Вид аналитики».
  • Сумма. Сумма затрат в валюте управленческого учета.
  • Сумма (регл). Сумма затрат в валюте регламентированного учета.
  • Налоговое назначение. Исходное налоговое назначение затрат. 

Предусмотрено заполнение данной табличной части по кнопке «Заполнить». При нажатии кнопки открывается список отбора характера затрат. После выбора в табличную часть документа будут подобраны остатки затрат по выбранному характеру затрат. При необходимости в списке выбора можно указать вариант выбора всех нематериальных статей затрат, по которым будут анализироваться остатки затарт.

 

Получатели

 

На закладке указываются новые значения аналитики затрат. Каждая строка закладки «Получатели» является расширением соответствующей строки закладки «Затраты». Ввод новых строк, удаление, сортировка на данной закладке не доступны. Новые значения аналитики затрат можно предварительно заполнить исходными значениями, нажав кнопку «Заполнить» - «Заполнить по данным о затратах». Тогда новые значения подразделения, заказа, аналитики, счета затрат, налогового назначения будут заполнены соответствующими данными с закладки «Затраты», либо данными из шапки документа. Далее значения аналитики затрат можно откорректировать вручную. 

Способ заполнения  «Заполнить» - «Заполнить по данным о затратах и распределить между налоговыми назначениями» используется, если документ вводится, чтобы распределить нематериальные затраты, имеющие определенное налоговое назначение, на все налоговые назначения, по которым осуществлялось производство. После нажатия кнопки нужно выбрать способ, которым затраты будут распределены между налоговыми назначениями.

Результатом проведения документа является изменение аналитики затрат с точностью до номенклатурной группы, заказа, налогового наначения, но не до вида продукции. Для распределения с точностью до вида продукции вводится документ «Распределение прочих затрат».

Для каждой строки документа может быть заполнен флаг «Не корректировать НУ». Для строк, для которых флаг заполнен, не формируются предварительные корректировки по регистрам «Корректировки НЗП НУ», «Корректировки Затраты НУ» и пр., т.е. изменение налогового назначения затрат не сопровождается начислением валовых расходов (дохода) и налогового кредита (обязательств). Корректировка налогового назначения затрат формируется независимо от заполнения флага  «Не корректировать НУ».
