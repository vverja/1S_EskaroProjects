Форма «Разделение документа «Корректировка стоимости списания товаров» по видам учета»

Форма используется в контексте обновления на версию 1.2.8 и более поздние

Форма предназначена для разделения документов «Корректировка стоимости списания товаров» созданных в предыдущих версиях на несколько в соответствии со следующими ограничениями:

1. Документ «Корректировка стоимости списания товаров» не может быть проведен более чем по одному виду учета

2. Если документ «Корректировка стоимости списания товаров» проводиться по регламентированному учету, то в нем должна быть обязательно заполнена организация

Форма создает на каждый вид учета и организацию (для регл. учета) отдельный документ вместо старого документа проведенного по нескольким видам учета. Заполнять список обрабатываемых документов можно вручную, а можно с помощью автоматического заполнения по кнопке «Заполнить». При этом подобраны будут только документы, которые требуют разделения, и находящиеся в указанном интервале.
