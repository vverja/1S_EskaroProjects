Документ «Оплата по среднему заработку»

Документ «Оплата по среднему заработку» регистрирует замену плановых начислений работника на определенный период начислением оплаты по среднему заработку с «обычным» вариантом расчета.

Данный вариант начислений используется для оплаты командировок, а также в других случаях, предусмотренных законодательством. Документ также используется и для расчета средней для расчета "Доплата до средней", но в отличии от других начислений окончательный расчет производится не текущим документом, а в документе "Начисление зарплаты работникам организаций"

Информация, необходимая для расчета начислений, вносится в шапку документа. Результаты автоматического расчета, который выполняется нажатием кнопки «Рассчитать все», отображаются в табличной части документа.

Некоторые особенности расчета среднего заработка для оплаты по среднему:

На странице расчет среднего заработка рассчитывается сумма средней. При необходимости можно заполнить эту сумму в ручную и перезаполнить и пересчитать данные на странице "Начисления"


В расчете средней используются дополнительные параметры:
- Период средней - месяцы, которые будут анализироваться при заполнении табличной части "Расчет средней"
- Вид учета времени для средней – определяет каким образом будет рассчитываться средняя по рабочим дням, по рабочим часам или по календарным дням (для отпуска используется по умолчанию расчет по календарным дням)
- Число месяцев - число месяцев, которые анализируются в табличной части "Расчет средней"
- Расчет премии за год - тут указывается год за который начислена годовая премия, участвующая в расчете средней. Сумму модно в случае необходимости проставить вручную. Сумма годовой премии попадет в табличную часть "Расчет средней" в соответствующие строки в размере 1/12

Особенности заполнения и расчета таблицы "Расчет средней"
Суммы заполняются помесячно. Каждый месяц разделен на 4 части
- "По годовой премии" - 1/12 суммы указанной как годовая премия в дополнительных параметрах для расчета средней
- "По премиям (учитывают полностью)" - сумма начисленных премий  деленные на количество месяцев, которое указано в виде расчета премии
- "По премиям (учитывают пропорционально)" - сумма начисленных премий  деленные на количество месяцев, которое указано в виде расчета премии. Кроме того полученная сумма умножается на отработанные часы и делится на норму времени
- (для средних) По основному заработку, для оплаты по среднему" - основная сумма за месяц (виды расчета, которые войдут в эту сумму можно gосмотреть в базовых у вида расчета средней)

Важную роль в расчете играет Коэффициент, это коэффициент увеличения окладов. Его изменения в системе регистрируются документом "Изменение коэффициентов повышения окладов"
Если Коэффициент изменения окладов введен в середине месяца, то при заполнении табличной части "Расчет средней" за месяц будет введено 2 строки: одна – с начала месяца до ввода нового коэффициента, вторая – с даты ввода коэффициента и до конца месяца. Соответственно все суммы начислений и времени будут разделены пропорционально календарным дням.

Если в информационной базе не внесено достаточно данных для расчета средней, то пользователь может внести суммы вручную. Но необходимо обратить внимание, на порядок заполнения. Важно что месяцы должны вводится от меньшего к большему и строка с видом расчета "(для отпуска) По основному заработку, для расчета отпуска" должна быть последней в каждой группе месяцев.

Столбец "Сумма (полная)" носит чисто информативный характер, тут при расчете отображается начальная сумма, которая используется в дальнейшем расчете, до применения пропорций и коэффициентов

По кнопке "Рассчитать" возле реквизита "Сумма средней" Итоговая сумма в столбце "Сумма" табличной части "Расчет средней" делится на сумму времени, которое используется в расчете (какое время будет использоваться определяется реквизитом "Вид учета времени для средней:")
