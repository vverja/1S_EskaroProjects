Документ «Оприходование материалов из производства»

Документом отражаются операции возврата из незавершенного производства на склад материалов и возвратных отходов.

 

Реквизиты шапки документа:

  • Подразделение - подразделение в управленческом учете, по которому были учтены материальные затраты в производстве.
  • Подразделение организации - подразделение организации (юридического лица), по которому были учтены материальные затраты в производстве.
  • Склад. Склад, на который передаются материалы и возвратные отходы из производства.

 

Для возврата материалов из производства в шапке документа выбирается вид операции «материалы». Данной операцией отражается возврат материалов не только по производственным затратам, но и по другим: затраты по браку в производстве, общепроизводственные, административные затраты, затраты на сбыт, прочие опереационные затраты, вложения во внеоборотные активы.

 

Табличная часть заполняется следующим образом:

 

Материалы

  • Номенклатура. Материал, который возвращается из производства на склад.
  • Характеристика номенклатуры. Характеристика материала, если для  данного материала ведется учет по характеристикам. Значение реквизита выбирается из справочника «Характеристики номенклатуры».
  • Серия номенклатуры. Серия материала, если для  данного материала ведется учет по сериям. Значение реквизита выбирается из справочника «Серии номенклатуры».
  • К.мест, Мест, Ед.мест, Количество, Ед., К - «К. мест» - вводится, если в первичном документе указано количество мест, а не количество ТМЦ. В этом случае необходимо также указать единицу измерения места. Программа выполняет расчет количества ТМЦ и подставляет результат  в строку в реквизит «Количество». Рядом выводится единица измерения этого количества - единица хранения номенклатуры. Выводится также найденный коэффициент пересчета между единицами измерения. В учете операция отражается по количеству в единице хранения.
  • Тип стоимости. Способ определения стоимости списываемых материальных затрат. Стоимость может быть фиксированной, в этом случае она явно указывается в табличной части документа, а может быть рассчитываемой. Тогда указывается только количество списываемых из незавершенного производства материалов, а их стоимость рассчитывается при закрытии месяца.
  • Цена. Цена единицы материала. Реквизит указывается, если выбран тип стоимости «Фиксированная».
  • Сумма. Стоимость материалов в валюте управленческого учета. Реквизит указывается, если выбран тип стоимости «Фиксированная».
  • Сумма (регл). Стоимость материалов для бухгалтерского и налогового учета. Значение реквизита рассчитывается автоматически при изменении суммы в валюте упр. учета. Реквизит указывается, если выбран тип стоимости «Фиксированная».
  • Счет затрат. Исходный счет бухгалтерского учета незавершенного производства.
  • Аналитика. Аналитика учета затрат на счете затрат. Если документ отражается только в управленческом учете, то в качестве аналитики выбирается номенклатурная группа, по которой велся учет затрат в производстве.
  • Счет учета. Счет учета возвращаемых материалов на складе по бухгалтерскому учету.
  • Статья затрат. Статья учета затрат по материалам в производстве. Для производственных затрат статья затрат должна иметь вид затрат «материальные».
  • Заказ затраты. Заказ покупателя или заказ на производство. Если указан заказ, то он будет выступать в качестве аналитики учета затрат при списании затрат с незавершенного производства.
  • Заказ резерв. Заказ покупателя, заказ на производство или внутренний заказ. Материалы, возвращенные из производства, будут зарезервированы на складе под этот заказ.
  • Статус возврата. При возврате материалов из незавершенного производства формируется новая партия. В этом реквизите нужно указать статус партии материалов – «купленный», «продукция» или «в переработку».
  • Качество. Качество возвращаемых материалов. Если из производства возвращаются бракованные материалы, обработка которых еще не начата, то значением реквизита качества может быть «Брак» (если складской учет бракованных материалов требуется вести обособленно от материалов нормального качества.).

Табличную часть можно заполнить в диалоговом окне подбора. При подборе будут показаны остатки по ТМЦ в незавершенном производстве.

Предусмотрено заполнение данной табличной части по кнопке «Заполнить». Возможны следующие варианты заполнения:

  • Заполнить по остаткам. Заполнение производится по остаткам незавершенного производства. При заполнении по остаткам предыдущие значения строк в табличной части стираются.
  • Добавить из требования-накладной. В табличную часть добавляются данные из документов «Требование-накладная», которые выбирает пользователь в открывающемся диалоговом окне.

 

Для оприходования возвратных отходов из производства в шапке документа выбирается вид операции «отходы и брак». В этом случае заполнение табличной части «Материалы» имеет следующие отличия:

Для операции «возвратные отходы» необходимо уточнить что данной операцией оформляется отражение возвратных расходов только из производства.

  • возвратные отходы учитываются только по фиксированной стоимости, поэтому явного указания типа стоимости не требуется;
  • статья затрат должна быть только производственного характера и вид затрат должен быть «материальные»;
  • табличная часть заполняется только вручную, автоматические способы заполнения по кнопке «Заполнить» будут недоступны.
