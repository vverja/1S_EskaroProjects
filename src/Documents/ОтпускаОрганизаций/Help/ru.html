Документ «Отпуска организации»

Документом «Отпуска организации» регистрируются различные отпуска работников организаций: ежегодные, в том числе дополнительные, учебные и пр., кроме отпусков по уходу за ребенком. Информация о видах отпусков по которым будет вестись учет вводится в справочнике "Виды отпусков". Отпуска по уходу за ребенком регистрируются документом «Отпуск по уходу за ребенком».

Учет остатков отпусков организован в зависимости от настройки "Вести учет остатков отпусков" учетной политики организации. Если учет остатков отпусков по организации не ведется то в документе доступна единственная табличная часть "Работники организации", в которой указываются:

  • работник;
  • вид отпуска (можно создавать собственные виды отпуска в справочнике "Виды отпусков");
  • период отпусков;
  • отмечается, нужно ли напомнить о необходимости уточнить и зарегистрировать в программе дату возврата (взводится флажок «Напомнить»);
  • если на время отсутствия (возможно длительного) работника необходимо принять на работу замещающего работника, тогда следует взвести флажок «Освобождать ставку»;
  • указывается, за какой рабочий период предоставляется отпуск.

Для документа в данном случае доступна единственная операция - "Отпуск".

В случае ведения учетов остатков в организации в документе доступны следующие виды операций:

  • отпуск (используется для регистрации различных отпусков работников организаций);
  • компенсация отпуска (используется для регистрации в разрезе рабочих периодов компенсированных отпусков);
  • корректировка отпуска (используется в случае необходимости откорректировать  информацию об использовании отпусков. Также используется для ввода информации об остатках отпусков либо для ввода количества дней отпуска использованых до начала ведения учета);

В зависимости от выбранной операции появляются такие табличные части:

  • работники организации;
  • учет отпусков по периодам;

В табличной части "Работники организации" документа указывается:

  • работник;
  • вид отпуска (можно создавать собственные виды отпуска в справочнике "Виды отпусков");
  • период отпусков;
  • отмечается, нужно ли напомнить о необходимости уточнить и зарегистрировать в программе дату возврата (взводится флажок «Напомнить»);
  • если на время отсутствия (возможно длительного) работника необходимо принять на работу замещающего работника, тогда следует взвести флажок «Освобождать ставку»;
  • указывается, за какой рабочий период предоставляется отпуск.

В табличной части "Учет отпусков по периодам" документа указывается:

  • работник;
  • вид отпуска
  • начало и окончание рабочего периода;
  • количество дней отпуска, которые требуется использовать
  • вспомогательная информация о количестве положенных дней отпуска и об остатке отпусков.

Табличная часть "Учет отпусков по периодам" документа заполняется автоматически (для этого предназначено меню «Заполнить») по данным табличной части "Работники организации". Для каждого работника производится распределение его основного отпуска по рабочим периодам. Информация о полагающемся работнику размере отпуска предварительно следует ввести в регистре "Положенные ежегодные отпуска работников". Рабочий период работника начинается с даты его приема на работу, в колонке "Уже использовано основного" отражается уже использованный отпуск за этот рабочий период, в колонке "Полагается дней основного" отражается возможный размер отпуска (плановый размер пропорционально отработанному периоду), в колонке "Использовать дней основного" отражается сколько дней основного отпуска используется в этом периоде по результатам документа.

Автоматическое заполнение. Документ может быть автоматически заполнен отпусками, внесенными в график отпусков организации, или работниками организации, отобранными по произвольным критериям; для этого предназначено меню «Заполнить» над табличной частью документа.

Ввод на основании. На основании документа «Отпуска организации» можно ввести документы:

  • «Возврат на работу организации», в который будут перенесен список работников, по которым указаны даты возврата из отпусков и выставлен флажок «Напомнить», и будут перенесены даты возврата из отпусков;
  • «Невыходы в организациях», в который будут перенесен список работников, ушедших в неоплачиваемые отпуска.
  • «Начисление отпуска организаций», в котором можно отразить оплату отпуска каждого из работников.

Печатные формы. По документу можно получить две унифицированные печатные формы – П-6 и П-7. Приказы по форме П-6 формируются на каждого работника из документа. Кроме того, можно получить диаграмму использования ежегодных отпусков – вариант печатной формы «Диаграмма».

Проведение документа. По датам начала командировки, указанным в документе, регистрируется состояние работников "В отпуске". По датам окончания командировки, указанным в документе, регистрируется состояние работников "Работает", если в строке сброшен флажок «Напомнить»; если флажок «Напомнить» взведен, регистрируется  "намеченное кадровое событие" - возврат на работу.

При проведении документов программа проверяет, были ли работники, указанные в документе, приняты на работу и не уволены. При обнаружении таких ситуаций выдается сообщение об ошибке и документ не проводится.

Внимание! Для работников, работающих в организации в том числе и по внутреннему совместительству, отпуска регистрируются отдельно для каждого места работы.
