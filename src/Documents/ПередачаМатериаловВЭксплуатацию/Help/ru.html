Документ "Передача материалов в эксплуатацию"

Документ предназначен для отражения в учете операций передачи в эксплуатацию специальной одежды, специальной оснастки и инвентаря.

При вводе документа в шапке необходимо указать следующие реквизиты:

  • Склад. Место хранения, с которого передается в эксплуатацию спецодежда, спецоснатска или инвентарь. Значение реквизита выбирается из справочника "Склады". Реквизит является обязательным для заполнения.

  • Подразделение. Производственное подразделение, в которое передается в эксплуатацию спецодежда, спецоснатска или инвентарь. Значение реквизита выбирается из справочника "Подразделения". Реквизит является обязательным для заполнения.

  • Подразделение организации. Производственное подразделение организации, в которое передается в эксплуатацию спецодежда, спецоснатска или инвентарь. Значение реквизита выбирается из справочника "Подразделения организаций". Реквизит является обязательным для заполнения.

В табличной части необходимо указать список материалов и количество, списываемое со склада. При этом заполняются следующие реквизиты:

  • Материал. Передаваемая в эксплуатацию спецодежда, спецоснастка или инвентарь. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

  • Характеристика материала. Характеристика спецодежды, спецоснастки или инвентаря, если для  данного материала ведется учет по характеристикам. Значение реквизита выбирается из справочника "Характеристики номенклатуры".

  • Серия материала. Серия спецодежды, спецоснастки или инвентаря, если для  данного материала ведется учет по сериям. Значение реквизита выбирается из справочника "Серии номенклатуры".

  • Качество. Качество передаваемой спецодежды, спецоснастки или инвентаря.  Значение реквизита выбирается из справочника "Качество".
  • Мест. Количество мест передаваемой спецодежды, спецоснастки или инвентаря.
  • Единица мест. Единица измерения количества мест передаваемой спецодежды, спецоснастки или инвентаря.
  • Количество. Количество передаваемой спецодежды, спецоснастки или инвентаря. Значение реквизита заполняется автоматически при указании количества мест.
  • Единица мест. Единица измерения передаваемой спецодежды, спецоснастки или инвентаря.
  • Работник. Физическое лицо, которому передается спецодежда, спецоснастка или инвентарь.  Значение реквизита выбирается из справочника "Физические лица" и является обязательным для заполнения.
  • Способ отражения расходов. Способ отражения расходов спецодежды, спецоснастки или инвентаря.
  • Схема назначения использования. Схема назначения использования спецодежды, спецоснастки или инвентаря.
  • Срок полезного использования. Срок полезного использования спецодежды, спецоснастки или инвентаря.
Для целей бухгалтерского и налогового учета материалов в эксплуатации указываются:
  • Счет учета (БУ). Счет бухгалтерского учета спецодежды, спецоснастки или инвентаря на складе (например, 221 для МБП или 1531 для необоротных активов). Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции на основании данных регистра сведений "Номенклатура организаций".
  • Флажок Учитывался как Запас (НУ). По этому флажку определяется счет налогового учета материала. Если флажок установлен, то это "Запасы", иначе - "Оборудование".
  • Статья прироста/убыли запасов. Статья, по которой был приходован материал. Выбирается из справочника "Статьи налоговых деклараций". Реквизит является обязательным для заполнения.
  • Вид налоговой деятельности. Вид налоговой деятельности, по которому материал будет использоваться в эксплуатации. Выбирается из справочника " Виды налоговой деятельности". Реквизит является обязательным для заполнения.
  • Вид деятельности НДС. Вид  деятельности НДС, по которому материал будет использоваться в эксплуатации. Выбирается из перечисления "Виды деятельности НДС". Реквизит является обязательным для заполнения.

Особенности проведения документа

По табличной части формируются движения приход по регистру накопления "Материалы в эксплуатации".

Производится списание регистров накопления "Товары на складах", "Товары организаций", "Партии товаров на складах" по управленческому, бухгалтерскому и налоговому учету и формируются движения приход по регистрам накопления "Партии материалов в эксплуатации...".

В бухгалтерском учете формируется проводка: Дт счета передачи (например, 222) - Кт счета учета (например, 221).
