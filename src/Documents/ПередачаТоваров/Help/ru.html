Документ "Передача товаров"

Документ предназначен для отражения в управленческом, бухгалтерском и налоговом учете следующих операций:

  • Передача материалов переработчику. Передача собственных материалов стороннему переработчику для выпуска продукции по договору на переработку сырья.
  • Передача продукции заказчику. Передача продукции, выпущенной из давальческого сырья, заказчику.

Общие реквизиты документа

В шапке документа можно выбрать следующие реквизиты:

  • Склад. Место хранения, с которого списываются передаваемые материалы или продукция. Значение реквизита выбирается из справочника "Склады". Реквизит является обязательным для заполнения.

  • Заказ. В зависимости от выбранной в документе операции:

    • Передача материалов переработчику -  Заказ поставщику, по которому будет производится выпуск продукции сторонним переработчиком. Значение реквизита выбирается из списка документов "Заказ поставщику". Выбирать можно только заказы с операцией "переработка". При выборе реквизита автоматически заполняются реквизиты "Контрагент" и "Договор". Реквизит является обязательным для заполнения.

    • Передача продукции заказчику - Заказ покупателя, по которому производился выпуск продукции из давальческого сырья. Значение реквизита выбирается из списка документов "Заказ покупателя". Выбирать можно только заказы с операцией "переработка". При выборе реквизита автоматически заполняются реквизиты "Контрагент" и "Договор". Реквизит является обязательным для заполнения.

Закладка "Материалы"

Данная закладка доступна для ввода если в документе выбрана операция "Передача материалов переработчику".

На закладке указываются материалы, которые передаются стороннему переработчику.

  • Номенклатура. Материал, передаваемый в переработку. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

  • Характеристика номенклатуры. Характеристика материала, если для данного материала ведется учет по характеристикам. Значение реквизита выбирается из справочника "Характеристики номенклатуры".

  • Серия номенклатуры. Серия материала, если для данного материала ведется учет по сериям. Значение реквизита выбирается из справочника "Серии номенклатуры".

  • Мест. Количество мест передаваемого материала.

  • Единица. Единица измерения материала.

  • Количество. Количество материала в единице хранения остатков. Значение реквизита заполняется автоматически при указании количества мест.

  • Цена. Залоговая цена материала.

  • Сумма. Залоговая стоимость материала.

Для целей бухгалтерского учета материалов, отданных в переработку, указываются:

  • Счет учета. Счет бухгалтерского учета материалов на складе. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции на основании данных регистра сведений "Номенклатура организаций".
  • Счет передачи. Счет бухгалтерского учета отданных в переработку материалов. Значение реквизита выбирается из плана счетов бухгалтерского учета.

Закладка "Продукция"

Данная закладка доступна для ввода если в документе выбрана операция "Передача продукции заказчику".

На закладке указывается продукция, выпущенная из давальческого сырья, которая передается заказчику.

  • Номенклатура. Продукция, передаваемая заказчику. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

  • Характеристика номенклатуры. Характеристика продукции, если для данной продукции ведется учет по характеристикам. Значение реквизита выбирается из справочника "Характеристики номенклатуры".

  • Серия номенклатуры. Серия продукции, если для данной продукции ведется учет по сериям. Значение реквизита выбирается из справочника "Серии номенклатуры".

  • Мест. Количество мест передаваемой продукции.

  • Единица. Единица измерения продукции.

  • Количество. Количество продукции в единице хранения остатков. Значение реквизита заполняется автоматически при указании количества мест.

Закладка "Тара"

На закладке указывается возвратная тара, в которой передаются материалы или продукция.

  • Номенклатура. Передаваемая тара. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

  • Количество. Количество передаваемой тары в единице хранения остатков.

  • Цена. Залоговая цена тары.

  • Сумма. Залоговая стоимость тары.

Для целей бухгалтерского учета материалов, отданных в переработку, указываются:

  • Счет учета. Счет бухгалтерского учета тары на складе. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции на основании данных регистра сведений "Номенклатура организаций".

Особенности проведения документа

По данным табличной части "Тара" производится списание регистров накопления "Товары на складах", "Партии товаров на складах" по управленческому, бухгалтерскому и налоговому учету и формируются движения приход по регистрам регистров накопления "Товары переданные" и "Партии товаров переданные".

Операция "Передача материалов переработчику"

По данным табличной части "Материалы" производится списание регистров накопления "Товары на складах", "Товары организаций", "Партии товаров на складах" по управленческому, бухгалтерскому и налоговому учету и формируются движения приход по регистрам накопления "Товары переданные" и "Партии товаров переданные".

Операция "Передача продукции заказчику"

По данным табличной части "Продукция" производится списание регистров накопления "Товары на складах", "Товары организаций" и "Товары в резерве на складах".

Данная операция не отражается в бухгалтерском учете.
