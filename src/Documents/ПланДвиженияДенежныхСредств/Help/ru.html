Документ «План движения денежных средств»

Документ «План движения денежных средств» предназначен для ввода прогноза движения денежных средств на период.  В отличие от документов «Заявка на расходование средств» и «Планируемое поступление денежных средств», отражающими события, которые  произойдут с достаточно высокой вероятностью, документом   «План движения денежных средств» вводится «грубый» прогноз. 

Обязательным является заполнение реквизитов «Сценарий», «Дата планирования»,  «Валюта», «Вид денежных средств» заголова документа

Реквизит «Сценарий» определяет периодичность планирования, используемую в данном документе.

Реквизит «Дата планирования» выравнивается по периодичности планирования (на начальную границу).

Планы движения разных видов денежных средств (безналичных, наличных) обязательно вводятся различными документами. Поступление (приход) и списание (расход) денежных средств одного вида можно вносить одним документом.

 

Состав плана вносится в табличную часть документа. План в обязательном порядке задается с точностью до вида движения (приход, расход) и статьи ДДС. Кроме того, по отдельным статьям ДДС план может быть детализирован до проекта, контрагента, подотчетного лица, кассы ККМ, договора, сделки с контрагентом, расчетного документа в расчетах с подотчетным лицом.

 

Состав документа может быть заполнен в автоматизированном режиме, для этого используется форма заполнения, вызываемая по кнопке «Заполнить».

В форме заполнения на двух экранах производится настройка правил заполнения документов. На первом экране настраиваются правила заполнения планируемых поступлений ДДС, на втором - правила заполнения планируемых списаний (платежей) ДДС. Переключение между экранами производится при помощи кнопки «Перейти к платежам/Перейти к поступлениям».

Каждый экран содержит источники планирования, расположенные на нескольких закладках. Закладка «Отбор» каждого экрана используется для фильтрации выбранных источников. В ней можно накладывать условия на отбираемые для заполнения источники. Закладку нужно заполнять отдельно для каждого из экранов.

Собственно заполнение документа производится кнопкой «Выполнить».

 

Предварительные прогнозы движения денежных средств, составленные при помощи этого документа, не отражаются в отчетах «Платежный календарь» и «Планируемые поступления денежных средств». Предварительные прогнозы можно проанализировать при помощи отчета «Планы движения денежных средств».

Запланированные поступления денежных средств можно сравнить с фактическими с помощью отчета «Сравнительный анализ движения денежных средств».
