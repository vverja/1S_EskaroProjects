Документ "Платежное требование (выставленное)"

Документ предназначен для учета поступления безналичных денежных средств по платежным требованиям. Также, с его помощью печатаются формы платежных требований. 

Выбор варианта использования производится из меню, выпадающего по кнопке «Операция».

В управленческом учете документ проводится в два этапа: отражение в оперативном учете (информация о переданном в банк платежном требовании) и отражение движения денежных средств (оплата документа банком, фактическое поступление).

Общие принципы работы с документом аналогичны платежному поручению (входящему).
