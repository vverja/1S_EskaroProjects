Документ "Подтверждение корректировок использования (налоговый учет)"

Документ "Подтверждение корректировок использования (налоговый учет)" предназначен для подтверждения корректировок по факту использования товарно-материальных ценностей на вид налоговой деятельности (либо вид деятельности по НДС), отличный от той, по которой предполагалось использовать ценности первоначально.

 

Предполагаемые корректировки использования в течении периода накапливаются по следующим "направлениям":

  • Партии товаров: предполагаемые корректировки регистрируются при использовании партий на складах или переданных партий товаров, за исключением списания в незавершенное производство, брак и затраты;
  • Незавершенное производство: предполагаемые корректировки регистрируются при расчете себестоимости продукции;
  • Брак в производстве: предполагаемые корректировки регистрируются при расчете себестоимости продукции;
  • Затраты: предполагаемые корректировки регистрируются при распределении затрат, если это не распределение на незавершенное производство и брак;
  • Строительство ОС: предполагаемые корректировки регистрируются при выпуске ОС.

 

В документе присутствуют отдельные  табличные части для этих направлений. Каждую из табличных частей можно заполнить по остаткам накопленных предварительных корректировок с помощью кнопки "Заполнить".

В состав табличных частей входят такие реквизиты:

  • Объект корректировки (возможно с уточняющим значением): для партий товаров, например, это "Номенклатура" и "Документ оприходования";
  • Целевые параметры налогового учета (которые были указаны при приходовании): "Вид деят. НДС", "Вид нал. деят." и признак "Амортизируется";
  • Фактические параметры налогового учета (которые были указаны при использовании): "Вид деят. НДС по факту", "Вид нал. деят. по факту" и признак "Амортизируется по факту";
  • Суммы налогового кредита: показанная при приходовании ("НДС кредит") и при использовании ("НДС кредит по факту");
  • Признак, определяющий, подтверждается ли корректировка налогового кредита ("Метод корректировки"). Если указан метод "На налоговый кредит", то корректировка признается, т.е. сумма подтвержденного налогового кредита по НДС ("НДС кредит подтвержденный") равен сумме н/к, показанной при использовании ("НДС кредит по факту"), при этом корректируются суммы налогового кредита. Если указан метод "На налоговые обязательства", то сумма уменьшения налогового кредита оформляется в виде "условной продажи",то есть начисления налоговых обязательств, с последующей выпиской налоговой  накладной (с видом операции "Условная продажа"). Если же выбран метод "На затраты", то корректировка не признается, и подтвержденный НДС будет равен указанному при поступлении;
  • Суммы валовых расходов: показанная при приходовании ("Сумма в/р") и при использовании ("Сумма в/р по факту");
  • Подтвержденная сумма валовых расходов ("Сумма в/р подтверждённая"), которая по данным "Сумма в/р по факту" и "НДС кредит подтвержденный";
  • Статья валовых расходов ("Статья в/р"), по которой будет производится корректировочные проводки по счету ВР
  • Для табличных частей "Партии товаров" и "Незавершенное производство" выбирается также статья прироста-убыли запасов ("Ст. прироста-убыли запасов"). Это та статья, по которой будет отражаться в "Декларации по налогу на прибыль" использование запасов "не по тому виду деятельности". Для табличной части "Партии товаров" данная статья по умолчанию заполняется как статья, с которой партия была списана при использовании, для табличной части "Незавершенное производство" заполняется предопределённым значением.

Примечание. В табличной части "Строительство ОС" отсутствуют реквизиты, относящиеся к валовым расходам, т.к. при приходовании и строительстве основных средств валовых расходов не возникает, а значит нет и предмета корректировки.

 

Признанные корректировки налогового кредита методом "На налоговый кредит" отражаются на выбранном счете учета НДС, а также отражаются в декларации об НДС по выбранной статье декларации.

В случае, если корректировка налогового кредита не признаётся, то суммы корректировок относятся, по бухгалтерскому учету, на доходы или затраты.

Статья декларации по НДС, счета учета НДС, счет учета НДС (налоговых обязательств), счета доходов и затрат, а также аналитика по этим счетам указываются в шапке документа.
