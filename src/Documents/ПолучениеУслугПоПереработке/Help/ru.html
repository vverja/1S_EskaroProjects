Документ "Получение услуг по переработке"

Документ предназначен для отражения в управленческом, бухгалтерском и налоговом учете операции получения услуги по выпуску продукции сторонним переработчиком и отражения отчета об использованных материалах.

Общие реквизиты документа

  • Заказ. Заказ поставщику, по которому производился выпуск продукции сторонним переработчиком. Значение реквизита выбирается из списка документов "Заказ поставщику". Выбирать можно только заказы с операцией "переработка". При выборе реквизита автоматически заполняются реквизиты "Контрагент" и "Договор". Реквизит является обязательным для заполнения.

Закладка "Продукция"

На закладке указывается продукция, услуги по выпуску которой оформляются данным документом.

  • Номенклатура. Продукция, выпущенная сторонним переработчиком. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

  • Характеристика номенклатуры. Характеристика продукции, если для данной продукции ведется учет по характеристикам. Значение реквизита выбирается из справочника "Характеристики номенклатуры".
  • Серия номенклатуры. Серия продукции, если для данной продукции ведется учет по сериям. Значение реквизита выбирается из справочника "Серии номенклатуры".
  • Мест. Количество мест продукции.

  • Единица. Единица измерения продукции.

  • Количество. Количество продукции в единице хранения остатков. Значение реквизита заполняется автоматически при указании количества мест.
  • Цена. Цена услуги по выпуску продукции сторонним переработчиком.
  • Сумма. Стоимость услуги по выпуску продукции сторонним переработчиком.
  • Номенклатурная группа. Указывается номенклатурная группа продукции. Значение реквизита выбирается из справочника "Номенклатурные группы" и заполняется автоматически при выборе продукции значением, указанным в справочнике "Номенклатура" в поле "Номенклатурная группа".
  • Заказ. Указывается заказ покупателя, в затратах по которому отражаются услуги по переработке. Значение выбирается из списков документов "Заказ покупателя".

Предусмотрено заполнение данной табличной части по кнопке "Заполнить". Заполнение производится по заказу поставщику, указанному в шапке документа.

Закладка "Доп. услуги"

На закладке указывается перечень дополнительных услуг, оказанных переработчиком.

  • Номенклатура. Оказанная услуга. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

  • Количество. Количество оказанных услуг.
  • Цена. Цена оказанной услуги.
  • Сумма. Стоимость оказанной услуги.
  • Статья затрат. Статья затрат, по которой отражается в затратах оказанная услуга. Значение реквизита выбирается из справочника "Статьи затрат".
  • Хар-р затрат. В колонке выводится характер затрат, соответствующий выбранной в строке статье затрат.
  • Вид аналитики. В колонке выводится название аналитики, которую нужно указать для выбранной в строке статьи затрат в колонке "Аналитика".
  • Аналитика. Аналитика статьи затрат. Вводимые значения в данной колонке зависят от характера затрат, выбранной в строке статьи затрат.
  • Заказ. Указывается заказ покупателя, в затратах по которому отражается оказанная услуга. Значение выбирается из списков документов "Заказ покупателя".
Для целей бухгалтерского и налогового учета указываются:
  • Счет затрат. Счет бухгалтерского учета, на котором отражается оказанная услуга. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе статьи затрат.

Закладка "Использованные материалы"

На закладке указываются материалы, которые были использованы переработчиком при выпуске продукции.

  • Номенклатура. Материал, переданный в переработку. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

  • Характеристика номенклатуры. Характеристика материала, если для данного материала ведется учет по характеристикам. Значение реквизита выбирается из справочника "Характеристики номенклатуры".

  • Серия номенклатуры. Серия материала, если для данного материала ведется учет по сериям. Значение реквизита выбирается из справочника "Серии номенклатуры".

  • Мест. Количество мест использованного материала.

  • Единица. Единица измерения материала.

  • Количество. Количество материала в единице хранения остатков. Значение реквизита заполняется автоматически при указании количества мест.

  • Цена. Залоговая цена материла, по которому он был передан в переработку.

  • Сумма. Залоговая стоимость материала.

  • Номенклатурная группа. Указывается номенклатурная группа продукции, на выпуск которой был использован материал. Значение реквизита выбирается из справочника "Номенклатурные группы" и заполняется автоматически при выборе продукции значением, указанным в справочнике "Номенклатура" в поле "Номенклатурная группа".
  • Заказ. Указывается заказ покупателя, на выпуск продукции по которому был использован материал. Значение выбирается из списков документов "Заказ покупателя".
Для целей бухгалтерского учета материалов, отданных в переработку, указываются:
  • Счет учета. Счет бухгалтерского учета отданных в переработку материалов. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции на основании данных регистра сведений "Номенклатура организаций".

Предусмотрено заполнение данной табличной части по кнопке "Заполнить". Заполнение производится материалами, которые были переданы в переработку по заказу, указанному в шапке документа.

Закладка "Учет затрат"

На закладке указываются следующие реквизиты:

  • Статья затрат. Указывается статья затрат, по которой отражаются услуги по выпуску продукции сторонним переработчиком. Значение реквизита выбирается из справочника "Статьи затрат". В реквизите можно указывать только нематериальные статьи затрат.
  • Счет затрат. Счет бухгалтерского учета, на котором отражаются услуги по выпуску продукции сторонним переработчиком и использованные при выпуске материалы. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе статьи затрат.

Закладка "Параметры взаиморасчетов"

На закладке в документе заполняются «Счет учета расчетов с контрагентом» и «Счет учета расчетов по авансам». Эти счета заполняются в соответствии с заданными в регистре сведений «Контрагенты организаций» значениями по умолчанию для каждого контрагента. При необходимости пользователь может изменить корреспонденцию счетов.

Особенности проведения документа

По данным табличной части "Использованные материалы" производится списание регистров накопления "Партии товаров переданные" по управленческому, бухгалтерскому и налоговому учету

По данным табличных частей "Продукция", "Доп. услуги", "Использованные материалы" формируются движения приход регистров в зависимости характера затрат указанных в табличной части статей затрат:

  • Незавершенное производство - для статей с характером затрат "Производственные затраты";
  • Затраты - для статей с характером затрат "Общепроизводственные затраты", "Административные затраты", Затраты на сбыт", "Прочие операционные затраты";
  • Брак в производстве - для статей с характером затрат "Брак в производстве";

По данным табличной части "Использованные материалы" производится списание регистра накопления "Товары переданные". В бухгалтерском учете формируется проводка: Дт счета затрат (например, 231) - Кт счета учета (например, 206).
