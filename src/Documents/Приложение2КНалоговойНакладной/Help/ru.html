Документ "Приложение 2 к налоговой накладной"

Для регистрации изменений количественных и стоимостных показателей выписанных покупателю налоговых накладных, а также для формирования печатной формы соответствующего документа («Расчет корректировки») в типовой конфигурации предусмотрен документ «Приложение №2 к налоговой накладной».

Документ может быть введен только на основании документа «Налоговая накладная». Реквизиты шапки и табличной части документа полностью заполняются из этого документа.

В дополнительных, по сравнению со стандартным набором реквизитов табличных частей документов конфигурации, указывается следующая информация:

·       причина корректировки, необходимая при формировании печатной формы документа;

·       изменение количества по выбранному товару;

·       изменение цены  по выбранному товару;

·       изменение суммы НДС;

·       изменение суммы.

Отметим, в соответствии с действующим законодательством Украины, корректировку по товару можно проводить либо по цене, либо по количеству.
