Документ "Принятие к учету НМА"

Документ «Принятие к учету НМА» предназначен для отражения принятия нематериальных активов к управленческому, бухгалтерскому и налоговому учету.

Принятие НМА к учету означает, что формирование стоимости объекта окончено. Указанная в документе стоимость списывается со счета учета инвестиций (бух. учет) и с регистра «Поступление НМА» (упр. учет).

При принятии к учету для объекта НМА устанавливаются основные характеристики, влияющие на отражение объекта в учете (счет учета, счет начисления амортизации), методы и дополнительные параметры по расчету амортизации (способ отражения расходов, срок полезного использования и т.д.). 

Документ формирует движения постановки объекта на учет и производит запись остальных характеристик в соответствующие регистры сведений:

Для управленческого учета

  • «Первоначальные сведения о нематериальных активах»
  • «Способы отражения расходов по амортизации НМА»
  • «Поступление НМА», в случае, если вид операции "Нематериальные активы", расход по регистру накопления на сумму принятого к учёту НМА.

  • и т.д.

    Для бухгалтерского учета

  • «Первоначальные сведения о нематериальных активах организаций»
  • «Способы отражения расходов по амортизации НМА (бухгалтерский учет)»
  • «Счета бухгалтерского учета нематериальных активов»

  • и т.д.

    Для налогового учета

  • «Первоначальные сведения о нематериальных активах организаций (НУ)»


Документ имеет три вида операций:

  • Нематериальные активы. Собственно принятие к учету нематериального актива.
  • Объекты строительства. Данный вид операции используется в случае, если формирование первоначальной стоимости выполнялось индивидуально для каждого объекта строительства.
  • Ввод начальных остатков.  Режим ввода начальных остатков по НМА.
