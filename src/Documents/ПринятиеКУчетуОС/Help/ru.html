Документ "Принятие к учету ОС"

Документ "Принятие к учету ОС" предназначен для отражения принятия основных средств к управленческому, бухгалтерскому и налоговому учету. Принятие основных средств к учету выполняется только после того, как первоначальная стоимость объекта будет полностью сформирована.

Операция принятия к учету может быть отражена в:

  • управленческом учете;
  • бухгалтерском учете;
  • налоговом учете.

Если в документе указан признак бухгалтерского или налогового учета, то в шапке документа обязательно должен быть заполнен реквизит Организация, на баланс которой принимается основное средство. При проведении документа только по управленческому учету указание организации не требуется.

Закладка "Основные средства"

Для оформления принятия к учету основных средств необходимо указать данные оборудования, которое будет приниматься к учету в качестве объектов основных средств, и перечень объектов основных средств. Такие данные указываются на закладке Основные средства. Одним документом могут быть приняты к учету несколько однотипных основных средств.

 В зависимости от способа формирования стоимости оборудования и вида учета необходимо выбрать значение реквизита Операция:

  • Оборудование. Данный вид операции используется в случае, если до принятия к учету по оборудованию велся количественный учет; стоимость такого оборудования определяется автоматически стоимостью соответствующей партии, которая будет списана, поэтому первоначальная стоимость основных средств явно в документе не указывается. Стоимость каждого основного средства будет равна стоимости единицы оборудования из партии. Способ списания партий указывается в настройках учетной политики для каждого вида учета. Списание партий оборудования выполняется всегда при проведении документов, по другим материальным ценностям момент списания партий определяется параметром Списывать партии при проведении документов в настройках учетной политики.
    Количественный учет оборудования ведется по местам хранения, поэтому в шапке документа необходимо заполнить реквизит Склад. Значение реквизита выбирается из справочника "Склады".

    Для указания объекта оборудования следует заполнить реквизиты:
    • Номенклатура. Номенклатура оборудования, выбирается из справочника "Номенклатура";
    • Характеристика. Характеристика номенклатуры, если для выбранной позиции номенклатуры ведется учет по характеристикам;
    • Серия. Серия номенклатуры, если для выбранной позиции номенклатуры ведется учет по сериям.

      По бухгалтерскому учету для такого оборудования используется, как правило, счета 1521 и 1531, на которых учитывается приобретение основных средств и других необоротных материальных активов, для налогового учета - счет ОБ "Оборудование".  Счет бухгалтерского учета стоимости оборудования в запасе следует указать в реквизите шапки документа Счет БУ. Счет налогового учета определяется автоматически, в зависимости от значения флажка "Учитивалось как запас НУ": если флажок установлен - ЗА "Запасы", иначе - ОБ "Оборудование".

  • Объекты строительства. Данный вид операции используется в случае, если формирование первоначальной стоимости выполнялось либо индивидуально для каждого объекта строительства, либо затраты, связанные с несколькими объектами строительства, не могут быть разделены и поэтому учитываются агрегировано по нескольким объектам.  Стоимость объекта строительства, принимаемого к учету, в этом случае указывается пользователем вручную.

    В шапке документа следует заполнить следующие реквизиты:

    • Объект. Объект строительства, по которому учитывались затраты на принимаемое к учету оборудование, выбирается из справочника "Объекты строительства".
    • Сумма (УУ). Стоимость основного средства, принимаемого к управленческому учету (указывается в валюте управленческого учета).
    • Сумма (БУ). Стоимость основного средства, принимаемого к бухгалтерскому учету (указывается в валюте регламентированного учета);
    • Сумма (НУ). Стоимость основного средства, принимаемого к налоговому учету (указывается в валюте регламентированного учета);
    • Счет (БУ). Счет, на котором по бухгалтерскому учету формировались затраты, связанные с оборудованием, как правило, это счет 1522 "Изготовление основных средств" или 1532 "Изготовление других необоротных материальных активов";

      Стоимость объекта строительства, указанная в шапке, будет одинакова для всех основных средств из табличной части данной закладки, поэтому при проведении документа эта сумма будет распределена равными частями на основные средства, указанные в табличной части документа. Если принимаются к учету несколько основных средств, стоимость которых различна, но объект строительства, на котором учитывались затраты, один, то такие основные средства не считаются однотипными, принятие их к учету следует оформлять разными документами "Принятие к учету ОС".

      По кнопке Рассчитать суммы стоимость объекта строительства, указанного в реквизите "Объект", будет рассчитана автоматически. Сумма всех затрат, накопленных по указанному объекту будет подставлена в реквизиты Сумма(УУ), Сумма (БУ), Сумма (НУ). Эти суммы могут быть скорректированы вручную пользователем.
  • Прочее. Данный вид операции используется только при принятии основных средств к управленческому учету, при выборе данного вида операции флажки бух. учете и налог. учете становятся недоступными для выбора. При таком виде операции можно принять основные средства к управленческому учету без указания какого-либо объекта учета, на котором предварительно аккумулировались затраты по объектам основных средств. Пример: при списании основного средства с управленческого учета какая-либо его часть может быть использована в дальнейшем как самостоятельный объект.

    Стоимость принимаемых к управленческому учету основных средств указывается в валюте управленческого учета в реквизите Стоимость (упр. учет).

Табличная часть документа

В табличной части указывается перечень основных средств, которые будут приняты к учету:

  • Основное средство. Основное средство, выбирается из справочника "Основные средства";
  • Инв. №. Инвентарный номер основного средства с точки зрения управленческого учета, инвентарные номера основных средств по управленческому учету должны быть уникальными в рамках всего предприятия;
  • Инв. № (Орг). Инвентарный номер основного средства с точки зрения регламентированного учета, такие инвентарные номера должны быть уникальными в рамках учета по организации.
  • Количество. Каждое основное средство, принимаемое к учету, является отдельным объектом со своим уникальным инвентарным номером, поэтому количество в таком случае всегда равно 1 и не доступно для изменения. Исключение составляют малоценные основные средства, например, инструмент и т.д., для которых в конфигурации предусмотрен количественный учет.

Закладка "Общие сведения"

На этой закладке указываются общие для всех видов учетов реквизиты для оформления операции принятия к учету.

  • Способы отражения расходов по амортизации. Способы отражения в учете расходов по амортизации, значение выбирается из справочника "Способы отражения расходов по амортизации". Значение реквизита необходимо устанавливать, если основные средства относятся к амортизируемому имуществу и по ним будет выполняться расчет и начисление амортизации.

Закладка "Управленческий учет"

На этой закладке указаны реквизиты, необходимые для отражения операции принятия к управленческому учету основных средств.

  • Подразделение предприятия. Подразделение предприятия, за которым будет числиться основное средство, выбирается из справочника "Подразделения", значение реквизита обязательно к заполнению.
  • Материально-ответственное лицо. Сотрудник предприятия, который является ответственным за основное средство, выбирается из списка работников предприятия, значение реквизита обязательно к заполнению.
  • Входит в состав основного средства. В реквизите устанавливается основное средство, в состав которого входят принимаемые к учету основные средства. Если основное средство не имеет объекта - родителя, то значение реквизита должно быть пустым; значение реквизита не обязательно к заполнению; в дальнейшем с помощью документа "Комплектация ОС" можно будет установить принадлежность основных средств другим объектам. Значение  родительского элемента выбирается из справочника "Основные средства".
  • Состояние принятие к учету. Состояние принятия к учету выбирается из справочника "Состояния ОС" с видом состояния "Принято к учету". При проведении данные о состоянии записываются в регистр сведений, дата записи в регистр будет считаться датой принятия к учету основного средства. Реквизит является обязательным для заполнения.
  • Принятие к учету с вводом в эксплуатацию. Момент принятия к учету может совпадать с моментов ввода в эксплуатацию основного средства, если эти моменты совпадают, то флажок должен быть установлен. При установленном флажке становится доступным для ввода реквизит для указания состояния ввода в эксплуатацию.
  • Состояние ввода в эксплуатацию. Состояние ввода в эксплуатацию, выбирается из справочника "Состояния ОС", при проведении документа данные о состоянии записываются в регистр сведений, дата записи в регистр будет считаться датой ввода в эксплуатацию основного средства.  Реквизит доступен и является обязательным для ввода только при установленном флажке "Принятие к учету с вводом в эксплуатацию".
  • Группа реквизитов для установки параметров амортизации основных средств (реквизиты заполняются только в случае принадлежности основных средств к амортизируемому имуществу):
    • Способ начисления амортизации. Способ расчета амортизации, в зависимости от выбранного способа для заполнения доступны те или иные параметры амортизации.
    • Параметр выработки. Параметр, по которому будет учитываться выработка основных средств в случае выбора способа амортизации "Пропорционально объему продукции" или "По единым нормам амортизационных отчислений (на 1000 км. пробега)", примерами таких параметров могут быть пробег, количество выпущенной продукции, количество технологических операций и т.д. Значение реквизита выбирается из справочника "Параметры выработки ОС".
    • Предполагаемый объем выработки. Максимально возможное значение выработки, которое предусмотрено для основного средства; значение задается только для указанного параметра выработки.
    • Полезный срок использования. Предполагаемый срок использования основного средства в месяцах, значение реквизита может быть указано даже если оно не будет использоваться в алгоритмах расчета амортизации.
    • Ликвидационная стоимость. Ликвидационная стоимость, то есть предполагаемая стоимость, которую предприятие ожидает получить от реализации основного средства по истечении срока его эксплуатации.
    • График амортизации по году. Если в выбранном способе расчета амортизации алгоритм подразумевает вычисление суммы годовой амортизации, то в этом реквизите можно указать график распределения суммы годовой амортизации по месяцам; значение реквизита выбирается из справочника "Годовые графики амортизации ОС".
    • Индивидуальный график. Индивидуальный график, согласно которому рассчитывается амортизация, используется только в случае установки способа амортизации "По индивидуальному графику амортизации"; значение реквизита выбирается из справочника "Индивидуальные графики амортизации".
    • Начислять амортизацию. Признак того, что с момента проведения документа начинается начисление амортизации, реквизит доступен только при установленном значении реквизита "Принятие к учету с вводом в эксплуатацию". При установленном значении реквизита "Начислять амортизацию" становится доступном реквизит "Начислять в текущем месяце".
    • Начислять в текущем месяце. Признак того, что амортизация по основным средствам будет начислена и в месяце ввода в эксплуатацию основных средств, если признак начисления амортизации в текущем месяце не установлен, то амортизация начнет начисляться с месяца, следующего за месяцем, в котором основные средства были введены в эксплуатацию.

Если в документе были заполнены данные закладки "Бухгалтерский учет", то для заполнения закладки "Управленческий учет" данными аналогичными бухгалтерским данным достаточно нажать кнопку Заполнить по данным бух. учета.

Закладка "Бухгалтерский учет"

На этой закладке указаны реквизиты, необходимые для отражения операции принятия к бухгалтерскому учету основных средств.

  • Подразделение организации. Подразделение организации, за которым будет числиться основное средство, выбирается из справочника "Подразделения организации", значение реквизита обязательно к заполнению.
  • Материально-ответственное лицо. Сотрудник организации, который является ответственным за основное средство, выбирается из списка работников организации, значение реквизита обязательно к заполнению.
  • Состояние принятие к учету. Состояние принятия к учету выбирается из справочника "Состояния ОС" с видом состояния "Принято к учету". При проведении данные о состоянии записываются в регистр сведений, дата записи в регистр будет считаться датой принятия к учету основного средства. Реквизит является обязательным для заполнения.
  • Принятие к учету с вводом в эксплуатацию. Момент принятия к учету может совпадать с моментов ввода в эксплуатацию основного средства, если эти моменты совпадают, то флажок должен быть установлен. При установленном флажке становится доступным для ввода реквизит для указания состояния ввода в эксплуатацию.
  • Состояние ввода в эксплуатацию. Состояние ввода в эксплуатацию, выбирается из справочника "Состояния ОС", при проведении документа данные о состоянии записываются в регистр сведений, дата записи в регистр будет считаться датой ввода в эксплуатацию основного средства.  Реквизит доступен и является обязательным для ввода только при установленном флажке "Принятие к учету с вводом в эксплуатацию".
  • Назначение реквизитов для установки параметров амортизации основных средств  полностью аналогично тем, которые используются для управленческого учета. В дополнение к ним также используются реквизиты:
    • Счет учета. Счет бухгалтерского учета стоимости основных средств, принятых к учету, например, счет 104 "Машины и оборудование".
    • Счет амортизации. Счет учета расходов по амортизации основных средств, например, счет 131 "Износ основных средств".

Если в документе были заполнены данные закладки "Управленческий учет", то для заполнения закладки "Бухгалтерский учет" данными аналогичными управленческим данным достаточно нажать кнопку Заполнить по данным упр. учета.

Закладка "Налоговый учет"

На этой закладке указаны реквизиты, необходимые для отражения операции принятия к налоговому учету основных средств.

  • Учитывается как:
    • Производственное. Признак того, что основное средство может амортизироваться в налоговом учете. Принятые к учету производственные средства учитываются в налоговом учете на отдельном счете:
      • ОСО "Основные средства учет по объектам", если оно учитывается пообъектно и не на консервации;
      • ОСВ "Основные средства, выведенные из эксплуатации", если ОС учитывается пообъектно и на консервации;
      • ОСГ "Основные средства учет по группам", если ОС учитывается погруппно. 
    • Непроизводственное. Признак того, что основное средство не амортизируется в налоговом учете. Принятые к учету непроизводственные средства учитываются в налоговом учете на отдельном счете ОСН "Непроизводственные основные средства", при этом вид налоговой деятельности для них всегда равен "Не облагаемая".
    • Запас. Признак должен быть установлен для тех объектов, которые не могут быть признаны основными фондами в налоговом учете. Для таких объектов при принятии к учету партии оборудования списываются, и в дальнейшем эти объекты в налоговом учете не фигурируют. Опция "Учитывается как запас" доступно только при принятии к учету оборудования.
  • Вид налоговой деятельности и Вид деятельности по НДС. Виды налоговой деятельности, для которых будет использовано основное средство. Если при принятии к учету виды налоговой деятельности окажутся отличными от тех, которые были указаны при поступлении оборудования (или затрат на строительство), то будет зарегистрирована необходимость проведения корректировок налогового кредита.
  • Налоговая группа. Группа налогового учета основных средств по классификации налогового учета.
  • Начислять амортизацию. Признак того, что с момента проведения документа начинается начисление амортизации, реквизит доступен только при установленном значении реквизита "Принятие к учету с вводом в эксплуатацию".
