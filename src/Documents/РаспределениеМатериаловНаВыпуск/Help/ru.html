Документ «Распределение материалов на выпуск»

Документ предназначен для ввода данных, обычно содержащихся в так называемых «Производственных отчетах», в которых указывается, какие материалы и в каком количестве были израсходованы за период, и на какую продукцию (услуги) они пошли. То есть фиксируются  данные о прямых материальных затратах. Это может быть удобно, когда надо ввести данные с производственного отчета, например, за день.

 

Прямые затраты в количественном выражении можно указать также в документах выпуска:

  • «Отчет производства за смену».
  • «Акт об оказании производственных услуг».
  • «Поступление товаров из переработки».

 

Заполнение и смысл реквизитов

 

При вводе документа в шапке необходимо указать следующие реквизиты:

  • Заказ - аналитика затрат, можно указать заказ покупателя или заказ на производство.
  • Подразделение организации - подразделение организации (при отражении документа по бухгалтерскому учету).
  • Подразделение - подразделение в управленческом учете, по которому вводится операция.
  • Инвентаризация – инвентаризация незавершенного производства, документ указывается, если предполагается распределение материалов на выпуск заполнять в автоматическом режиме по данным документа инвентаризация.

 

Продукция

 

Группа реквизитов, характеризующая перечень продукции, на которую списываются материалы:

  • Продукция - номенклатура продукции. Обязательный для заполнения реквизит.
  • Характеристика продукции - при необходимости можно указать характеристику, если учет выпуска и складской учет  ведется для этой продукции по характеристикам.
  • Серия продукции - при необходимости можно указать серию, если учет выпуска и складской учет  ведется для этой продукции по сериям.
  • Спецификация - спецификация, по которой изготовлялась продукция.
  • Номенклатурная группа – разрез учета затрат по выпуску продукции.
  • Заказ – заказ покупателя или заказ на производство, под который выпускалась продукция.
  • Счет затрат (БУ). Счет бухгалтерского учета незавершенного производства.

 

Предусмотрены следующие варианты автозаполнения табличной части:

  • Заполнить по нормативам. Для этого способа заполнения необходимо указать значение реквизитов «Период с» «по». Программа анализирует объем выпуска по подразделению за указанный период, и рассчитывает потребность материалов под этот выпуск по нормативам - спецификациям, указанным в документах выпуска. Из полученного количества материалов вычитается количество материалов, указанное как прямые затраты в документах выпуска и других документах «Распределение материалов на выпуск». Полученные данные – перечень продукции выпущенный в подразделении за указанный период – записывается в табличную часть «Продукция», необходимые материалы и их количество, которое требуется согласно нормативу списать из производства на выпуск продукции, записываются в табличную часть «Материалы», а в табличную часть «Распределение материалов» записывается распределение материалов на выпуск согласно нормативам, пропорционально количеству применяемости материалов в продукции. Можно ввести любое количество документов «Распределение материалов на выпуск» за месяц, заполняемых этим способом. Например, этим документом закрывать рабочий день или смену. В этом случае документов выпуска продукции за смену может быть несколько, и в этих документах расход материалов из НЗП можно не указывать. Затем одним документом «Распределение материалов на выпуск» указываем количество материалов, использованных на выпуск по группе (за период) документов выпуска.
  • Заполнить по расчетным данным. Заполнение производится, исходя из остатка прямых затрат в НЗП на конец месяца, определяемым начальным остатком НЗП, приходом в НЗП по требованиям накладным, расходом из НЗП документами выпуска и документами «Распределение материалов на выпуск», «Списание НЗП», «Оприходование материалов из производства», «Инвентаризации НЗП».
  • Заполнить по выпуску. Для этого способа заполнения необходимо указать значение реквизитов «Период с» «по». Программа анализирует объем выпуска по подразделению за указанный период.
  • Добавить из документа выпуска. В табличную часть можно добавить строки из табличных частей документов учета выпуска продукции: «Отчет производства за смену», «Акт об оказании производственных услуг», «Поступление товаров из переработки».

 

Материалы

 

В табличной части «Материалы» необходимо указать список материалов, которые списываются из затрат производства (из НЗП) и включаются в себестоимость выпуска как прямые затраты, и количество этих материалов.

  • Материал - номенклатура материала, полуфабриката, списываемого из производства. Обязательный для заполнения реквизит.
  • Характеристика материала - при необходимости можно указать характеристику, если учет этого материала ведется по характеристикам.
  • Серия материала - при необходимости можно указать серию, если учет для этого материала ведется по сериям.
  • Единица хранения - программа выводит единицу хранения указанной номенклатуры материала.
  • Количество - количество списания из НЗП на выпуск, в единицах хранения.

 

Далее указать следующую группу реквизитов, которые характеризуют разрез затрат, по которому списывается материал из НЗП под выпуск продукции:

  • Номенклатурная группа.
  • Заказ.
  • Статья затрат.
  • Счет затрат (БУ). Счет бухгалтерского учета незавершенного производства.

 

Предусмотрены следующие варианты автозаполнения табличной части:

  • Заполнить по остаткам. Табличная часть заполняется остатками материалов в незавершенном производстве.
  • Заполнить по результатам инвентаризации. Заполнение производится, исходя из:
    • Остатка прямых затрат в НЗП на конец месяца, определяемым начальным остатком НЗП, приходом в НЗП по требованиям накладным, расходом из НЗП документами выпуска и документами «Распределение материалов на выпуск», «Списание НЗП», «Оприходование материалов из производства»;
    • Инвентаризации НЗП. Этот вариант документа - регламентный и вводится один раз в конце месяца (последней датой), если нужно зафиксировать распределение в количественном выражении пропорционально натуральным нормативам остатка затрат, которые образовались после вычета учетного остатка НЗП на конец периода и прямых затрат. Важно: при изменении количества материалов по производственным хозоперациям в течение месяца, регламентный документ «Распределение материалов на выпуск» требует корректировки, в противном случае учетные остатки НЗП на конец месяца разойдутся с результатами инвентаризации, указанными в документе «Инвентаризация НЗП».
  • Добавить из требование-накладная. В табличную часть можно добавить строки из табличной части документов «Требование-накладная».
  • Добавить из документа выпуска. В табличную часть можно добавить строки из табличных частей «Материалы» документов учета выпуска продукции: «Отчет производства за смену», «Акт об оказании производственных услуг», «Поступление товаров из переработки».

 

Распределение материалов

 

В табличной части «Распределение материалов» необходимо указать прямые затраты в количественном выражении по материалам, указанным в первой табличной части - т.е. указать включение строк первой табличной части (списываемых материалов) на выпуск продукции, в разрезах выпуска продукции.

Табличную часть можно заполнить автоматически по кнопке «Заполнить» - «Заполнить распределения». В этом случае каждый материал будет отнесен на всю указанную продукцию пропорционально количеству продукции.

 

Возвратные отходы

 

В табличной части указываются возвратные отходы, которые должны быть отнесены на продукцию. Табличную часть можно заполнить по данным остатков возвратных отходов в незавершенном производстве.

 

Распределение возвратных отходов

 

В табличной части «Распределение возвратных отходов» необходимо указать включение строк табличной части «возвратные отходы» на выпуск продукции, в разрезах выпуска продукции.

Табличную часть можно заполнить автоматически по кнопке «Заполнить» - «Заполнить распределения». В этом случае затраты по возвратным отходам будут отнесены на всю указанную продукцию пропорционально количеству продукции.

 

Отражение в учете

 

Документ формирует записи о включении материалов в продукцию в специальном учетном регистре, только в количественном выражении. Проводки не формируются. Стоимостной расчет для количества прямых затрат производится при закрытии месяца, при этом формируются соответствующие проводки.
