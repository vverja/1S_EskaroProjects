Документ "Установка коэффициентов для заполнения Приложения "СВ" Декларации по прибыли"

Документ определяет структуру производственных затрат в себестоимости реализованной продукции, работ, услуг.

При заполнении Декларации, себестоимость реализованной продукции, работ, услуг распределяется между статьями Приложения пропорционально коэффициентам, заданным в документе.

Документ вводится ежеквартально в последний день квартала.

Сумма коэффициентов в документе должна быть равна единице.

Документ может быть заполнен автоматически. При автоматическом заполнении учитываются:

- производственные затраты отчетного периода: определяются по дебетовым оборотам счета 23;

- производственные затраты в НЗП и нереализованной продукции:

- общая сумма остатков определяется по учетным данным: остатки по счетам 23, 25, 26 на начало периода;

- распределение суммы остатков между статьями Приложения СВ выполняется в соответствии с коэффициентами, данными в документе "Установка коэффициентов для заполнения Приложения "СВ" Декларации по прибыли" в конце предыдущего налогового периода.
