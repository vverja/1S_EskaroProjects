Документ «Расчет по модели бюджетирования»

Документ «Расчет по модели бюджетирования» служит для выполнения сложных расчетов и формирования на их основании групп бюджетных транзакций. Вид исходных данных для расчета определяется реквизитом «Основной источник данных». Если для расчета недостаточно данных основного источника, остальные исходные данные вносятся в таблицу «Дополнительные источники данных».

В качестве основных и дополнительных исходных данных могут выступать:

  • ресурсы (показатели) источника данных для расчетов бюджетирования;
  • реквизиты документа «Бюджетная операция»;
  • произвольные данные, введенные вручную.

После заполнения реквизита «Основной источник данных» и при необходимости закладки «Дополнительные источники данных» возможные для использования в расчетах значения отображаются в таблице «Показатели для расчета». Возможно ручное добавление новых показателей для расчета; при этом необходимо указать представление и значение добавленного показателя.

Табличная часть документа представляет свод правил для формирования бюджетных транзакций. Для удобства работы с табличной частью, данные текущей строки вынесены на закладку «Параметры расчета (по строке таблицы)». Расположенная на этой закладке панель меню позволяет выполнять все стандартные действия с табличной частью (добавление, удаление, редактирование и т.д.). Перемещение по строкам табличной части производится с помощью стрелок вверх/вниз.

Для задания правила необходимо выполнить следующие действия:

  1. Указать статью оборотов, по которой будет формироваться бюджетная транзакция.
  2. При необходимости указать профиль изменения по периодам рассчитанных значений для суммы и количества, при этом датой, относительно которой будут анализироваться смещения, указанные в профиле, будет дата расчета по модели, указанная пользователем;
  3. Указать формулу для расчета количественного и суммового значения транзакции на основании выбранных показателей для расчета. Окно для ввода формулы открывается при нажатии кнопки выбора соответствующего элемента управления.
  4. Указать правила установки значений аналитических измерений транзакции.

Для каждого вида аналитики есть несколько способов задания значения:

  1. «Фиксированное значение» – значение аналитики задается пользователем.
  2.  «По источнику данных расчета» – выбирается в случае, когда соответствующая аналитика присутствует в измерениях источника данных или бюджетной операции, выбранных как поставщик данных для расчета.
  3. «По профилю: распределение» - указывается профиль изменения по соответствующему измерению, в этом случае сумма оборотов после распределения будет равна распределяемому параметру (рассчитанной сумме или количеству), то есть профиль будет использоваться как база для распределения параметра. Например. Для некоторого набора подразделений задан профиль распределения, в котором в качестве коэффициентов базы распределения задана площадь помещений, занимаемая подразделениями. Имея общую сумму затрат по отоплению, можно эту сумму распределить согласно площади занимаемой подразделениями.
  4. «По профилю: умножение» - указывается профиль изменения по соответствующему измерению, однако данные профиля используются не как база распределения, а как коэффициенты, на которые умножается рассчитанное значение. Например. Если профиль задан для подразделений с базой распределения по площади помещений, то зная стоимость затрат на отопление 1 кв. м. площади можно получить затраты, связанные с отоплением каждого подразделения умножив стоимость отопления 1 кв. м. на площадь помещений.


Если в правиле указано и распределение по периодам, и распределение по профилю для измерений, расчет производится в следующем порядке:

  •  сначала производится расчет суммовых и количественных оборотов по указанным для них формулам;
  • проводится распределение по периодам;
  • проводится распределение по профилям для измерений.

Чтобы выполнить расчет и сформировать бюджетные операции по правилам, указанным в модели расчета, необходимо нажать кнопку «Выполнить расчет».

Перечень сформированных по модели расчета бюджетных операций можно посмотреть, нажав кнопку «Просмотр операций». В случае необходимости из режима просмотра можно перейти в режим корректировки и вручную изменить суммы и измерения оборотов, для этого нужно нажать кнопку «Включить возможность изменений».

При проведении расчета формируются проводки на счетах плана счетов бюджетирования.

Реквизит «Регламентный расчет» указывает, будет ли документ включен в состав документов, которые пересчитываются и перепроводятся с помощью обработки «Регламентный пересчет по моделям бюджетирования».
