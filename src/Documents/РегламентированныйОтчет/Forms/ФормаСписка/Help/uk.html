Список регламентованих звітів

Список "Регламентована звітність" містить список всіх введених форм звітності. Список відкривається по кнопці "Журнал звітів" з форми списку довідника "Регламентовані звіти". При відкритті журналу автоматично встановлюється відбір по тому звіту, на якому перебував курсор  довіднику "Регламентовані звіти" в  момент натискання кнопки

При відкритті раніше введених звітів відразу відкривається форма звіту, без виклику "стартової" форми
