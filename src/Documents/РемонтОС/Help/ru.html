Документ "Ремонт ОС"

Документ "Ремонт ОС" предназначен для отражения в учете ремонта основных средств для целей управленческого, бухгалтерского и налогового учета.

В шапке документа вносятся сведения об объекте строительства, с которого списываются затраты по ремонту, для бухгалтерского учета - счет объекта строительства, а так же суммы ремонта, отдельно для каждого из видов учета.

Сумму ремонта можно получить автоматически - при нажатии кнопки "Рассчитать суммы" суммы ремонта заполнятся автоматически по данным учета на момент формирования документа. При необходимости эти суммы можно откорректировать вручную.

Если установить флажок "Использовать общий способ отражения расходов", то затраты на ремонт в управленческом и бухгалтерском учете будут списаны по способу отражения затрат, указанному в шапке документа. В противном случае используются способы отражения амортизации ремонтируемых основных средств.

В табличной части "Основные средства" указываются объекты основных средств, которые ремонтируются.

Для более быстрого заполнения табличной части документа  необходимо воспользоваться кнопкой "Подбор".

После формирования списка основных средств необходимо воспользоваться кнопкой "Заполнить" для  распределение сумм ремонта равными частями с учетом погрешности.
