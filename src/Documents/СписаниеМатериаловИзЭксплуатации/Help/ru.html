Документ "Списание материалов в эксплуатации"

Документ предназначен для отражения в учете операций списание из эксплуатации специальной одежды, специальной оснастки и инвентаря.

При вводе документа в шапке необходимо указать следующие реквизиты:

  • Указывать документ передачи.  Если флажок установлен, то становиться возможным указание документа "Передача материалов в эксплуатацию", по  которому были переданы материала в эксплуатацию. В этом случае указание документа является обязательным.

ВАЖНО! При работе в режиме расширенной аналитики, заполнение флажка и указание документа передачи обязательно

  • Подразделение. Производственное подразделение, из которого списывается из эксплуатации спецодежда, спецоснастка или инвентарь. Значение реквизита выбирается из справочника "Подразделения". Реквизит является обязательным для заполнения.

  • Подразделение организации. Производственное подразделение организации, из которого списывается из эксплуатации спецодежда, спецоснастка или инвентарь. Значение реквизита выбирается из справочника "Подразделения организаций". Реквизит является обязательным для заполнения.

В табличной части необходимо указать список материалов и количество, возвращаемое на склад. При этом заполняются следующие реквизиты:

  • Материал. Списываемая из эксплуатации спецодежда, спецоснастка или инвентарь. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

  • Характеристика материала. Характеристика спецодежды, спецоснастки или инвентаря, если для  данного материала ведется учет по характеристикам. Значение реквизита выбирается из справочника "Характеристики номенклатуры".

  • Серия материала. Серия спецодежды, спецоснастки или инвентаря, если для  данного материала ведется учет по сериям. Значение реквизита выбирается из справочника "Серии номенклатуры".

  • Качество. Качество списываемой спецодежды, спецоснастки или инвентаря.  По этому качеству материал когда-то был передан в эксплуатацию. Значение реквизита выбирается из справочника "Качество".
  • Мест. Количество мест списываемой спецодежды, спецоснастки или инвентаря.
  • Единица мест. Единица измерения количества мест списываемой спецодежды, спецоснастки или инвентаря.
  • Количество. Количество списываемой спецодежды, спецоснастки или инвентаря. Значение реквизита заполняется автоматически при указании количества мест.
  • Работник. Физическое лицо, из которого списывается спецодежда, спецоснастка или инвентарь.  Значение реквизита выбирается из справочника "Физические лица" и является обязательным для заполнения.
  • Способ отражения расходов. Способ отражения расходов спецодежды, спецоснастки или инвентаря.
  • Схема назначения использования. Схема назначения использования спецодежды, спецоснастки или инвентаря.
  • Срок полезного использования. Срок полезного использования спецодежды, спецоснастки или инвентаря.

В документе организовано автоматическое заполнение по остаткам. При нажатии на кнопку заполнения командной панели открывается форма отбора остатков, в которой можно задать необходимые условия для заполнения табличной части документа. Если установлено несколько флажков отражения в учетах, то количество остатков берется максимальным по выбранным учетам. При установленном флажке "Указывать документ передачи" заполнение производится в разрезе документов передачи.

Особенности проведения документа

По табличной части формируются движения расход по регистрам накопления "Материалы в эксплуатации"  и "Партии материалов в эксплуатации...". В режиме расширенной аналитики формируются движения по регистрам "Материалы в эксплуатации"  и "Учет затрат".

В бухгалтерском учете формируется  проводки, по закрытию амортизации. (Используется только для необоротных активов).
