Документ «Списание незавершенного производства»

 

Документом отражается операция списания из незавершенного производства материальных ценностей. По регламентированному учету такое списание относится  на счета потерь, недостач или прочие счета, указанные пользователем.

Также для типа стоимости "Фиксированная" документ может формировать приходные движения по регистрам "Затраты", "Затраты (бухгалтерский учет)", "Затраты (налоговый учет)". 

 

Реквизиты шапки документа:

  • Подразделение - подразделение в управленческом учете, по которому были учтены затраты.
  • Подразделение организации - подразделение организации (юридического лица), по которому были учтены затраты.
  • Инвентаризация. Указывается документ «Инвентаризация НЗП», далее табличную часть документа можно заполнить данными, указанными в инвентаризации, для этого нужно выбрать кнопку «Заполнить»-«Заполнить по инвентаризации».
  • Тип стоимости. Способ определения стоимости списываемых материальных затрат. Стоимость может быть фиксированной, в этом случае она явно указывается в табличной части документа, а может быть рассчитываемой. Тогда указывается только количество списываемых из незавершенного производства материалов, а их стоимость рассчитывается при закрытии месяца.

 

Документ в общем случае имеет две закладки: "Материалы", "Бухгалтерский учет" (для расчетной стоимости), или "Учет затрат" (для фиксированной стоимости).

Закладка «Учет затрат» показывается при выборе фиксированного типа стоимости.

На закладке указывается счет, на который списываются затраты, и аналитика выбранного счета.

При включеном флажке "Отражать по затратным регистрам" при проведении документ будет формировать приходные движения по регистрам "Затраты", "Затраты (бухгалтерский учет)", "Затраты (налоговый учет").

На закладке необходимо заполнить реквизиты для отражения таких приходных движений по затратным регистрам: статью затрат, подразделение, подразделение организации, номенклатурную группу, заказ, проект, вид налоговой деятельности, вид деятельности НДС, признак "Амортизируется".

 

Закладка «Бухгалтерский учет» показывается при включении флажка отражения документа в бухгалтерском учете для расчетной стоимости. На закладке указывается счет, на который списываются затраты, и аналитика выбранного счета.

 

Материалы

  • Номенклатура. Перечень списываемых ТМЦ, обязательный для заполнения реквизит.
  • Характеристика номенклатуры - заполняется при необходимости, если учет затрат ведется по характеристикам для этой номенклатуры.
  • Серия номенклатуры - заполняется при необходимости, если учет затрат ведется по сериям для этой номенклатуры.
  • К.мест, Мест, Ед.мест, Количество, Ед., К.  «К. мест» - вводится, если в первичном документе указано количество мест, а не количество ТМЦ. В этом случае необходимо также указать единицу измерения места. Программа выполняет расчет количества ТМЦ и подставляет результат  в строку в реквизит «Количество». Рядом выводится единица измерения этого количества - единица хранения номенклатуры. Выводится также найденный коэффициент пересчета между единицами измерения. В учете операция отражается по количеству в единице хранения.
  • Статья затрат. Статья затрат по которой расходы были учтены в незавершенном производстве. Статья затрат должна иметь производственный характер.
  • Номенклатурная группа. Номенклатурная группа продукции, на которую были отнесены расходы в незавершенном производстве.
  • Заказ. Заказ на производство или заказ покупателя, к которому привязаны расходы в незавершенном производстве.
  • Цена. Реквизит доступен для ввода, если тип стоимости фиксированный. Это цена единицы ТМЦ, списываемого из незавершенного производства.
  • Сумма. Сумма по управленческому учету, которая должна быть списана из НЗП.
  • Сумма (Регл) Сумма по бухгалтерскому и налоговому учету, которая должна быть списана из НЗП.
  • Сумма НДС Сумма НДС по налоговому учету, которая должна быть списана из НЗП.
  • Счет затрат (БУ). Счет, по которому были учтены затраты в бухгалтерском учете.
  • Вид нал. деят. Вид налоговой деятельности по которому затраты были учтены в незавершенном производстве.
  • Вид деят. НДС Вид деятельности НДС по которому затраты были учтены в незавершенном производстве.
  • Аморт. Признак "Амотризируется" по которому затраты были учтены в незавершенном производстве.

 

Табличную часть можно заполнить в диалоговом окне подбора. При подборе будут показаны остатки по ТМЦ в незавершенном производстве.

Документ может быть автоматически заполнен при нажатии кнопки «Заполнить». Доступны следующие стратегии заполнения:

  • Заполнить по остаткам. Табличная часть заполняется материалами, которые числятся в НЗП на дату документа.
  • Заполнить по инвентаризации. Копируются данные из табличной части «Материалы» документа «Инвентаризация НЗП», который указан в шапке документа.
  • Добавить из требования-накладной. В табличную часть добавляются данные из документов «Требование-накладная», которые выбирает пользователь в открывающемся диалоговом окне.

 

Особенности проведения

 

Материальные ценности по разрезу затрат, указанному в документе, должны быть в наличии в производстве в количественном и стоимостном выражении (аналитика: статье затрат, характеристике, серии, номенклатурной группе).

 

Допускается списание большего количества, чем есть к моменту операции в НЗП. В этом случае образовавшийся отрицательный остаток должен быть «перекрыт» документами «Требование-накладная» вводимыми позже.

Для типа стоимости "Фиксированная", при включеном флажке "Отражать по затратным регистрам" при проведении документ будет также формировать приходные движения по регистрам "Затраты", "Затраты (бухгалтерский учет)", "Затраты (налоговый учет)".

Если налоговые измерения НЗП (на закладке "Материалы") и затрат (на закладке "Учет затрат") различаются, при необходимости также формируются движения по корректировкам использования НЗП.
