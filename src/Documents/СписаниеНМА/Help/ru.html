Документ "Списание НМА"

Документ «Списание НМА» предназначен для отражения в учете выбытия нематериального актива, в связи с моральным и физическим износом, а также при ликвидации в связи с чрезвычайной ситуацией.

  • управленческому учету;
  • бухгалтерскому учету;
  • налоговому учету.

Выбывающие объекты выбираются из справочника "Нематериальные активы".

Для каждого объекта НМА указывается текущая балансовая стоимость, сумма накопленной на начало месяца амортизации, расчетная сумма амортизации, которая должна быть доначислена за текущий месяц.

Для более быстрого заполнения табличной части документа  необходимо воспользоваться кнопкой "Подбор".

После формирования списка нематериальных активов, подготавливаемых к передаче необходимо воспользоваться кнопкой "Заполнить" для заполнения табличной части  информацией об нематериальных активах по данным учета (регистра накопления "Стоимость НМА" и "Стоимость НМА (бухгалтерский учет)", сумма амортизации за текущий месяц определяется расчетным путем в документе.

Отражение в управленческом учете

При списании формируются движения по регистру накоплению "Стоимость НМА", доначисленная за месяц амортизация распределяется по регистрам учета затрат аналогично документу "Амортизация НМА".

Отражение в бухгалтерском учете

При списании с учета каждого из перечисленнных в табличной части нематериального актива, формируются проводки:

• по доначислению амортизации, если это необходимо;
• по списанию амортизационных начислений и балансовой стоимости.
• по списанию остаточной стоимости НМА на указанный в форме счет учета расходов. 

Дополнительно в регистрах сведений снимается флаг начисления амортизации по перечисленным в документе объектам учета.

Доначисленная амортизация распределяется по регистрам учета затрат, аналогично документу "Амортизация НМА".

В качестве счета учета расходов при списании нематериальных активов по бухгалтерскому учету по умолчанию предлагается счет 976 "Списание необоротных активов".
