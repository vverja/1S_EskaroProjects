Документ "Регламентные операции налогового учета по налогу на прибыль"

Документ предназначен для выполнения регламентных действий, которыми сопровождается окончание налогового периода в учете налога на прибыль.

Документ следует вводить в конце налогового периода, после того, как в системе отражены все первичные операции по налоговому учету налога на прибыль для указанной Организации. Налоговый период "привязан" к дате подачи декларации о налоге на прибыль и, как правило, равен кварталу. В редких случаях нормативные документы обязывают подавать декларацию не по окончании квартала, и для этих целей в документе можно установить другой период при помощи реквизита Начало налогового периода.

Набор реквизитов "Выполняемые действия" описывает те регламентные действия, которые выполняет документ. Снятие/установка флажков соответствующих реквизитов позволяет отменить/выполнить указанное действие. Следует понимать, что для правильного окончания налогового периода все действия должны быть выполнены, поэтому флажки рекомендуется снимать только для того, чтобы обособленно проанализировать движения документа по каждому из действий.

Выполняемые действия означают следующее:

  • Распределить амортизацию по видам налоговой деятельности - это действие предусматривает, что амортизация тех объектов основных средств, которые используются в нескольких видах налоговой деятельности, будет распределена по конкретным видам в пропорции, заданной в табличной части.
  • Распределить суммы улучшения ОС - выполнение этого действия означает, что накопленные суммы модернизации будут распределены между балансовой стоимостью основных средств и валовыми расходами. Сумма валовых расходов рассчитывается исходя из разрешенного годового процента, заданного в параметрах налогового учета. Суммы валовых расходов по улучшению основных средств, используемых в нескольких видах налоговой деятельности, распределяются между конкретными видами аналогично амортизации.
  • Сформировать остаток незавершенного производства - это действие предназначено для формирования актуального остатка прироста/убыли незавершенного производства. Имеющийся остаток незавершенного производства на счете ЗА "Запасы" будет списан, а новый остаток будет сформирован по данным регистра накопления "Незавершенное производство (налоговый учет)".

Табличная часть "Распределение вал. расх. и амортизации по видам налоговой деятельности" задает пропорции между видами налоговой деятельности, используемые при распределении валовых расходов и амортизации, описанных выше. При нажатии кнопки "Заполнить" табличная часть заполняется суммами валовых доходов, полученных от разных видов налоговой деятельности.
