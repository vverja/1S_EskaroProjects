Документ «Установка базы распределения затрат»

В документе вручную устанавливаются базы распределения косвенных затрат для тех способов распределения, где выбран вариант базы распределения «Вручную».

В документе в реквизите «Способ распределения» необходимо указать способ распределения с базой распределения «Вручную». Табличную часть документа можно заполнить по шаблону - способу распределения затрат с любой базой распределения, кроме «Вручную». Шаблон задается в реквизите «Способ распред. (шаблон)», затем нужно нажать кнопку «Заполнить» - «Заполнить по шаблону». Табличная часть будет заполнена данными по стратегии, определенной в шаблоне. Такие данные можно в дальнейшем скорректировать вручную.

 

Особенности проведения

 

База распределения косвенных затрат будет запоминаться в регистрах сведений:

  • База распределения затрат – для целей управленческого учета.
  • База распределения затрат (бухгалтерский учет) – для целей бухгалтерского учета.
  • База распределения затрат (налоговый учет) – для целей налогового учета.
  • База распределение затрат (международный учет) – для целей МСФО.
