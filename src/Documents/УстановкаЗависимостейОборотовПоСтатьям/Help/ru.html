Документ «Установка зависимости оборотов по статьям»

Документ предназначен для установки зависимостей между бюджетными транзакциями. Установленные такими документами зависимости используются в дальнейшем в документах «Бюджетная операция» и «Расчет по модели бюджетирования».

В случае, если измерения какой – либо формируемой этими документами бюджетной транзакции совпадают с установленными ранее в качестве влияющих, автоматически формируются зависимые транзакции по правилам, определяемым в табличной части документа «Установка зависимостей оборотов по статьям».

Соответственно, измерения зависимой транзакции также могли быть использованы в качестве влияющих, таким образом формируется иерархическая структура зависимости.

Настройка зависимости заключается в следующем. В шапке документа указываются значения аналитических измерений бюджетирования (сценарий, статья оборотов, ЦФО, проект, контрагент, номенклатура), при совпадении с которыми измерений сформированной бюджетной транзакции будет запущен механизм формирования зависимых оборотов по правилам, указанным в табличной части документа.

Обязательным к заполнению является реквизит «Статья оборотов», остальные заполняются по необходимости. Таким образом, дается возможность для одной и той же статьи оборотов, но разной аналитики, например, по проектам, формировать различные ветки зависимостей.

Соответственно, для каждой зависимой транзакции в табличной части документа указываются:

  • зависимая статья оборотов;
  • значения аналитических измерений зависимой транзакции (номенклатура, контрагент, проект, ЦФО); если значения не указаны явно, они подставляются из аналитики влияющей транзакции;
  • профиль изменения по периодам (при необходимости разнесения по времени влияющей и зависимой транзакций);
  • если для зависимой статьи ведутся суммовые обороты, указывается источник получения таких оборотов (количественные или суммовые обороты из влияющей транзакции) и коэффициент для получения суммового значения зависимого оборота;
  • если для зависимой статьи ведутся количественные обороты, указывается источник получения таких оборотов (количественные или суммовые обороты из влияющей транзакции) и коэффициент для получения количественного значения зависимого оборота.

Установленная модель зависимости начинает действовать с момента (даты) проведения документа.

Важно. Для предотвращения зацикливания алгоритма формирования зависимых транзакций, не допускается использование в качестве зависимой статьи оборотов, которая уже была выбрана в качестве влияющей в одном из предыдущих узлов структуры зависимостей.

Для просмотра существующей структуры зависимостей используется обработка «Структура зависимостей оборотов».
