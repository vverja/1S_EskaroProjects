Документ «Установка соответствия счетов БУ и МСФО»

Предназначен для настройки правил трансляции данных (в первую очередь - проводок) из украинского бухгалтерского учета в международный учет.

На закладке «Соответствия счетов» указываются правила соответствия записей по счетам национального (украинского) бухгалтерского учета записям по счетам международного учета. Указанные правила будут использоваться при трасляции данных начиная с даты, выбранной в шапке документа.

Табличная часть содержит следующие реквизиты:

  • «Счет хозрасчетный» -  выбирается счет бухгалтерского учета, для которого задается правило соответствия;
  • «Вид движения» - указывается, будет ли это соответствие оборотов по дебету или по кредиту счета;
  • «Субконто хозр.» - указывается конкретное значение (значения) субконто счета бухгалтерского учета, если именно для этого значения (набора значений) задается особое правило соответствия;
  • «Счет международный» - выбирается счет международного учета, для которого задается правило соответствия;
  • «Субконто межд.» - указывается конкретное значение (значения) субконто счета международного учета, если именно для этого значения (набора значений) задается данное правило соответствия;
  • «Реквизит» - указывается реквизит субконто счета бухгалтерского учета, если правило соответсвия задается для значения этого реквизита;
  • «Значение» - указывается значение реквизита субконто счета бухгалтерского учета (если заполнен предыдущей реквизит);
  • «Комментарий» - произвольный комментарий.

Нажатием на кнопку  «Заполнить» можно автоматически заполнить табличную часть документа по уже имеющимся в информационной базе данным о правилах соответствия счетов либо импортировать такие правила из внешнего xml-файла.

На закладке «Исключения проводок» можно указать проводки украинского бухгалтерского учета, которые не будут транслироваться в международный учет, даже если на закладке «Соответствия счетов» все правила для такой трансляции заданы.

При проведении документа данные фиксируются в регистрах сведений «Соответствие счетов БУ и МСФО» и «Исключение проводок».
