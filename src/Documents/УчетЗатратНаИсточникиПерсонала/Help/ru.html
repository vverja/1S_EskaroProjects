Документ "Учет затрат на источники персонала"

Документ предназначен для регистрирации затрат на привлечение кандидатов в разрезе источников информации.

 

Затраты вводятся в табличную часть документа. Для каждой строки табличной части указываются:

  • Дата
  • Источник информации
  • Сумма
  • Валюта
  • Курс

 

Для анализа затрат предназначен отчет "Эффективность затрат на привлечение".
