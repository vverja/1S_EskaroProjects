Документ «Задание агента»

Документ Используется для выгрузки в МУ документов «Задание». Выгрузка этих документов не является необходимой в процессе выгрузки данных из ЦБД в МУ. Документы «Задание агента» предназначены для постановки сотруднику списка задач, которые ему необходимо выполнить в течение дня. Например, создание какого-либо документа, создание фотографии, обмен данными и т.п.

Данный документ может быть сформирован на основании документов "Планирование для агентов" или "Маршрут".

При формировании документа следует заполнить следующие реквизиты:

1. Организация, от которой оформляется документ.

2. Агент, которому предназначается данный документ.

3. Контрагент, для которого требуется выполнить задания.

4. Торговая точка, при использовании режима посещений по торговым точкам.

 

Заполнение списка заданий для агента можно выполняться несколькими способами:

1. Можно формировать новые задачи вручную, выбирая пункты заданий из справочника "Виды заданий"

2. Можно заполнить дерево заданий целиком элементами из справочника "Виды заданий", при этом дерево задач будет заполнено с соблюдением иерархии из справочника "Виды заданий" - для этого нужно нажать кнопку "Заполнить", расположенную на командной панели страницы "Перечень задач".

3. Также можно добавлять нужные задачи из справочника "Виды заданий" методом подбора - для этого нужно нажать кнопку "Подбор", расположенную на командной панели страницы "Перечень задач".

При добавлении нового элемента в список заданий агента можно указывать комментарий к заданию и обязательность выполнения определенного задания. При установке этого флажка, в МУ будет невозможно сохранить документ без указания выполнения этой задачи.

В результате загрузки данных от агентов по выполнению заданий производится заполнение результатов выполнения задач. Результаты отображаются в колонке "Результат" - видимость данной колонки регулируется с помощью флажка "Показывать результаты".

На вкладке "Дополнительно" есть флажок "Обновить в МУ" - он используется в том случае, если текущий документ требуется принудительно
обновить даже если он уже был выгружен ранее и отредактирован агентом в МУ. При этом документ задание в МУ будет полностью обновлен.

Примечание: Если требуется назначить категорию текущему документу, то это нужно сделать также на вкладке "Дополнительно", а не через регистр сведений, как для остальных документов.

Поддерживается возможность распечатки заданий агента - для этого используется кнопка "Печать", расположенная на панели управления в нижней части формы.
