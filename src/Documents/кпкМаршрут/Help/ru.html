Документ «Маршрут»

Документ используется для указания порядка посещения торговых точек или клиентов (в зависимости от того, какой режим использования торговых точек указан в документе-основании «План посещения»), т.е. маршрута.

Вводится на основании документа "План посещений" на каждую дату. Данный документ можно сформировать только на основании документа "План посещений", формирование документа другим способом запрещено.

Использование документа имеет смысл только в том случае, если порядок следования торговых точек в плане посещения не соответствует требуемому порядку посещения, а также может быть использован для включения внеплановых посещений в маршрут, указания рекомендуемого времени посещения, указания напоминаний торговому агенту к любому из пунктов маршрута.

В форме редактирования документа для каждого пункта маршрута можно создавать/редактировать документ "Задание агента". Для этого используются кнопки расположенные в подменю "Задание" панели управления табличной части.
