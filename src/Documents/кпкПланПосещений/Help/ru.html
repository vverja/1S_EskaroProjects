Документ «Планирование для агентов»

Документ используется для составления планов посещений торговых точек или клиентов для торговых агентов, а также для формирования данных по предполагаемым объемам и суммам продаж для агентовв разрезе номенклатурных групп.

Данный документ может формироваться в 3-х режимах, способ формирования документа переключается с помощью меню "Операция" основной панели формы.

1. Планирование плана посещений (вкладка "План посещений");

2. Планирование продаж агентами (вкладка "План продаж").

3. Планирование посещений и продаж агентами.

 

Планирования посещений:

Предусмотрено два типа формирования документа (в зависимости от того, какой тип константы «Тип планирования посещений» указан на форме "Настройка параметров работы с мобильным приложением"):

  • Использование плана посещений с использованием торговых точек. В этом случае в табличной части документа будут отображаться колонки "Торговая точка" и "Контрагент".
  • Использование плана посещений с использованием клиентов. В этом случае в табличной части документа будет отображаться только колонка "Контрагент".

При формировании документа "Планирование для агентов" необходимо выбрать агента, для которого формируется данный план, а также интервал планирования, который может быть равен месяцу, недели, 2 неделям, 4 неделям или быть заданным пользователем

Редактировать табличную часть документа можно как в диалоге документа, так и в печатной форме плана.

Для удобства заполнения табличной части документа могут применяться кнопки, расположенные на командной панели табличной части:

  • Кнопка "Заполнить" производит заполнение текущаго плана посещений на основании другого плана или торговыми точками, присвоенных агенту (присвоение торговых точек агенту производится в справочнике "Торговые точки").
  • Кнопка "Подбор" используется для подбора клиентов/торговых точек по зонам или по клиентам.
  • Кнопка "Сортировка" предназначена для сортировки введенных значений по клиентам или по зонам.

 

На панели усправления в нижней части формы расположена кнопка "Обновить маршруты", которая предназначена для автообновления документов "Маршрут", введенных на основании текущего плана посещений, например, в случае изменения текущего плана.

 

На основании текущего документа может быть введен документ "Задание агента". Для удобства формировании этих документов на форме предусмотрена возможность их создания/редактирования либо с помощью контекстного меню (для этого нужно кликнуть правой кнопкой мыши по нужной ячейке в текущей строке табличной части и выбрать необходимое действие), либо с помощью кнопок, расположенных в подменю "Задание" командной панели табличной части "План посещений".

Таким образом, если для какого-либо дня установлен документ "Задание агента", то эта ячейка будет подсвечиваться зеленым цветом.

 

Планирования объемов продаж:

В табличной части документа на вкладке "Плана продаж" может указываться оценка предполагаемых объемов продаж для номенклатурных групп. При этом указывается предполагаемое количество и сумма продаж по каждой отдельно взятой номенклатурной группе.

Строки табличной части могут заполняться подбором из справочника "Номенклатурные группы" с помощью кнопки, расположенной на командной панели табличной части.
