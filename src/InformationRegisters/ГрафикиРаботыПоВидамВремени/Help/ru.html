Регистр сведений «Графики работы по видам времени»


В этом регистре хранится информация о времени, которое используется для расчета в регистре расчета "Основные начисления работников организаций".


Описание измерений регистра:

  • График работы - либо справочник график работы, либо работник по действительному месту работы ;

  • Вид учета времени - "По дням", "По часам", "По вечерним часам", "По ночным часам";

  • Дата - дата года;

  • Месяц - дата, первое число месяца;

  • План - либо "Да", либо "Нет";

  • Сотрудник - работник, его основное место работы, заполняется документами "Табель учета рабочего времени организации", "Ввод индивидуальных графиков работы организации". Пара Сотрудник - График полностью определяет работника, эти реквизиты будут различными в случае регистрации индивидуальных графиков внутреннего совместительства и замен. 


    Описание ресурсов регистра:

  • Основное значение -  для вида учета времени - "По дням" - смены; для вида учета времени - "По часам" - часы; "По вечерним часам", "По ночным часам" - часы;

  • Дополнительное значение -  для вида учета времени - "По дням" - часы, для вида учета времени - "По часам" - смены; "По вечерним часам", "По ночным часам" - 0.


    Описание реквизитов регистра:

  • Документ - документом может быть: "Табель учета рабочего времени организации", "Сдельный наряд на выполненные работы", "Ввод индивидуальных графиков работы организации".
