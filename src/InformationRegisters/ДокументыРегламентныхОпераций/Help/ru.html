Регистр сведений «Документы регламентных операций»

Регистр сведений предназначен для настройки соответствия между регламентной операцией и видами документов, которые должны быть созданы и проведены для ее выполнения (кнопка командной панели «Документы регламентных операций» формы «Настройка закрытия месяца»).

Для одной регламентной операции может быть задано произвольное количество документов. Как правило, указывается один вид документа.

Список может быть заполнен автоматически значениями по умолчанию (кнопка командной панели «Заполнить по умолчанию»), перед этим все ранее созданные записи будут удалены. Для регламентной операции «Начислить зарплату» документы при заполнении по умолчанию не устанавливаются.

При необходимости список соответствий может быть отредактирован. Для некоторых регламентных операций документ может не указываться.
