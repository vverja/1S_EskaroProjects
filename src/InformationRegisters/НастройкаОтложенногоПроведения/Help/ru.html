Список организаций для отложенного проведения

Предназначен для настройки списка организаций, для которых используется отложенное проведение. Для каждой организации необходимо указать

  • дату начала действия отложенного проведения
  • настройку, в соответствии с которой будет выполняться допроведение документов

См. также механизм «Отложенное проведение».
