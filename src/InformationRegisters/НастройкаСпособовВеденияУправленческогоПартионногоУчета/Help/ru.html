Настройка способа ведения управленческого партионного учета

В этом регистре сведений для каждой организации определяется способ ведения управленческого партионного учета.

 

Существуют следующие способы:

 

  • Не ведется - управленческий партионный учет по организации не ведется;
  • По компании в целом - управленческий партионный учет ведется по предприятию в целом (без учета организации);
  • По головной организации - управленческий партионный учет ведется в целом по организациям, объединенным общей головной компанией;
  • По текущей организации - управленческий партионный учет ведется отдельно для данной организации.

    Изменение порядка детализации партионного учета

    При смене порядка детализации, с которой ведется партионный учет (склады, организации и т.п.), выполняются следующие действия:

    1. Списываются старые партии документом "Списание товаров"
    2. Устанавливаются новые значения учетной политики (установить/снять нужные флажки) и способов ведения управленческого партионного учета
    3. Вводятся документы оприходования на дату начала ведения учета "по-новому". При необходимости данные детализируются (по складам, по организациям и т.п.)
    4. Устанавливается запрет на внесение изменений в документы с более ранней датой
    5. Перепроводятся все документы с более поздней датой
