Настройки согласования заявок

Предназначен для настройки использования механизма согласования заявок на расходование денежных средств. Указываются организации, для которых применяется данный механизм, и дата начала использования механизма согласования. 

Настройка механизма согласования заявок

Настройка механизма выполняется

  • в управляемом приложении (раздел "Заявки на расход денежных средств", панель навигации, подчиненный раздел "НСИ учета денежных средств")
  • в обычном приложении (интерфейс "Заведующий учетом", меню "Настройка учета - Согласование заявок на расход ДС"

Для работы с механизмом согласования заявок необходимо:

  • Заполнить список организаций, для которых используется согласование заявок. Если работа ведется в управляемом приложении, после этого необходимо перезапустить программу
  • Ввести маршруты согласования заявок
  • Настроить маршруты согласования: указать соответствие маршрутов и подразделений

Для того чтобы введенные ранее заявки начали участвовать в согласовании, после настройки механизма их необходимо перепровести.

См. также: Согласование заявок.
