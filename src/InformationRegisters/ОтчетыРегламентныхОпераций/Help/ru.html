Регистр сведений «Отчеты регламентных операций»

Регистр сведений предназначен для настройки списка отчетов, который будет доступен пользователю для контроля результатов выполнения регламентной операции (кнопка командной панели «Отчеты регламентных операций» формы «Настройка закрытия месяца»).

Список отчетов будет доступен пользователю из формы «Регламентная операция» (кнопка командной панели «Отчеты»).

Для одной регламентной операции может быть задано произвольное количество отчетов.

Настройка состава отчетов не является обязательной.
