Регистр сведений "Параметры амортизации (бухгалтерский учет)"

Содержит информацию о параметрах амортизации бухгалтерского учета.  Регистр используется для расчета амортизации по бухгалтерскому учету.

Регистр периодический, изменяется документами:

  • "Ввод начальных остатков ОС";
  • "Изменение параметров начисления амортизации ОС";
  • "Модернизация ОС";
  • "Передача ОС";
  • "Принятие к учету ОС".
