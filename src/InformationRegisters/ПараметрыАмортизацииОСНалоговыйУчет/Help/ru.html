Регистр сведений "Параметры амортизации (налоговый учет)"

Содержит информацию о параметрах амортизации налогового учета.  Регистр используется для расчета амортизации по налоговому учету.

Регистр периодический, изменяется документами:

  • "Ввод начальных остатков ОС";
  • "Изменение параметров начисления амортизации ОС";
  • "Модернизация ОС";
  • "Передача ОС";
  • "Принятие к учету ОС".
