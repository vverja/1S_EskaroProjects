Регістр відомостей "Початкові відомості про НМА(бухгалтерський облік)"

Містить початкові відомості про нематеріальні активи  організацій  для цілей бухгалтерського обліку:

  • Ознака нарахування амортизації (погашення вартості);
  • Спосіб нарахування амортизації;
  • Первісна вартість;
  • Обсяг продукції (робіт) для нарахування амортизації при використанні способу нарахування "Пропорційно обсягу продукції (робіт)";
  • Строк корисного використання;
  • Ліквідаційна вартість

Дані з регістра використовуються для розрахунку амортизації об'єктів НМА. 

По даним регістра визначається організація, в якій враховується об'єкт  в певний момент часу

Регістр періодичний, змінюється документом:

  • "Прийняття до обліку НМА".
