Регистр сведений "Показатели при нормальной мощности"

Регистр сведений предназначен для хранения параметров распределения постоянных общепроизводственных затрат: базы распределения и суммы постоянных затрат при нормальной мощности. Значения из регистра используются при расчете себестоимости выпуска по управленческому учету.
