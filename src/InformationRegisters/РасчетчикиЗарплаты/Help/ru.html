Список расчетчиков зарплаты предприятия

Расчетчиком может являться зарегистрированный в ИБ пользователь. В списке расчетчиков указывается, какое подразделение какой расчетчик «обслуживает».

При автоматическом заполнении документов по начислению зарплаты, в которых указан расчетчик (в поле «Ответственный»), в табличные части по начислениям и удержаниям заносятся данные только по тем подразделениям, которые рассчитывает этот расчетчик.
