Регистр сведений "Сведения о группах основных средств налоговый учет".

В данном регистре по каждой из налоговых групп хранятся сведения, используемые для начисления амортизации основных средств в налоговом учете.

Измерением является налоговая группа ОС, характеристики которой хранят ресурсы:

  • Пообъектно  - когда этот признак установлен, конфигурация будет хранить балансовую стоимость для каждого основного средства в этой группе, если же признак снят, то балансовая стоимость будет храниться целиком для всей группы. 
  • Статья амортизации  - статья налоговой декларации, по котрой будет списываться амортизация объектов, принадлежащих группе.
  • Статья списания стоимости - статья налоговой декларации, по котрой будет списываться остаточная стоимость группы или объекта группы в определенных законодательством случаях.
  • Минимальная стоимость объекта -  балансовая стоимость основного средства, по достижении которой основное средство должно быть списано по налоговому учету полностью.
