Регистр сведений "Соответствие подразделений и подразделений организаций"

Данный регистр предназначен для хранения информации о правилах соответствия подразделений организации  подразделениям.

 

Для каждой пары организация и подразделение может быть задано соответствующее подразделение организации. Одинаковым парам подразделение+организация могут соответствовать разные подразделения организации.

 

При заполнении соответствий "от организаций", в форме списка данного регистра расположена кнопка "Заполнить". При нажатии кнопки  "Заполнить" регистр сведений дополняется списком пар подразделений и организаций, которые отсутствуют в списке. Заполнение производится по тем подразделениям, по которым есть работающие сотрудники.

 

Информация из регистра используется при заполнении документов одного вида учета по данным другого вида учета, например, заполнение документа "Прием на работу" на основании документа "Прием на работу в организацию".
