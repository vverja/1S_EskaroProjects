Регистр сведений «Способы отражения расходов по амортизации НМА (бухгалтерский учет)»

В регистре хранятся данные о соответствии способов отражения расходов по амортизации нематериальным активам организаций для бухгалтерского учета.

Данные из регистра используются при расчете амортизации.

Регистр периодический, изменяется документами:

  • «Изменение способов отражения расходов по амортизации НМА»;
  • «Принятие к учету НМА».
