Регистр сведений «Счета учета малоценных активов»

Регистр сведений «Счета учета малоценных активов» предназначен для хранения реквизитов учета малоценных активов. Эти реквизиты используются для подстановки в документы по умолчанию при вводе малоценного актива (номенклатуры).

Сведения о реквизитах учета малоценки хранятся в следующих ресурсах регистра:

  • Схема налогового назначения;
  • Срок полезного использования;
  • Способ отражения расходов.

Реквизиты учета задаются в разрезе организаций и номенклатурных единиц. Это означает, что для каждой номенклатурной единицы можно задать набор своих реквизитов учета. В поле «Номенклатура» можно указать не только номенклатурную единицу, но и группу справочника «Номенклатуры». Тогда правила подстановки реквизитов распространятся на все номенклатурные единицы, входящие в группу, кроме тех, для которых заданы свои реквизиты учета. Если в заданном правиле учета (записи регистра «Номенклатура организаций») номенклатура не указана, то эти реквизиты учета будут подставляться для всех номенклатурных единиц, не имеющих индивидуальных правил учета или заданных реквизитов учета для групп справочника, в которые они входят.
 
Есть возможность указания реквизитов учета в разрезе организаций. Если от организации учет малоценных активов не зависит, это измерение можно не заполнять.
