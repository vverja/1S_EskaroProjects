Регистр сведений «Сведения агентов»

Данный регистр содержит сведения, необходимые для организации работы с ПО «Агент+».

Логически сведения поделены на четыре закладки:

1. Общие:

• Мобильное устройство, используемое торговым агентом;

• Ассортимент. Ассортимент товаров для выгрузки в МУ;

• Список складов. Список складов, остатки по которым выгружаются агенту;

• Основная организация. Организация, от имени которой по умолчанию будут оформляться документы в МУ;

• Мобильное приложение, которое используется на мобильном устройстве агента;

• Пароль ZIP. Пароль, используемый для защиты архива;

2. Прочие:

• Основной склад. Склад, который проставляется при загрузке документов продаж из МУ;

• Передвижной склад. Передвижной склад агента (автомобиль);

• Вид оплаты. Вид оплаты, который проставляется в документы по умолчанию;

• Список касс. Указывается список касс в разрезе организаций, используемых агентом при оформлении кассовых документов. Таким образом, при загрузке кассовых документов в реквизит "Касса" будет подставляться соответствующее значение из этого списка для той организации, которая указана в документе;

• Касса ККМ. Касса ККМ, используемая по умолчанию для заполнения реквизитов документов при загрузке в «1С»;

• Доступность типов цен. Данная настройка регулирует доступность выбора типов цен для агента при оформлении документов. Возможна выгрузка не больше 32 типов цен для конкретного агента;

• Префикс номеров документов. Первая часть префикса номеров документов, создаваемых в МУ. Вторая часть префикса берется из справочника "Организации".

2. Обмен данными:

• Почтовый ящик. Персональный электронный адрес агента, используется только при обмене через электронную почту

• FTP-папка. Папка агента на FTP-сервере, используется только при обмене через FTP;

• Агент+ Сервер папка. Папка агента для Агент+ Сервере, используется только при обмене через "Агент Плюс СОД".

 

2. GPS:

• Цвет маршрута. Цвет маршрута агента, которым будет отображаться маршрут и трек перемещения агента при работе с картами.

• Карта агента. Файл карты агента, которая будет использоваться при построении маршрута агента (используется, когда на форме настроек констант "Настройка параметров работы с мобильным приложением" в константе "Приложение просмотра карт" указано "APPlus.dll").

Примечание: Если в форме настроек констант "Настройка параметров работы с мобильным приложением" в константе "Режим выгрузки контрагентов" выбран способ "Из договоров агента" или "Из плана и договоров", то на текущей форме становится видимой вкладка «Договоры агента». На этой вкладке производится заполнение списка договоров, с которыми работает данный торговый представитель. Соответственно в МУ агента будут выгружены только контрагенты-владельцы указанного списка договоров, а также взаиморасчеты по ним. Для удобства заполнения списка, на панели в верхней части формы расположены соответствующие кнопки.
