Регистр сведений «Сведения о контрагентах»

Регистр используется для указания дополнительных сведений для контрагентов, необходимых для работы с ПО «Агент+».

При добавлении новой записи в регистр следует указывать следующие дополнительные настройки:

• Статус. Используется для привязки к контрагентам статуса, который выбирается из справочника "Статусы контрагентов";

• Доступность типов цен. Устанавливает доступность выбора типов цен для контрагента при оформлении документов непосредственно в МУ;

• Категория. Категория контрагента. Используется при формировании матрицы товаров;

• Использовать персональные цены. Этот флажок устанавливает признак учета персональных цен контрагентов, при условии, что на форме настроек констант "Настройка параметров работы с мобильным приложением" выбран флажок "Использовать персональные цены";

• Широта, Долгота. GPS-координаты расположения офиса организации. Требуется для графического отображения местоположения контрагента на карте;

• Цвет, которым будет отображаться флажок расположения офиса контрагента на карте (следует заполнять, если на форме настроек констант "Настройка параметров работы с мобильным приложением" в константе "Режим использования торговых точек" указан способ "Не использовать").

В форме регистра сведений есть возможность быстрого заполнения основных реквизитов регистра. Вызов формы для заполнения реквизитов осуществляется по кнопке "Изменить реквизиты", расположенной на панели управления формой.
