Регистр сведений «Сведения организаций»

Регистр используется для указания дополнительных сведений для организаций, необходимых для работы с ПО «Агент+».

При добавлении новой записи в регистр следует указывать следующие дополнительные настройки:

• Вид печатных форм. Выбирается из списка значений:  официальные, упрощенные, общие. В зависимости от выбранного значения в МУ будут доступны только те печатные формы документов из справочника "Печатные формы", для которых указан выбранный тип печатной формы;

• Широта, Долгота. GPS-координаты расположения офиса организации. Требуется для графического отображения флажка организации на карте;

• Использовать НДС. Признак использования НДС организацией;

• Цвет, которым будет отображаться флажок расположения офиса организации на карте.
