Отчет "Анализ начислений работникам организаций"

Отчет представляет собой средство анализа полноты проведенных расчетов зарплаты - в нем сопоставляются зарегистрированные для работников плановые начисления и удержания и фактически начисленные суммы по этим основаниям.

Отчет по умолчанию настроен так, чтобы в результате получился список работников организации сгруппированных по месяцу начисления, организации, подразделению и сотруднику.

Возможные показатели отчета

  • Начислено
  • Учтено при расчете ЕСН
  • Учтено при формировании проводок

 

Возможные группировки отчета

  • Организация
  • Подразделение организации
  • Должности организации
  • Работник
  • Месяц начисления
