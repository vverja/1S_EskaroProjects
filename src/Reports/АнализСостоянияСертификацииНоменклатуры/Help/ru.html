Отчет «Анализ состояния сертификации»

Отчет  позволяет получить информацию о состоянии сертификации серий номенклатуры, для которой требуется сертификация. В справочнике «Номенклатура» у таких позиций установлены флаги «Требуется внутренняя сертификация», «Требуется внешняя сертификация».

Если в форме отчета установлен флаг «Выводить позиции, по которым есть остатки на складе», то будут отобраны данные только по тем сериям номенклатуры, по которым на дату формирования отчета есть складские остатки, иначе в отчет будет выведена информация по всем сериям номенклатуры, для которой требуется сертификация. При этом установив необходимость вывода в отчет показателя «Остаток на складе», всегда можно получить данные о складских остатках серий номенклатуры на дату формирования отчета.
