Звіт "Аналіз субконто"

Звіти ® Аналіз субконто

Звіт призначений для представлення даних по обраному виду субконто: початкове й кінцеве сальдо, обороти за період по рахунках. Звіт можна сформувати не тільки по обраному субконто або декільком субконто, але й за значенням субконто або реквізиту значення субконто.

Звіт формується з деталізацією по рахунках. Настроювання звіту дозволяють задати додаткову деталізацію по субрахунках.

Дані можна вивести з розбивкою по періодах часу: місяць, рік і т.д.

У звіті можна вивести розгорнуте сальдо. При цьому розгорнуте сальдо розраховується для кожного рівня групування.

У звіті можна встановити наступні настройки:
