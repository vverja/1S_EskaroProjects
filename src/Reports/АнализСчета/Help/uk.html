Звіт "Аналіз рахунку"

Звіти ® Аналіз рахунку 

Звіт призначений для представлення даних по оборотам між обраним рахунком і всіма іншими рахунками за певний період.

По змісту виведеної інформації звіт схожий на звіт Обороти рахунку. Відмінність полягає у формі представлення інформації.

Звіт можна сформувати з деталізацією по субрахунках або по об'єктах аналітичного обліку (субконто). 

Дані можна вивести з додатковою розбивкою по періодах часу: місяць, рік і т.д.

У звіті можна вивести розгорнуте сальдо. При цьому розгорнуте сальдо розраховується для кожного рівня групування й по рахункові в цілому.
