Отчет «Остатки товаров на складах»

В отчете «Остатки товаров на складах» можно вывести данные об остатках товаров с распределением по складам.
 
В отчете можно установить быстрый отбор по складам компании, по номенклатуре. Кроме того, в отчете отображаются товары, зарезервированные по заказам покупателей. С помощью быстрого отбора можно отобрать товары, зарезервированные по конкретным заказам покупателей.

Товары в отчете могут выводиться в базовых и основных единицах измерения. Для розничных складов предусмотрена дополнительная оценка  товаров в розничных ценах.

Отчет может быть выведен в виде кросс-таблицы или в виде списка. Переключение вида отчета производится в диалоговом окне настроек отчета, которое открывается по кнопке «Настройка».
