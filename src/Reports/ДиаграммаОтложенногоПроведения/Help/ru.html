Диаграмма отложенного проведения

Диаграмма позволяет в наглядной форме увидеть текущее состояние отложенного проведения по каждой организации, для которой применяется механизм отложенного проведения.

Отображается следующая информация:

  • Начало отложенного проведения
  • По какую дату выполнено допроведение
  • За какой период необходимо допровести документы
  • Период запрета изменения данных. Отображается интервал дат от общей даты запрета изменения данных до самой поздней из дат запрета, установленных для организации
    • Раньше интервала всем пользователям запрещено вносить изменения
    • Позже интервала всем пользователям разрешено вносить изменения
    • Внутри интервала разрешено вносить изменения только некоторым, квалифицированным пользователям

Есть возможность получить статистику текущего состояния отложенного проведения (кнопка «Статистика») - отчет, который детализирует сведения диаграммы и содержит:

  • Дату начала отложенного проведения
  • Общее количество документов, которые поддерживают отложенное проведение и находятся позже даты начала отложенного проведения
  • Количество документов, требующих допроведения:
    • начиная с даты начала отложенного проведения
    • раньше даты начала отложенного проведения

Диаграмма доступна пользователям с ролью «Выполнение регламентных операций (регл. учет)».
