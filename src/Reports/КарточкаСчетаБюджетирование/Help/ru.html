Отчет «Карточка счета (бюджетирование)»

Отчет является одним из стандартных отчетов по бюджетированию, позволяющим получить детальную информацию обо всех движениях по выбранному счету (группе счетов) бюджетирования за период. Для каждого движения, помимо счетов и аналитики дебетуемой и кредитуемой сторон, указывается:

  • сумма в валюте управленческого учета;
  • сумма в валюте сценария (для плановых данных)
  • сумма в валюте операции (для счетов, по которым ведется валютный учет)
  • количество (для счетов, по которым ведется количественный учет


По своему функциональному наполнению (возможностям использования) данный отчет полностью аналогичен карточке счета бухгалтерского учета, только для построения данного отчета используются счета, объекты аналитического учета (субконто) и данные подсистемы бюджетирования.

Единственным отличием является необходимость указания сценария планирования, являющегося разделителем данных по плану счетов бюджетирования. Если сценарий не указан – отчет строится по фактическим данным, введенным вручную или транслированным в подсистему бюджетирования.
