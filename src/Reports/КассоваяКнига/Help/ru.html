Отчет "Кассовая книга"

Отчет «Кассовая книга» предоставляет данные для заполнения регламентированных форм учета наличных денежных средств, таких как

  • Кассовая книга;
  • Журнал регистрации приходных ордеров;
  • Журнал регистрации расходных ордеров;
  • Книга учета принятых и выданных денег.

    Данный отчет формируется по организации, выбранной в настройках отчета за указанный период.

    Если по организации ведется учет денежных средств в кассах обособленных подразделений, то в случае когда поле "Обособленное подразделение" заполнено отчет будет сформирован только по кассе обособленного подразделения. В противном случае отчет будет сформирован только по организации без учета касс обособленных подразделений.
