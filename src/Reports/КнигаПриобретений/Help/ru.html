Отчет "Книга приобретений"

Отчет «Книга приобретений» позволяет сформировать книгу приобретений в соответствии с Приказом ГНА Украины от 30.05.1997 г. № 165 с учетом последующих изменений и дополнений за заданный пользователем период.

 Отчет формируется на основании записей в регистре «Книга приобретений», сформированными документами «Запись книги приобретения» и  «ГТД Импорт».

 

В отчет могут быть добавлены итоговые строки (с опеределенной периодичностью). Для этого необходимо установить соответствующие флажки на форме отчета.


Отчет может быть выгружен в файл в формате XML. Реализация выгрузки данных  в формат XML выполнена в соответствии с форматом (стандартом) электронного документа отчетности плательщиков налогов, утвержденным ГНА Украины приказом №357 от 01.07.2004.

Для того, чтобы использовать данную возможность необходимо:
 -  иметь  установленным в системе  xml парсер четвертой версии. Четвертый xml парсер можно скачать в свободном доступе на сайте Microsoft:   http://download.microsoft.com/download/9/6/5/9657c01e-107f-409c-baac-7d249561629c/msxml.msi
 Кроме этого он включен в последние версии ОС Microsoft (WinXP, Win2003).
