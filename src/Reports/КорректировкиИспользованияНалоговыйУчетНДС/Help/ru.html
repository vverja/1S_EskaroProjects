Отчет "Корректировки использования налоговый учет НДС"

Отчет предназначен для анализа произведённых корректировок налогового кредита по НДС, вызванных использованием ценностей на вид налоговой деятельности (либо вид деятельности по НДС, признак амортизируемости), отличный от той, по которой предполагалось использовать ценности первоначально. Корректировки данного типа регистрируются при поомощи документа "Подтверждение корректировок использования (налоговый учет)"

Настройка отчета производится в форме «Настройка» (вызывается по кнопке «Настройка»).

Аналитическими разрезами (группировками) отчета являются:

  • Организация
  • Тип объекта корректировки
  • Объект корректировки
  • Расшифровка объекта корректировки
  • Вид налоговой деятельности
  • Вид деятельности НДС
  • Амортизируется
  • Вид налоговой деятельности по факту
  • Вид деятельности НДС по факту
  • Амортизируется по факту
  • Статья декларации НДС
  • Регистратор (документ движения)

Показателями отчета являются:

  • Сумма н/к при поступлении
  • Сумма н/к при использовании
  • Сумма н/к подтвержденная
  • Сумма корректировки н/к

Тип объекта корректировки - показывает, при использовании какого типа объектов была произведена корректировка. Возможные значения:

  • Партии товаров
  • Незавершенное производство
  • Брак в производстве
  • Затраты
  • Строительство ОС

Объект корректировки, расшифровка объекта корректировки - показывают, для какого конкретного объекта была зарегистрирована корректировка.

  • Для "Партии товаров" объектами будут являться элементы справочника "Номенклатура", а расшифровкой - документ оприходования;
  • Для "Незавершенное производство" объект - "Статья затрат", в качестве расшифровки - конкретная затрата
  • Для "Брак в производстве", "Затраты" объектом будет являться "Статья затрат", расшифровка не производится
  • Для "Строительство ОС" объектом будет являться "Объект строительства", расшифровка не производится

Вид налоговой деятельности , Вид деятельности НДС, Амортизируется - параметры налогового учета, которые были отражены при поступлении ценностей как целевые

Вид налоговой деятельности по факту, Вид деятельности НДС по факту, Амортизируется по факту - параметры налогового учета, по которым объект был фактически использован
