Отчет "Корректировки налогового кредита по НДС"

Отчет предназначен для анализа сумм корректировки налогового кредита, зарегистрированных в системе, которые отражаются в декларации об НДС в строке "16.2 Корректировка налогового кредита: другие случаи".

Настройка отчета производится в форме «Настройка» (вызывается по кнопке «Настройка»).

Аналитическими разрезами (группировками) отчета являются:

  • Организация
  • Тип корректировки
  • Регистратор (документ движения)
  • Договор контрагента
  • Ставка НДС

Показателями отчета являются:

  • База НДС
  • Сумма н/к

Тип корректировки - показывает, какого типа корректировка повлияла на налоговый кредит. Корректировки могут быть трёх типов:

  • Первичный документ - корректировка зарегистрирована первичным документом. Например, документом "Запись книги приобретения", который отражает полученный расчет корректировки.
  • Корректировки по использованию - корректировки, вызванные использованием ценностей на вид налоговой деятельности (либо вид деятельности по НДС, признак амортизируемости), отличный от той, по которой предполагалось использовать ценности первоначально. Регистрируются при помощи документа "Подтверждение корректировок использования (налоговый учет)".
  • Корректировки неправильно зарегистрированных документов - корректировки, вызванные неправильной регистрацией налоговых документов. Возникают при регистрации документа "Корректировка ожидаемого и подтвержденного НДС".

Договор контрагента, Ставка НДС - эти группировки будут не пустыми только для корректировок, которые были зарегистрированы первичным документом.
