Отчет «Учетная политика по МСФО»

Отчет «Учетная политика по МСФО» позволяет вывести в систематизированном виде (как это обычно включается в состав отчетности) информацию об основных элементах учетной политики организации для целей международного учета за отчетный период.

Основанием для автоматизированного заполнения отчета служат настройки регистра сведений «Учетная политика (международный учет)» на конец отчетного периода.

Автоматически заполняется только для конкретных организаций (индивидуальная отчетность).

При установке флага «На английском языке» все текстовые поля отчета (название отчета, и т.д.) отображаются на английском языке, при сбросе флага - на русском. Чтобы название организации также отображалось на английском языке, необходимо внести такой вариант названия в регистр сведений «Собственные контрагенты»


Ограничение текущей версии программы: не обслуживаются автоматически случаи изменения учетной политики по сравнению с предыдущим годом (в отчете при автозаполнении выводятся данные об учетной политике на конец отчетного периода, как наиболее актуальная информация для пользователей).
