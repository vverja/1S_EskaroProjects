Конструктор настройки показателя монитора эффективности

Форма предназначена для создания показателя отчета "Монитор эффективности".

Показатель может быть следующего вида:

  •  Стандартный показатель. Содержит поля: "Текущее значение", "Значение предыдущего периода", "Изменение %", "Изменение (абс)"
  •  План-факт. Содержит, кроме полей стандартного показателя, поля: "План", "% Выполнения", "Отклонение %", "Отклонение (абс)"
  •  План (прогноз). Содержит поля: "Текущее значение", "Прогноз"

При настройке любого вида показателя необходимо выбрать набор данных, по данным которого будут формироваться поля отчета. После этого необходимо сопоставить полям отчета поля набора данных.

При завершении работы конструктора модифицируется схема компоновки и настройки отчета.
