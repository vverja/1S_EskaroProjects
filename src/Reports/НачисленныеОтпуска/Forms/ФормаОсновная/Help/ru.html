Отчет "Начисленные отпуска"

Отчет предоставляет возможность сформировать списки работников с суммами начисленных отпускных, а также с количеством дней соответствующего отпуска по кадровым и расчетным документам.

Отчет по умолчанию настроен так, чтобы в результате получился список с максимальным количесвом показателей.

Возможные показатели отчета

  • Организация
  • Подразделение
  • Наименование работника (из справочника Сотрудники)
  • Дней отпуска по кадровым документам
  • Дней отпуска по расчетным документам
  • Отпускные (сумма начисленных отпускных)

 

Возможные группировки отчета

  • Организация
  • Подразделение 
  • Сотрудник
