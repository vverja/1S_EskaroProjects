Отчет «Оборотно-сальдовая ведомость  (бюджетирование)»

Отчет является одним из стандартных отчетов по бюджетированию, позволяющим получить информацию о планируемых остатках и движениях средств и источников финансирования предприятия в разрезе выбранной для бюджетирования структуры плана счетов.

По своему функциональному наполнению (возможностям использования) данный отчет полностью аналогичен оборотно-сальдовой ведомости по бухгалтерскому учету, только для построения данного отчета используются счета, объекты аналитического учета (субконто) и данные подсистемы бюджетирования.

Единственным отличием является необходимость указания сценария планирования, являющегося разделителем данных по плану счетов бюджетирования. Если сценарий не указан – отчет строится по фактическим данным, введенным вручную или транслированным в подсистему бюджетирования.

В качестве расшифровок данного отчета выступают другие стандартные отчеты подсистемы бюджетирования – «Карточка счета (бюдж.)» и «Оборотно-сальдовая ведомость по счету (бюдж.)».
