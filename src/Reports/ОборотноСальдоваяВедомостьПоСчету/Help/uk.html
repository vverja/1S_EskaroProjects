Звіт " Оборотно-Сальдова відомість по рахунку"

Звіти ® Оборотно-сальдова відомість по рахунку

Звіт призначений для формування оборотно-сальдової відомості по обраному рахунку за певний період часу.

По складу виведеної інформації звіт нагадує фрагмент звіту Оборотно-Сальдова відомість.

Звіт можна сформувати з деталізацією по субрахунках або по об'єктах аналітичного обліку (субконто). 

Дані можна вивести з додатковою розбивкою по періодах часу: місяць, рік і т.д.

У звіті можна вивести розгорнуте сальдо. При цьому розгорнуте сальдо розраховується для кожного рівня угруповання й по рахункові в цілому.

У звіті можна встановити наступні настройки:

Показники
Групування
Відбір
Сортування
Оформлення
Додаткові дані
Діаграма
