Отчет «Оборотно-сальдовая ведомость по счету (бюджетирование)»

Отчет является одним из стандартных отчетов по бюджетированию, позволяющим получить информацию об операциях, отраженных по конкретному счету бюджетирования учета за заданный период.

По своему функциональному наполнению (возможностям использования) данный отчет полностью аналогичен оборотно-сальдовой ведомости по счету бухгалтерского учета, только для построения данного отчета используются счета, объекты аналитического учета (субконто) и данные подсистемы бюджетирования.

Единственным отличием является необходимость указания сценария планирования, являющегося разделителем данных по плану счетов бюджетирования. Если сценарий не указан – отчет строится по фактическим данным, введенным вручную или транслированным в подсистему бюджетирования.

Отчет можно сформировать в валюте управленческого учета компании и в валюте, выбранной для сценария планирования. В качестве расшифровки данного отчета выступает другой стандартный отчет подсистемы бюджетирования – «Карточка счета (бюдж.)».
