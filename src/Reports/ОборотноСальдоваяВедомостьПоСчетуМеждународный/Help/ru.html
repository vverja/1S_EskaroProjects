Отчет «Оборотно-сальдовая ведомость по счету (МСФО)»

Отчет предназначен для получения данных об остатках на начало и конец периода и  оборотах за период по выбранному счету международного плана счетов. Отчет позволяет получить детализацию по субсчетам счета.

Строки отчета могут быть дополнительно расшифрованы карточкой счета.

По своему функциональному наполнению  отчет  аналогичен оборотно-сальдовой ведомости по счету бухгалтерского учета, только для построения данного отчета используются счета и данные международного учета.
