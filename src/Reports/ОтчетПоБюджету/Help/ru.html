Отчет «Отчет по бюджету»

Отчет предназначен для анализа остатков по счетам и оборотов по статьям, указанным в подчиненных справочниках «Счета бюджета» и «Статьи бюджета» для какого – либо элемента справочника «Бюджеты».

В общем виде, выходная форма отчета делится на три блока:

  1. Остатки на начало интервала отчета по счетам, выбранным в справочнике «Счета бюджета».
  2. Обороты за интервал отчета по статьям, выбранным в справочнике «Статьи бюджета».
  3. Остатки на конец интервала отчета  по счетам, выбранным в справочнике «Счета бюджета»

Порядок настройки отчета сходен с другими отчетами.

На основной форме указываются:

  • сценарий, по которому будут анализироваться обороты. Если сценарий не указан – будут отбираться фактические данные, внесенные вручную или транслированные в подсистему бюджетирования из других подсистем;
  • бюджет, в соответствии с настройками которого будет строиться отчет;
  • интервал отчета, кратный периодичности планирования выбранного сценария.
  • отборы по ЦФО и проектам (при необходимости, реквизиты вызываются по кнопке «Отбор» главного меню);
  • группировка отчета, значения которой будут разворачиваться по горизонтали при формировании отчета. Важно: состав списка для выбора определяется детализирующими группировками отчета, выбранными на форме дополнительной настройки. Статьи и счета бюджета являются предопределенными группировками и всегда разворачиваются по вертикали.

На дополнительной форме настройки, вызываемой по кнопке «Настройка» главного меню основной формы, указываются те же самые реквизиты, что и на основной. Также, дополнительно настраивается:

  • состав показателей, выводимых в отчет;
  • вид отчета (список аналитических разрезов и их размещение в строках и столбцах);
  • отборы;
  • признак вывода показателей отчета в разных колонках.
