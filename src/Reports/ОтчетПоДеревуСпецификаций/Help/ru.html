Отчет "Отчет по дереву спецификаций"

Отчет предназначен для формирования печатных форм:

  • полного дерева спецификаций

  • списка исходных комплектующих с иерархией по родителю номенклатуры.

  • списка покупных изделий  дерева спецификаций. В печатную форму выводится номенклатура с выбранным видом воспроизводства «Покупка».

  • списка полуфабрикатов всего дерева спецификаций. В печатную форму выводится номенклатура с выбранным видом воспроизводства «Производство».

 

Кнопка «Настройка»

 

При нажатии на кнопку производится установка реквизитов, влияющих на работу отчета.

  • Выводить спецификации на дату - дата, на которую определяются основные спецификации номенклатуры.

 

 Группа реквизитов «Вид отчета»:

  • Флажок "Дерево спецификаций" -  добавляет в поле табличного документа печатную форму дерева спецификаций
  • Флажок "Исходные комплектующие" -   добавляет в поле табличного документа печатную форму списка исходных комплектующих из указанной спецификации продукции
  • Флажок "Покупные изделия" -  добавляет в поле табличного документа печатную форму списка покупных изделий из указанной спецификации продукции
  • Флажок "Полуфабрикаты" - добавляет в поле табличного документа печатную форму списка полуфабрикатов из указанной спецификации продукции

 

 Группа реквизитов «Настройка дерева спецификаций»:

  •  Флажок «Разворачивать дерево спецификаций» - при установке данного флажка при формировании дерева спецификаций будет выполняться раскрытие всех уровней дерева.
  • Флажок «Выводить выходные изделия» определяет вывод строк выходных изделий в дереве спецификаций.
  • Флажок «Выводить возвратные отходы» определяет вывод строк возвратных отходов в дереве спецификаций.
  • Количество уровней разворота дерева - дерево спецификаций будет развернуто до уровня, указанного в данном поле. Если в поле указано значение 0, то дерево спецификаций будет развернуто полностью по всем уровням. 

Группа реквизитов «Рассчитывать спецификацию на количество»:

  • Количество продукции - если значение > 0, спецификация будет рассчитана для указанного количества в указанной единице измерения
  • Единица измерения-  спецификация будет рассчитана для указанного количества в указанной единице измерения
