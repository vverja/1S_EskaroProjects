Отчет «Пробный баланс (МСФО)»

Отчет является одним из стандартных отчетов по международному учету, позволяющим получить наиболее общую информацию о данных международного учета.

По своему функциональному наполнению (возможностям использования) данный отчет полностью аналогичен оборотно-сальдовой ведомости по бухгалтерскому учету, только для построения данного отчета используются счета, объекты аналитического учета (субконто) и данные международного учета, но сальдо показываются в «свернутом» виде (то есть дебетовые сальдо - со знаком «плюс», кредитовые - со знаком «минус»).

В качестве расшифровок данного отчета выступают другие стандартные отчеты по международному учету – «Карточка счета (МСФО)» и «Оборотно-сальдовая ведомость по счету (МСФО)».
