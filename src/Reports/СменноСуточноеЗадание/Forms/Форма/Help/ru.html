Отчет «Сменно-суточное задание»

Отчет предназначен для выборки и отображения всех операций, назначенных к исполнению в производстве, то есть для визуализации плана-графика производства за некоторый интервал времени.

Настройка отборов

Отборы предназначены для быстрой установки фильтров на отбираемые данные. Накладываются на:

  • Рабочий центр (потребление только по указанному рабочему центру)
  • Заказ (потребление только по указанному заказу)
  • Проект (потребление только по указанному проекту)
  • Номенклатура (потребление только указанной номенклатуры)

Отчет имеет фиксированную структуру, в которой отображается следующая информация о производстве:

  • Дата
  • Подразделение производства
  • Смена
  • Исполняющий рабочий центр
  • Технологическая операция
  • Спецификация производства
  • Конечная продукция
  • Заказ
  • Проект
  • Время исполнения (производства)
  • Продукция выпуска
  • Характеристика продукции выпуска
  • Количество выпуска (производства, переработки)
