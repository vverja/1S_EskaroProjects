Отчет «Сравнительный анализ движения денежных средств»

Отчет предназначен для сравнительного анализа планируемых и фактических движений по статьям движения денежных средств, относящихся к различным сценариям.

Отчет может использоваться для:

  • план – фактного анализа исполнения планов движения денежных средств;
  • сравнения плановых данных по различным сценариям;
  • сравнения как плановых, так и фактических данных за различные временные отрезки.

Список данных для сравнения указывается на форме дополнительной настройки, открывающейся по кнопке «Настройка» главной формы отчета. Для каждой строки списка задается вид данных и период. Для сравнения можно использовать виды данных:

  • данные сценария - плановые данные по «Планам движения денежных средств»; для таких данных необходимо указать сценарий;
  • фактические данные - фактические движения денежных средств;
  • фактические данные с учетом заявок и планируемых поступлений - кроме фактических движений денежных средств, принимаются во внимание данные платежного календаря, внесенные документами «Заявки на расходование средств» и «Планируемые поступления денежных средств».

Шаг изменения интервалов отбора данных кратен наибольшей из периодичностей сценариев, выбранных для сравнения.

Важно. Первая строка списка служит для определения «эталонных» данных, с которыми будут сравниваться все остальные; она выделена жирным шрифтом.

На дополнительной форме настройки, помимо статьи баланса и списка данных для сравнения, указываются:

  • состав показателей для вывода в отчет;
  • способ вывода отклонений: абсолютное и (или) относительное;
  • признак детализации данных внутри интервала отбора по периодам;
  • состав детализирующих группировок отчета;
  • необходимые отборы.
