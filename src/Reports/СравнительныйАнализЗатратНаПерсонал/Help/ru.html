Отчет "Сравнительный анализ затрат на персонал"

Отчет позволяет провести сравнительный анализ планируемых и фактических затрат на персонал.

 

Затраты можно группировать по подразделениям и статьям затрат.

 

В отчет можно вывести несколько сценариев для сравнения. Каждый сценарий выводится в отдельной колонке.

Для планируемых данных необходимо указать сценарий, для которого выводить план, а также период сравнения. Для фактических данных достаточно указать только период сравнения.

 

Сравнивать затраты можно в валютах управленческого и регламентированного учетов.

 

Первый сценарий в списке сценариев для сравнения считается эталонным. Для других сценариев можно вывести абсолютные и процентные значения отклонения от эталонного сценария.
