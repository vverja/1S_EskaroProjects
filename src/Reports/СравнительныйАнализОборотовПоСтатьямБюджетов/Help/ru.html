Отчет «Сравнительный анализ оборотов бюджетов»

Отчет предназначен для сравнительного анализа планируемых и фактических движений по статьям оборотов подсистемы бюджетирования, относящихся к различным сценариям.

Отчет может использоваться для:

  • план – фактного анализа исполнения бюджетов;
  • сравнения плановых данных по различным сценариям;
  • сравнения как плановых, так и фактических данных за различные временные отрезки.

Список сценариев и интервалов отбора данных указывается на форме дополнительной настройки, открывающейся по кнопке «Настройка» главной формы отчета.

Шаг изменения интервалов отбора данных кратен наибольшей из периодичностей сценариев, выбранных для сравнения. Для анализа фактических данных, поле «Сценарий» в таблице необходимо оставить пустым.

Важно. Первая строка списка служит для определения «эталонных» данных, с которыми будут сравниваться все остальные; она выделена жирным шрифтом.

На главной форме отчета указывается:

  • статья баланса: счет или группа счетов плана счетов бюджетирования. Если статья указана, в отчет для сравнения будут отбираться обороты по статьям, в шаблонах проводок которых используются счета, входящие в выбранную статью баланса. При этом обороты, дебетующие выбранные счета, будут отражены с положительным значением, а кредитующие – с отрицательным. Итоговым значением отчета по каждому сценарию, таким образом, будет изменение статьи баланса за период отчета. Статьи оборотов группируются в соответствии со структурой справочника «Статьи оборотов по бюджетам». Если статья баланса не указана – будут отбираться обороты по всем статьям бюджетов.
  • отборы по статьям оборотов и ЦФО (при необходимости, реквизиты вызываются по кнопке «Отбор» главного меню).

На дополнительной форме настройки, помимо статьи баланса и списка данных для сравнения, указываются:

  • состав показателей для вывода в отчет;
  • способ вывода отклонений: абсолютное и (или) относительное;
  • признак детализации данных внутри интервала отбора по периодам;
  • состав детализирующих группировок отчета;
  • необходимые отборы.
