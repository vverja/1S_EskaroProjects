Отчет «Сравнительный анализ остатков на счетах бюджетирования»

Отчет предназначен для сравнительного анализа планируемых и фактических остатков по плану счетов бюджетирования, относящихся к различным сценариям. Отчет может использоваться для:

  • план – фактного анализа исполнения бюджета по балансу;
  • сравнения планируемых остатков по различным сценариям;
  • сравнения как планируемых, так и фактических остатков на различные даты

Список сценариев и дат получения остатков указывается на форме дополнительной настройки, открывающейся по кнопке «Настройка» главной формы отчета. Шаг изменения даты остатка данных кратен наибольшей из периодичностей сценариев, выбранных для сравнения. Для анализа фактических данных, поле «Сценарий» в таблице необходимо оставить пустым.

Важно. Первая строка списка служит для определения «эталонных» данных, с которыми будут сравниваться все остальные; она выделена жирным шрифтом.

Настройка отчета производится на дополнительной форме, открывающейся по нажатию кнопки «Настройка» панели меню основной формы отчета. Здесь указываются:

  • список данных для сравнения;
  • состав показателей для вывода в отчет;
  • способ вывода отклонений: абсолютное и (или) относительное;
  • состав детализирующих группировок отчета;
  • необходимые отборы.
