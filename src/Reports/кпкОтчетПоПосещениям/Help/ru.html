Отчет "Отчет по посещениям"

Данный отчет выводит информацию по документам, сформированным агентом за выбранный период, с возможностью выбора следующих типов группировок : по агентам, по времени создания, по торговой точке или по виду документа.

Кнопка «Отбор» на панели управления предназначена для отображения/скрытия панели, на которой выбираются параметры формирования отчета.

С помощью значений отбора, представленных на форме отчета, можно отобрать документы по следующим критериям: по типам документов, по агентам и по контрагентам. С помощью установки соответствующего флага можно отобрать только проведенные документы.

Для каждого документа в списке выводится следующая информация: номер в списке, дата, номер документа в КПК, вид документа, номер документа в базе 1С: Предприятия, сумма документа, количество строк в документе, агент, контрагент, торговая точка.
