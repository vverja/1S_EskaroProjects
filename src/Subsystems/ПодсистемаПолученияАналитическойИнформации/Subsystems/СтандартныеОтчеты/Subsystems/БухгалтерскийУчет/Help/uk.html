Настройка параметрів звіту

У верхній частині звіту знаходиться командна панель. За допомогою настроювань панелі можна змінити параметри звіту: установити інший період, організацію та ін.

Роздрукувати звіт можна по кнопці Друк.

Зберегти звіт можна по кнопці Зберегти як.

Виділяючи комірки табличної частини, можна відразу одержати підсумкову суму в поле Сума виділених комірок.

Панель настроювання

Панель настройка відкривається (ховається) по кнопці Настроювання.

Настроїти видимість панелей настройка звітів можна по кнопці Параметри панелі настроювань.

Панель настройка можна закривати при формуванні звіту, для цього потрібно встановити прапорець Закривати при формуванні звіту у параметрах панелі настройки.

Зберегти настройка звіту можна по кнопці Зберегти настройка звіту. Завантажити настройка звіту можна по кнопці Завантажити настройки звіту. Сформувати звіт з настроюваннями з історії звіту можна по кнопці Історія.

Показники

У звіті можуть відбиватися одночасно дані бухгалтерського й податкового обліку, а також дані по постійним і тимчасовим різницям в оцінці активів і зобов'язань. У звіті можна задати настройку контролю дотримання принципу БУ = НУ, можна виводити дані про кількість і валютну суму. Для цього призначена панель настроювань Показники.

Групування

У звіті можна задати групування по аналітичних об'єктах (субконто), по групах об'єктів. Для цього призначена панель настройка Групування. Порядок угруповання субконто змінюється по кнопці Нагору або Вниз.

На панелі можна встановити настройку деталізації по субрахунках або по субрахунках кореспондуючих рахунків (прапорець По субрахунках, По субрахунках кор. рахунків).

Дані можна вивести з додатковою розбивкою по періодах часу: тиждень, місяць, рік і т.д. Періодичність формування звіту можна задати в полі Обороти.

Відбір

На панелі Відбір можна вибрати параметр або реквізит, по якому буде зроблений відбір у звіті, а також задати вид порівняння (рівно, у списку і т.д.).

У колонку Поле слід вибрати параметр або реквізит, доступний для настройка звіту. В останньому стовпчику вказується конкретне значення обраного параметра або реквізиту параметра.

У колонку Вид порівняння вибирається умова порівняння. Тут можна вибирати серед наступних варіантів:

  • Рівно - параметр повинен точно дорівнювати значенню.
  • Не рівно - параметр не дорівнює значенню.
  • У списку - параметр повинен бути рівний одному зі значень, зазначених у графі Значення. При виборі такої умови в графі Значення з'являється можливість указати не одне значення, а список значень.
  • У групі - параметр повинен бути рівний або самому значенню, або підлеглому значенню.
  • У групі зі списку - аналогічно попередньому умові для списку значень.
  • Не в списку - параметр повинен бути не рівний жодному значенню, ні підлеглим значенням.
  • Не в списку зі списку - аналогічно попередньому умові для списку значень.
  • Заповнене (Не заповнене) - відбір по заповнених (незаповненим) значенням.

Якщо в табличній частині вказати дві або більше умов заповнення декількох рядків, то при відборі даних для формування звіту буде перевірятися одночасне дотримання всіх зазначених умов.

Сортування

На панелі сортування можна впорядкувати значення по графах (полям) звіту. Наприклад, сальдо на початок можна сортувати по зростанню або убуванню значень. Елементи довідника Контрагенти можна сортувати по будь-якому реквізиту: коду, ІПН, найменуванню й т. д. Можна задавати відразу кілька настройок для сортування.

Розгорнуте сальдо

Розгорнутим сальдо називається сальдо (залишок), складене із двох компонентів: дебетового сальдо й кредитового сальдо. Причому дебетове сальдо визначається як сума дебетових сальдо всіх нижчестоящих субрахунків. Кредитове сальдо визначається як сума кредитових сальдо всіх нижчестоящих субрахунків.

Ознаку виводу розгорнутого сальдо у звітах можна встановлювати для окремих рахунків, субконто рахунків. Ознаку виводу розгорнутого сальдо доцільно встановлювати для рахунків обліку розрахунків, щоб при формуванні звіту не відбувалося взаємозаліку боргів, не зв'язаних між собою.

Оформлення

На панелі оформлення можна управляти настройками заголовка, виводом підписів, виділенням від'ємних залишків. Також можна вибрати варіант оформлення звіту.

Додаткові дані

У звіт можна виводити додаткові дані, наприклад, додати у звіт дані по реквізиту номенклатури: ставка ПДВ, код номенклатури і т.д. Для цього призначена панель настроювань Додаткові дані.

Діаграма

Для виводу даних бухгалтерського обліку в графічному виді призначена кнопка Діаграма командної панелі звіту. Настройка діаграми можна змінювати на панелі настройка Діаграма: вибирати показник, змінювати тип діаграми, задавати підписи значень.
