Задача "Задача исполнителя"

Список задач содержит все задачи, которые необходимо выполнить текущему пользователю.

 

Виды задач, которые будут добавляться в список, определяются для всех пользователей сразу.

При изменении состава видов задач, существующие задачи остаются в списке. Ненужные задачи можно непосредственно удалить с помощью комбинации клавиш Shift-Del.

 

В список попадают как задачи, в которых явно указан исполнитель, так и задачи, которые может выполнять любой кадровик или расчетчик конкретной организации.

Если в настройках указана необходимость разделять задачи по ролям и/или организациям, каждый пользователь в своих настройках может указать, задачи каких ролей или организаций должны отображаться в списке.

 

После выполнения задачи она не исчезает из списка автоматически. Необходимо явно отмечать, что задача выполнена.

При выборе задач доступен множественный выбор задач (с помощью кнопок Ctrl и Shift).
